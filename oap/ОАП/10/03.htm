Точность чисел с плавающей запятой

Манипулятор потока setprecision, и функция-элемент precision задают точность чисел с плавающей запятой (количество знаков после запятой).

Вызов любой из этих установок точности действует для всех последующих операций вывода до тех пор, пока не будет произведена следующая установка точности.

Функция-элемент precision не имеет никаких аргументов и возвращает текущее значение точности.
Точность 0 восстанавливает точность по умолчанию, равную 6. 

	#include<iostream.h>
	#include<iomanip.h>
	void main()
	{
		double tt = sqrt(2.0);
		cout << "Точность 0 - 9 (функция - элемент precision)\n";
		for(int i = 0; i <= 9; i++)
		{
			cout.precision(i); cout << tt << endl;
		}
		cout << "Точность установлена манипулятором setprecision\n";
		for(i = 0; i <= 9; i++)
		{
			cout << setprecision(i) << tt <<  endl;
		}
	}
