Вывод потоков

Класс ostream в C++ обеспечивает возможность реализации форматированного и неформатированного вывода потоков следующих данных:
  • вывод данных стандартных типов с помощью операции поместить в поток cout;
  • вывод символов с помощью функции-элемента put;
  • неформатированный вывод с помощью функции-элемента write;
  • вывод целых чисел в десятичном, восьмеричном и шестнадцатеричном форматах;
  • вывод значений с плавающей запятой с различной точностью, с указанием по выводу десятичной точки, в экспоненциальном формате или в формате с фиксированной точкой;
  • вывод данных с выравниваем относительно какой-либо границы поля указанной ширины;
  • вывод данных с полями, заполненными заданными символами;
  • вывод буквами в верхнем регистре в экспоненциальном формате и при выводе шестнадцатеричных чисел.

Операция поместить в поток

Вывод в потоки может быть выполнен с помощью перегруженной операции "поместить в поток" <<.
Операция << перегружена для вывода элементов данных встроенных типов, для вывода строк, для вывода значений указателей.

В программе показан вывод строки: Добро пожаловать в мир C++! с использованием операции поместить в поток. 

		cout << "Добро пожаловать в мир C++!\n";

Переход на новую строку

может быть осуществлен с помощью управляющей последовательности \n и с помощью манипулятора потока endl (end line - конец строки), который вызывает переход на новую строку и, кроме того, приводит к сбросу буфера вывода (то есть заставляет буфер немедленно вывести данные, даже если он полностью не заполнен). 

		cout << "Добро пожаловать в мир C++ !" << endl;

Вывод значения выражения
		cout << "47 плюс 53 равняется " << (47 + 53) << endl;

Вывод строки

Известно, что символьная строка имеет тип char *. Необходимо напечатать значения этого указателя, то есть адрес в памяти первого символа этой строки, для чего надо привести тип этого указателя к типу void * (это следует делать и для любого указателя на переменную, если программист хочет вывести ее адрес).

Программа показывает печать переменной char * и как строки, и как адреса (шестнадцатеричное число). 

		char  *string = "проверка";
		cout << "Значение строки равно " << string 
			<< "\nЗначение адреса строки (void *) string равно " 
			<< (void *) string << endl;

		Результаты работы фрагмента программы
		Значение строки равно проверка
		Значение адреса строки (void *) string равно 0x00aa

Функция - элемент put выводит одиночный символ

Например,     cout.put ('А');     отображает на экране символ "А".

Вызовы функций put могут быть сцепленные     cout.put ('А').put('\n');    - оператор выведет на экран букву A, а затем выведет символ новой строки.

Функция put может также вызываться с помощью выражения, имеющего значение кода ASCII символа, выражение     cout.put(65)     также выводит букву А.
