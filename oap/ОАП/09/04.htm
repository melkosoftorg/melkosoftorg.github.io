Ввод потоков

Операция ввода потока может быть выполнена с помощью перегруженной операции "взять из потока" >>, которая обычно игнорирует во входном потоке так называемые символы разделители или пробельные символы (пробелы, знаки табуляции, знак новой строки).

Например, вычисление суммы двух целых чисел, которые вводятся с клавиатуры с помощью cin и операции взять из потока 

	int x, y;
	cout << "Введите два целых числа: "; cin >> x >> y;
	cout << "Сумма чисел " << x << " и " << y << " равна " << (x + y) << endl;
Относительно высокий приоритет операций >> и << может привести к возникновению проблем. Например, строка программы 
	cout << x << (x == y ? " " : " не ") << " равно " << y << endl;
не может быть скомпилирована должным образом, если условное выражение не выделено c помощью круглых скобок.

В программе, которая находит самый высокий балл, количество вводимых значений не известно и пользователь должен вводить признак конца файла (Ctrl-Z) для того, чтобы показать, что все значения уже введены 

	#include<iostream.h>
	void main()  	
	{
		int x, y = -1;
		cout << "Балл (в конце - Ctrl-Z): ";
		while (cin >> x)
		{
			if (x > y)
				y = x;
			cout << "Балл (или Ctrl-Z): ";
		}
		cout << "\nНаивысший балл: " << y << endl;
	}
		Результаты
			Балл (в конце - Ctrl-Z): 67
			Балл (или Ctrl-Z): 87
			Балл (или Ctrl-Z): 73
			Балл (или Ctrl-Z): 99
			Балл (или Ctrl-Z): 34
			Балл (или Ctrl-Z): 95
			Балл (или Ctrl-Z): ^z
			Наивысший балл: 99
