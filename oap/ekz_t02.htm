  8. Укажите верные утверждения:
    • Размер массива - это количество элементов, входящих в массив.
    • В массив могут входить элементы разных типов.
    • Тип массива может быть одномерный, двумерный
    • У каждого элемента массива есть свой номер (индекс)
    • Размер массива - это количество байт, занимаемых массивом в памяти компьютера

  9. Укажите НЕ верные утверждения:
    • Массивы могут быть только числовыми
    • Размерность массива - это количество элементов, входящих в массив
    • Массив - это упорядоченный набор однотипных данных
    • В массив могут входить элементы разных типов
    • Индекс элемента массива принимает значения от 0 до N-1, где N - размер массива
