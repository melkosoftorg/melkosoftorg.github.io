Математические операции, выражения, операторы

  1. Выражение представляет собой объединение операций и операндов.

  2. Операнд - это то, над чем выполняется операция.

  3. Оператор является законченной инструкцией для компьютера.
    В операторе 
    		y = a * b / (3 + b - a);

    y, a, b - операнды.

  4. Результат выражения определяется типами обоих операндов.

  5. Если действие производится с двумя операндами, операция называется бинарной или диадической.

  6. Если действие производится с одним операндом, операция называется унарной.

  7. Операция присваивания = выполняется следующим образом:
    • Вычисляется значение выражения в правой части присваивания.
    • После этого переменная, указанная в левой части, получает вычисленное значение.
      При этом тип выражения должен быть преобразуем с помощю операции преобразования к типу переменной.

  8. Можно применять последовательные присваивания.
    Например: sum = a = b = 25.7

  9. Операция деления по модулю % - операция нахождения остатка от деления целых чисел.

  10. Комбинированные операции
    Любое выражение вида 
    		<переменная> = <переменная> <операция> <выражение>;

    может быть заменено на 

    		<переменная> <операция> = <выражение>;

    ВыражениеСокращается до
    a = a + b;a += b;
    a = a - b;a -= b;
    a = a * b;a *= b;
    a = a / b;a /= b;
    a = a % b;a %= b;

  11. Операции приращения ++ и уменьшения -- позволяют использовать единственную операцию, которая добавляет 1 или вычитает 1 из любого значения. 
    	sum = a + b++ - сложить a и b, присвоить результат sum 
		         и увеличить b на единицу, 
	sum = a + ++b - увеличить b на единицу, сложить a и b, 
		         и присвоить результат sum.

	sum = a + b-- - сложить a и b, присвоить результат sum 
		         и уменьшить b на единицу, 
	sum = a + --b - уменьшить b на единицу, сложить a и b, 
		         и присвоить результат sum.

  12. Оператор запятая , используется для организации множественных выражений, расположенных внутри круглых скобок.
