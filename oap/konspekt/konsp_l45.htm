Наследование и другие возможности классов

  1. Наследование (inheritance) - возможность создания новых классов на основе существующих классов.

  2. Классы, на основе которых строятся другие классы, называются базовые

  3. Классы, построенные на основе базовых классов, называются производные

  4. Для задания шаблона производного класса используется синтаксис: 
    	class имя_производного_класса : необязательный_атрибут имя_базового_класса
	{
		тело_производного_класса
	} объекты_производного_класса (через запятую);

  5. Класс называют непосредственным (прямым) базовым классом (прямой базой), если он входит в список базовых при определении класса.

  6. Наличие нескольких прямых базовых классов называют множественным наследованием.

  7. Явный вызов деструктора имеет вид 
    	имя_объекта.имя_класса :: ~имя_класса();
