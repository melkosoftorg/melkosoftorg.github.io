Поиск и сортировка в двумерном массиве

  1. Массивы характеризуются
    1. именем (идентификатором),
    2. размерностью (одномерный, двумерный, трехмерный, ...),
    3. размером (количеством элементов).

  2. Примером двумерного массива являются таблицы значений, содержащие информацию в строках и столбцах.

  3. Чтобы определить отдельный табличный элемент, нужно указать два индекса:
    1. первый (по соглашению) указывает номер строки,
    2. второй (по соглашению) указывает номер столбца.

  4. Охарактеризуем массив    int mas[3][4];    - целочисленный двумерный массив - значениями элементов являются целые числа.
    1. Имя массива - mas.
    2. Размерность - двумерный.
    3. Размер - 12:
      • Количество строк - 3 (индексы 0, 1, 2).
      • Количество столбцов - 4 (индексы 0, 1, 2, 3).
      • Количество элементов массива - 12.

  5. В таблице указаны возможные значения элементов массива mas[3][4]. Чтобы обратиться к любому из элементов, достаточно указать значения i и j

    Индекс
    строки    		 Индекс столбца
    j
    0123
    i 0 -34011105
    1 45341567-1915
    2 19374-131412-15

    Например, значение элемента   mas[1][2] равно -191,   mas[0][3] равно 105,   mas[2][1] равно -1.

  6. Накопление суммы и произведения
    1. Обязательно задать начальные значения накапливаемым величинам. Сделать это до цикла.

      • При накоплении суммы начальное значение равно 0: sum = 0; kol = 0;

      • При накоплении произведения начальное значение равно 1: p = 1;

    2. Накопление суммы и произведения осуществлять в цикле.

    3. Накопление суммы.

      • Если накапливаем сумму значений элементов массива (обозначим ее sum), то формула:
        sum += mas[i][j];

      • Если накапливаем количество элементов (обозначим kol), то формула:
        kol++;

    4. Накопление произведения значений элементов массива производить по формуле:
      p *= mas[i][j];

  7. Работа со всеми элементами матрицы
    1. Просматривать все элементы матрицы безразлично в какой последовательности: по строкам или по столбцам.
    2. Если необходимо задать какие-либо начальные значения, сделать это до внешнего цикла.
    3. Все необходимые действия выполняются в теле внутреннего цикла.
    4. Вывод полученных результатов - за внешним циклом.

  8. Работа с элементами строки
    1. Внешний цикл - по индексу строки: вошли в строку и проанализировали каждый элемент этой строки.
    2. Если необходимо, задать начальные значения переменным.
    3. Внутренний цикл - по индексу столбца.
    4. Вывод полученных результатов для данной строки - за внутренним циклом.

  9. Работа с элементами столбца
    1. Внешний цикл - по индексу столбца: вошли в столбец и проанализировали каждый элемент этого столбца.
    2. Если необходимо, задать начальные значения переменным.
    3. Внутренний цикл - по индексу строки.
    4. Вывод полученных результатов для данного столбца - за внутренним циклом.

  10. Квадратная матрица - матрица, у которой количество строк и количество столбцов двумерного массива (матрицы) одинаково.
    Например, 
    	const int N = 4;
	int mas[N][N] = { { -34,  167, 763,   0},
			     { 483, -679, -31,   1},
			     { -83,   79, 231, -21},
			     {-183,  179, 123, 121}
                      }

  11. Диагональные элементы матрицы.

    1. Главная диагональ - совокупность элементов матрицы, у которых совпадают номера строки и столбца:
      mas[0][0], mas[1][1], mas[2][2], mas[3][3].

        0 1 2 3
      0 -34 167 763 0
      1 483 -679 -31 1
      2 -83 79 231 -21
      3 -183 179 123 121

      Над главной диагональю находятся элементы, индексы столбцов которых больше индексов строк.

      Под главной диагональю находятся элементы, индексы столбцов которых меньше индексов строк.

    2. Побочная диагональ - совокупность элементов матрицы, у которых сумма индексов на единицу больше порядка матрицы:
      mas[0][3], mas[1][2], mas[2][1], mas[3][0].

      Порядок матрицы - число строк.

      Поскольку в C++ индексация строк и столбцов, в отличие от математики, начинается не с 1, а с 0, то сумма индексов должна быть на единицу меньше числа строк.

        0 1 2 3
      0 -34 167 763 0
      1 483 -679 -31 1
      2 -83 79 231 -21
      3 -183 179 123 121

      Над побочной диагональю находятся элементы сумма индексов строк и столбцов которых меньше N-1.

      Под побочной диагональю находятся элементы сумма индексов строк и столбцов которых больше N-1.
