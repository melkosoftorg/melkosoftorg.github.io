Для конспекта

 

Адресные операции. Указатели.

  1. При объявлении переменной в программе компилятор выделяет для хранения этой переменной участок памяти с уникальным адресом.

  2. Адрес переменной - это просто число, с которым можно обращаться как с обычным числовым значением.

  3. Указатель - это переменная, содержащая адрес некоторых данных, а не их значение, это группа ячеек (как правило, две или четыре), в которых может храниться адрес переменной, содержащей некоторое значение.

  4. Ссылка на значение посредством указателя называется косвенной адресацией.

  5. Значениями указателей служат адреса участков памяти, выделенных для объектов конкретных типов, поэтому в определении и описании указателя всегда присутствует обозначение соответствующего ему типа.

  6. C поддерживает две специальные адресные операции: 
    	& - операцию определения адреса (ссылка в языке C++ определена как другое имя уже существующего объекта),
	* - операцию раскрытия ссылки (операция: разыменования, обращения по адресу, доступа по адресу).

  7. Определение и описание указателя-переменной на некоторый объект имеют вид: 
    	тип *имя_указателя;

  8. Чтобы указатель указывал на определенную переменную, ему присваивается ее адрес с помощью операции взятия адреса &: 
    	имя_указателя = &имя_переменной;	// присвоить адрес переменной указателю

  9. При определении указателя целесообразно выполнить его инициализацию: 
    	тип *имя_указателя = инициализирующее выражение;
	тип *имя_указателя (инициализирующее выражение);

  10. Правило использования указателей: указатель всегда должен иметь адрес до своего использования в программе.

  11. Определение ссылки: 
    	тип &имя_ссылки инициализатор 

	тип &имя_ссылки = инициализирующее выражение;
	тип &имя_ссылки (инициализирующее выражение);
