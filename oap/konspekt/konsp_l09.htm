Потоковый ввод - вывод

  1. Поток - это просто последовательность байтов.

  2. Текстовые потоки состоят из символов, они организованы в виде строк до 255 символов длиной с завершающим символом конца строки.

  3. Двоичный поток может содержать данные любых видов, в том числе и текстовые, байты данных в двоичном потоке не интерпретируются никаким особым образом, а воспринимаются как есть.

  4. Механизм ввода-вывода заключается в пересылке байтов данных от устройств в оперативную память и обратно эффективным и надежным способом, как на "низком", так и на "высоком" уровнях.

  5. Ввод-вывод на низком уровне (библиотека #include<iostream.h> - неформатированный ввод-вывод) обычно сводится к тому, что некоторое число байтов данных просто следует переслать от устройства в память или из памяти в устройство.

  6. Ввод-вывод на высоком уровне (библиотека #include<iomanip.h> - форматированный ввод-вывод) - байты группируются в такие значащие элементы данных, как, например, целые числа, числа с плавающей запятой, символы, строки, а также данные типов, определенных пользователем.

  7. Объект стандартного потока вывода cout класса ostream выводит значение на экран дисплея, тип значения - любой.

  8. Вывод в потоки может быть выполнен с помощью перегруженной операции "поместить в поток" <<.

  9. Манипулятор потока endl (end line) вызывает переход на новую строку, приводит к сбросу буфера вывода.

  10. Объект стандартного потока ввода cin класса iostream вводит значение с клавиатуры, тип значения - любой.

  11. Операция ввода потока может быть выполнена с помощью перегруженной операции "взять из потока" >>, которая обычно игнорирует во входном потоке так называемые символы разделители или пробельные символы (пробелы, знаки табуляции, знак новой строки).
      int x, y;
      cout << "Введите два целых числа: "; cin >> x >> y;
      cout << "Сумма чисел " << x << " и " << y << " равна " << (x + y) << endl;
