Для конспекта

 

Видео режимы - моды

  1. Режимы экрана определяются при вводе в программу одной из функций: textmode, initgraph, setgraphmode.

  2. При textmode (текстовом режиме) экран делится на ячейки 80 или 40 колонок в ширину и 25 строк в высоту. Каждая ячейка состоит из атрибута и символа. Символ выводится на экран как ASCII символ, а атрибут показывает, как символ представлен на дисплее (цвет, интенсивность и так далее.).
    Верхний левый угол экрана - это позиция (1, 1), с x координатой, направленной слева направо, и y координатой, направленной сверху вниз.

  3. При graphics mode (графическом режиме) экран делится на точки растра (пиксели); каждая точка растра выглядит одной точкой на экране. Количество точек (разрешающая способность) зависит от типа видеоадаптера и установленного режима.
    Верхний левый угол - это позиция (0, 0) с x координатой, направленной слева направо, и y координатой, направленной сверху вниз.

  4. Функции графической библиотеки хранятся в библиотеке graphics.lib, а их прототипы - в заголовочном файле graphics.h.

  5. Графический пакет включает в себя также драйверы графических устройств (файлы *.bgi) и штриховых (векторные) символические шрифты (файлы *.chr).

  6. Обычная начальная последовательность действий при работе с графикой может быть следующей:
    1. Создать функцию собственной библиотеки init() для инициализации графики и сохранить ее в файле init.h в каталоге H - библиотеки. 
      void init()
{
	// Требование автоопределения
	int graphdriver = DETECT, graphmode;
	// Инициализация графики
	// Зайти в подкаталог CHR с файлами драйверов и шрифтов
	initgraph(&graphdriver, &graphmode, "..\\..\\chr");
	int error = graphresult();
	if(error != grOk)
	{	puts("Ошибка графики");
		puts(grapherrormsg(error));
		exit(1);
	}
}
    2. Подключить созданный библиотечный файл к необходимому рабочему файлу. 
      #include<graphics.h>
#include<stdlib.h>
#include<stdio.h>
#include<conio.h>
#include"..\\h\\init.h"
    3. Инициализировать графику вызовом функции init() и по окончании работы закрыть графический режим. 
      void main()
{	
	init();

	// ... - тело программы

	getch();
	closegraph();
}
