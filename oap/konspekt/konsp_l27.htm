Числовые массивы

  1. Массив - индексированный набор данных определенного типа.

  2. Все элементы массива имеют один и тот же тип и расположены в памяти подряд.

  3. Каждый элемент или значение в массиве является отдельной переменной и с ней можно обращаться как с обычной переменной, но так как они собраны в массив, их можно трактовать как общность.

  4. Определение одномерного массива: 
    	тип имя_массива [константное выражение];

    тип - тип данных элементов массива,

    константное выражение определяет размер массива - количество элементов массива (N).

  5. Общий объем памяти в байтах определяется выражением 
    	N * (sizeof(тип))

  6. В языке C указатели и массивы тесно связаны. Имя массива без квадратных скобок является указателем на первый элемент массива.

  7. Способы инициализации массива
    • присваиванием     lin[0] = 4.12;     lin[1] = 12.9;     lin[2] = 0;

    • списком, заключенным в скобки
      int days[4] = {31, 28, 31, 30};
      int a[3][4] = {{5, 3, -21, 42}, {44, 15, 0, 6}, {97 , 6, 81, 2}};

    • с клавиатуры 
      	printf("ввести 10 целых чисел\n");
	for(i = 0; i <= 9; i++)
		scanf("%d", &a[i]);

    • сгенерировав случайные числа, используя функции rand(), random(), randomize(), srand().

        int rand(void) - выдает псевдослучайное число в диапазоне от 0 до 32767.

        void srand(unsigned int seed) - использует seed в качестве "затравки" для новой последовательности псевдослучайных чисел. Изначальный параметр seed равен 1.

        int random(int num) - возвращает случайное число в диапазоне от 0 до num-1.
        random(num) - это макро, определенное в виде rand()%num.

        void randomize(void) - инициализирует генератор случайных чисел некоторым случайным числом.
