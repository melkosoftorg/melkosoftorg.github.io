Многомерные массивы

  1. В многомерном массиве используется сразу несколько индексов: двумерный массив имеет два индекса, трехмерный - три, четырехмерный - четыре, и так далее.

  2. Количество измерений массива в языке C не ограничено, но имеется ограничение на общий размер массива.

  3. Многомерный массив в соответствии с синтаксисом языка есть массив массивов, то есть массив, элементами которого служат массивы.

  4. Определение многомерного массива: 
    	тип имя_массива[K1][K2]...[Kn]

  5. Имя массива должно быть уникальным. Его нельзя использовать для другого массива, переменной, константы, функции и так далее.

  6. Трехмерный массив вещественных чисел: 
    	#define S 2
	#define K 3
	#define L 2
	float con[S][K][L];

    • имя - con,

    • размерность - 3 - трехмерный,

    • размер - количество элементов - равен S ∙ K ∙ L = 2 ∙ 3 ∙ 2 = 12,

    • объем памяти равен sizeof(con) = 12 ∙ 4 = 48 байт.

  7. Имя многомерного массива, как и одномерного, является указателем на первый элемент массива.

  8. Для n-мерного массива:

    • Имя массива с n парами квадратных скобок после него и с индексами в скобках имеет значение соответствующего элемента массива.

    • Имя массива, после которого стоит меньше чем n пар скобок, представляет собой указатель на элемент массива.

  9. Для целочисленного массива x с количеством строк S = 2, количеством столбцов K = 4 
    	const int S = 2, K = 4;
	// определен и инициализирован списком двумерный целочисленный массив
	int x[S][K] = {	{ 6436,  7975, 24729, 21687}, 
				{22008, 31997, 24949, 20980}
			 };

    Индекс
    строки    		 Индекс столбца
    j
    0123
    i 0 6436 7975 24729 21687
    Адрес:   4076 Адрес:   4078 Адрес:   4080 Адрес:   4082
    1 22008 31997 24949 20980
    Адрес:   4084 Адрес:   4086 Адрес:   4088 Адрес:   4090

  10. Квадратные скобки [ ] используются для записи каждого индекса.
    Элемент x[0][2], расположенный на пересечении 0 строки и 2 столбца, равен 24729;
    элемент x[1][3], расположенный на пересечении 1 строки и 3 столбца, равен 20980.

  11. Адрес массива x равен адресу массива x[0], и оба эти адреса равны адресу первого элемента массива x[0][0], то есть 4076.
