Массивы и строки

  1. Символьная строка - это последовательность, состоящая из одного или более символов.

  2. В языке C нет специального типа, описывающего строки.

  3. Строки представляются в виде массива элементов типа char. Символы в строке можно представить расположенными в соседних ячейках памяти - по одному символу в ячейке.

  4. Последним элементом массива является символ \0 - "нуль-символ", использующийся для обозначения конца строки.

    Над  кем   судьбина   не  шутила ?\0
    Каждая ячейка - 1 байт Нуль
    символ

  5. Способы объявления и инициализации строк:

    • объявление и инициализация массива размера N типа char: 
      	const int N = 30;
	char str[N] = "Инициализация не обязательна";
    • объявление и инициализация массива типа char: 
      	char str[] = "Начальное значение обязательно";
    • объявление и инициализация указателя на массив типа char: 
      	char *str = "Начальное значение обязательно";

  6. Командой    char str_1[20];
    • объявлен массив типа char из 20 символов;
    • идентификатор (имя) str_1 указывает на начало массива;
    • для массива выделено место в памяти, хотя и без инициализации, содержимое массива пока не определено.

  7. Командой    char *str_2;
    • объявлен указатель типа char;
    • объявление не выделяет для строки никакой памяти, только место для хранения указателя;
    • если необходимо создать строку и сделать так, чтобы str_2 на нее указывал, придется выделить память для строки используя соответствующие функции: malloc(), calloc() или операции, связанные с динамическим распределением памяти new и delete.

  8. Объявление и инициализация списком двумерного строкового массива 
    	const int N = 4, M = 30;
	char str[N][M] = {	"Русь былую, удалую",		// 0 строка - str[0]
				"Ты потомству передашь;",	// 1 строка - str[1]
				"Ты схватил ее живую",		// 2 строка - str[2]
				"Под народный карандаш."	// 3 строка - str[3]
			};

  9. Элемент массива str[1][5] имеет значение т, элемент массива str[3][18] имеет значение д.
