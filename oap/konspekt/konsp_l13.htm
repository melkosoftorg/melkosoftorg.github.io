Классы памяти и область их действия

  1. Ключевые слова, используемые для обозначения класса памяти auto, extern, static, register.

  2. Класс памяти определяет область действия переменной и продолжительность ее существования в памяти, устанавливается при описании переменной с соответствующим ключевым словом.

  3. Внешние переменные определены вне функции, имеют глобальную область действия, ключевое слово extern, используются функциями в любом файле.

  4. Внешние статические переменные определены вне функции, имеют глобальную область действия, ключевое слово static, используются функциями в данном файле.

  5. Локальные переменные определены в теле функции, известны только одной функции, той, которая их содержит, ключевое слово auto, область действия ограничена блоком { }.

  6. Если глобальная и локальная переменные имеют одинаковое имя, то, чтобы получить их значения, надо по-разному к ним обращаться, например, переменная i 
    #include<iostream.h>
#include<iomanip.h>
int i = 0;		// глобальная переменная
void main()
{	int i = 7;	// локальная переменная
	cout << "Глобальная ::i = " << ::i << endl 
		<< "Локальная  i = " << i << endl;
}

  7. Статические переменные определены в теле функции, известны только одной функции, той, которая их содержит, ключевое слово static, область действия ограничена блоком { }, не исчезают, когда содержащая их функция закончит свою работу.

  8. Автоматические переменные существуют временно пока работает функция, внутри которой они описаны, область действия локальная, ключевое слово auto.

  9. Регистровые переменные существуют временно пока работает функция, внутри которой они описаны, область действия локальная, ключевое слово register.
