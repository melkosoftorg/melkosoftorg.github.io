Функции ввода - вывода в языке программирования C

  1. Источники данных для ввода и места назначения вывода имеют собирательное название устройства.

  2. Весь ввод и вывод в программах на C выполняется через потоки независимо от того, откуда вводятся данные и куда они выводятся.

  3. Поток - это последовательность символов, последовательность байт данных.

  4. Поток ввода - это поток байт, принимаемый программой.

  5. Вводом называется процесс предоставления компьютеру информации, необходимой для работы программы.

  6. Функции, используемые для ввода данных, зависят от типа данных и способа их представления.

  7. Функции ввода подразделяются на три иерархических уровня:
    • функции ввода отдельных символов (getch, getchar, getche),
    • функции ввода строк (gets),
    • функции форматированного ввода (семейство ... scanf).

  8. Поток вывода - это поток, посылаемый программой на устройства.

  9. Выводом называется процедура переноса данных из памяти компьютера в другое место (на экран, отпечатать на принтере, сохранить на диске в виде файла).

  10. Функции, используемые для вывода данных, зависят от типа данных и способа их представления.

  11. Функции вывода на экран делятся на три категории:
    • функции вывода строк (puts) 
      	puts("Выводимую строку заключить в кавычки");
    • функции вывода символов (putchar, putch) 
      	putchar('Y');
    • функции форматированного вывода (семейство ... printf).
      Вывести вещественную переменную y с двумя десятичными знаками, поставив знак + перед выводимым числом, в поле шириной 10, выровняв по его левому краю. 
      	float y = 456.896;
	printf("y = %+-10.2f\n", y);

  12. Cтрока формата - символьная строка, состоящая из двух типов объектов:
    • просто символы (переносятся в вывод без изменений - y = ),
    • спецификации преобразования (применяются для форматирования - %+-10.2f).

  13. Спецификации формата имеют форму: %[флажки][ширина][.точность][F|N|h|l]тип
