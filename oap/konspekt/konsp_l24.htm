Вложенные циклы - цикл в цикле

  1. Если имеется два или более параметров, изменяющихся строго заданным образом, рационально для многократного повторения вычислений использовать вложенные циклы.

  2. Вложенные циклы - это конструкция, в которой внутрь одного цикла помещен другой цикл.

  3. При работе с вложенными циклами
    • Циклы нигде не должны пересекаться.
    • Внешний цикл всегда должен начинаться раньше и заканчиваться позднее.
      Управляющая переменная внешнего цикла изменяется медленнее.
    • Внутренний цикл всегда должен начинаться позднее и заканчиваться раньше.
      Управляющая переменная внутреннего цикла изменяется быстрее.

  4. Пример программы с вложенными циклами for.
    Целочисленная переменная x - управляющая переменная внешнего цикла - изменяется от 2 до 9 с шагом 1. Используется для изменения цвета выводимых символов и задает горизонтальную позиции вывода на экран.
    Целочисленная переменная y - управляющая переменная внутреннего цикла - изменяется от 3 до 9 с шагом 2. Задает вертикальную позиции вывода на экран значения y. 
    #include<conio.h>
void main()
{
	int x, y; clrscr();
	// Вложенные циклы
	for(x = 2; x < 10; x++)
	{
		textcolor(x);
		for(y = 3; y < 10; y += 2)
		{
			gotoxy(x, y);
			cprintf("%d", y);
		}
		getch();
	}
}
