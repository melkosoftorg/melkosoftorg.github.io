Массивы структур

  1. Как правило, программе недостаточно всего одного экземпляра данных структурного типа, то есть необходим массив структур, для чего надо:

    • описать структуру - объявление структурного типа; 
      	struct point
	{
		int x, y;
	};
    • задать количество элементов массива структур, например, 
      	const int N = 3;
    • любым из трех известных способов определить структурированные объекты, и, если необходимо, инициализировать значениями, например, 
      	struct point pt[N] = {2, 5,  -3, 6,  -4, -2};

      Элементы массива структур инициализированы следующими значениями: 

      	pt[0].x = 2		pt[0].y = 5 
	pt[1].x = -3		pt[1].y = 6 
	pt[2].x = -4		pt[2].y = -2

  2. Если структура содержит другие структуры в качестве полей, то начальные значения должны строго следовать порядку как структур, так и полей в их определениях. 
    	const int N = 2;
	struct customer
	{	char fio[20];
		char it[25];
	};
	struct sale
	{	struct customer ber;
		char tit[20];
		float at;
	} mysale[N] = {
				{ "Александр Блок", "Годы жизни 1880 - 1921" },
				"Гамаюн, птица вещая",
				167.87,

				{ "Афанасий Фет", "Годы жизни 1820 - 1892" },
				"Псовая охота",
				132.63
			};

    В этом фрагменте выполняются следующие инициализации:

    • поле mysale[0].ber.fio инициализируется значением Александр Блок
    • поле mysale[0].ber.it инициализируется значением Годы жизни 1880 - 1921
    • поле mysale[0].tit инициализируется значением Гамаюн, птица вещая
    • поле mysale[0].at инициализируется значением 167.87

    • поле mysale[1].ber.fio инициализируется значением Афанасий Фет
    • поле mysale[1].ber.it инициализируется значением Годы жизни 1820 - 1892
    • поле mysale[1].tit инициализируется значением Псовая охота
    • поле mysale[1].at инициализируется значением 132.63
