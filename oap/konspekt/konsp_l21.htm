Для конспекта

 

Множественный выбор

  1. Переключатель switch представляет собой структуру, построенную по принципу меню, содержит возможные варианты условий и инструкции, которые следует выполнить в каждом конкретном случае. 

  2. switch (выражение)
{       	
	case n1:	инструкции;
			break; 
	case n2:	инструкции;
			break;
	...
	case nn:	инструкции;
			break;
	default:	инструкции;
			break;
}

    switch - ключевое слово, обозначающее начало команды выбора,

    выражение - переменная типа int или char,

    n1, n2 ... nn - целочисленная константа или константное выражение, заключенное в апострофы; вычисления начинаются с той ветви case, в которой константа совпадает со значением выражения,

    break - оператор, используемый для завершения данной команды выбора и передачи программного управления на первый оператор после структуры switch,

    default - метка ветви, которая выполняется, если ни одна из констант не подходит.

  3. enum - ключевое слово, применяется для задания перечислимого типа данных и для объявления идентификаторов объектов перечислимого типа данных.
      typedef enum { sun, mon, tues, wed, thur, fri, sat } days;

  4. Присвоение осуществляется в порядке возрастания значений по умолчанию от sun = 0 до sat = 6.
