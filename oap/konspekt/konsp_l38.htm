Указатели на структуру

  1. Определение указателя на структуру имеет вид 
    		имя_структурного_типа *имя_указателя_на_структуру;

  2. Обращение к элементу структуры через указатель 
    		( *имя_указателя ) . имя_элемента
		имя_указателя -> имя_элемента

  3. Операции стрелка -> - операция косвенного выбора компонента (элемента) структурированного объекта, адресуемого указателя.

  4. Таким образом, для обращения к элементу структуры можно использовать

    • либо уточненное имя: 
      	имя_структуры.имя_элемента
    • либо конструкцию, включающую указатель: 
      	указатель_на_структуру -> имя_элемента
	(*указатель_на_структуру).имя_элемента

    • либо конструкцию, включающую ссылку: 
      	ссылка_на_структуру.имя элемента
