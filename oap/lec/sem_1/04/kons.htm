Пункты для конспекта (лекция 3)

  1. Прежде чем вводить данные, необходимо описать их тип.
  2. Для формирования имен констант, переменных, ключевых слов используются - ...
  3. Определить переменную - ...
  4. Определить константу - ...
  5. Можно дать константе некоторое имя, определив ее в качестве макроса препроцессора.
  6. Макросы определяются перед функцией main() директивой препроцессора #define (пример)
  7. Типизированные константы - ...
  8. const double Pi = 3.14159265;
  9. Целые числа - ...
  10. Восьмеричное число начинается с цифры 0
  11. Шестнадцатеричное число начинается с 0x
  12. unsigned - беззнаковые числа
  13. Суффикс L (или l), присоединенный к любой константе, сделает ее представление типа long
  14. Записать все диапазоны целых типов данных
  15. Числа, которые могут содержать десятичную часть (вещественные), называются числами с плавающей точкой.
  16. При записи вещественных чисел необходимо придерживаться правил: ...
  17. Записать все диапазоны вещественных типов данных
  18. Значением символьных данных (char) может быть буква, цифра или другой символ клавиатуры.
  19. При присваивании значения переменной типа char необходимо значение (символ) заключать в апострофы
  20. Записать все диапазоны символьных типов данных
  21. Управляющие последовательности (знать все из таблицы)
  22. Строкой называется набор символов, слов, фраз или предложений.
  23. Последовательность \0 является специальным символом, который C вставляет после каждой строки, чтобы отметить конец строки. \0 - нулевой символ, рассматривается как один символ, занимает один элемент памяти.
  24. Чтобы определить строковую переменную, необходимо использовать тип char и указать максимальное число символов с учетом \0, которое может содержать строка.
