Описание функции: классический стиль

Классический стиль на C, имеет следующий формат: 
		тип_результата имя_функции() ;
Эта спецификация описывает имя функции ("имя_функции") и тип возвращаемых ею значений ("тип").

Это описание не содержит никакой информации о параметрах функции, что не вызовет обнаружение ошибки компилятором или преобразования типов к типу, уже принятому контекстно, в соответствии с принятыми соглашениями о преобразовании типов.

Описания функций в этом стиле 

			void get_parms();
			float get_ratio();
			void put_ratio();


Определение функции: классический стиль

Классический формат определения функций имеет примерно следующий вид: 
		тип_результата имя_функции(имена_параметров)
		определение параметров;
		{
			локальные описания;
			операторы;
		}
Например, определение функции get_parms в классическом стиле выглядит: 
		void get_parms(p1, p2)
		float *p1; float *p2;
		{ ... }
