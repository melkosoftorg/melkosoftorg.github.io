Оператор return - опеpатоp возвpата в точку вызова

Инструкция return имеет одну из двух форм: 
	return;
	return выpажение;
Примеры: 
	return; 
	return 3; 
	return (a + b);
Скобки в выражении return не обязательны; некоторые программисты ставят их для удобочитаемости.

Выражение в операторе return определяет возвращаемое функцией значение. Именно это значение будет результатом обращения к функции. Тип возвращаемого значения определяется типом функции.

Существует два основных способа использования оператора return:

  1. В том случае, когда функция возвращает значение, и вам необходимо использовать его в зависимости от того, какое значение возвращается в вызывающую программу, например: 
    	// Найти наименьшее  из  двух  целых чисел
	#include<iostream.h>
	int  min(int  x,  int  y)
	{
		if(x < y)
			return  x; 
		else
			return  y;
	}
	void main()
	{
		int  j,   k,   m;
		cout   <<   "Введите  два  целых  числа:      "; 
		cin  >>   j   >>  k; 
		m  =  min(j,   k);
		cout  <<   '\n' << m <<  “ - наименьшее из “ <<  j <<   “ и “ <<  k << endl;
	}
    Здесь функция использует оператор return для возвращения минимального из двух переданных ей значений.

  2. Возможность выхода из функции в некоторой точке до ее завершения.
    Функция может определить условие, по которому производится прерывание. Вместо того чтобы помещать все основные операторы функции внутрь оператора if, для выхода можно использовать оператор return.

    Если функция имеет тип void (не возвращает никакого значения), можно написать return без возвращаемого значения. 

    		int imin(int list[], int size))
		{
			int i, minindx, min;
			if (size <= 0) return(-1);
			...
		}
    Если параметр size меньше либо равен нулю, то массив list пуст, в связи с чем, оператор return вызывает выход из функции. В случае ошибки возвращается значение -1. Так как -1 никогда не может быть индексом массива, вызывающая программа регистрирует факт возникновения ошибки.
