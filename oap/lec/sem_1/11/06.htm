Определение функции

Формат определения в современном стиле предусматривает определение параметров функции в скобках, следующих за "имя_функции": 
	тип_результата имя_функции (список параметров, если он есть)
	{
		декларации;
		инструкции;
	}
В этом примере определение списка параметров содержит всю информацию о передаваемом параметре: тип и идентификатор.

Собственные функции помещаются после закрывающей фигурной скобки main() (если использовать прототипы функций), или до функции main() (прототип в этом случае не пишется).

Для современного стиля программирования на C определение функции с использованием прототипа: 

		void get_parms(float *p1, float *p2)
		{ ... }

Каждая функция имеет структуру, аналогичную структуре функции main() (см. рис.): после имени функции ставится пара круглых скобок, точка с запятой в конце строки отсутствует, инструкции, составляющие тело функции, заключаются в пару фигурных скобок.

Ряд описаний (константы, типы данных, переменные), содержащихся внутри функции (кроме функции main), "видимы" или определены только внутри этой функции.

Поэтому C не поддерживает вложенность функций; то есть нельзя объявить одну функцию внутри другой.

Определения функций могут располагаться в любом порядке в одном или в нескольких исходных файлах, но любая функция должна быть целиком расположена в каком-то одном файле.
