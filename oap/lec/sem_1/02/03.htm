Язык C++ - объектно-ориентированный язык

Язык C++ восходит в своем развитии главным образом к C, который сохранен как подмножество.
Еще одним источником вдохновения послужил язык Simula67, из которого была позаимствована концепция класса вместе с производными классами и функциями членами.

Язык программирования C++ был разработан сотрудником научно-исследовательского центра AT&T Bell Laboratories (Нью-Джерси, США) Бьярном Страуструпом в 1979 году. С++ содержит в себе все, что есть в С. Но, кроме того, он поддерживает объектно ориентированное программирование (Object Oriented Programming, OOP). Изначально С++ был создан для того, чтобы облегчить разработку больших программ. Объектно ориентированное программирование это новый подход к созданию программ.

На первых этапах разработки язык носил условное название "Си с классами". Название C++ придумал Рик Масситти. Оно указывает на эволюционную природу перехода к нему от С.
++ - операция приращения в языке С.

Язык C++ был создан с учетом следующих целей:

  • улучшить язык C,
  • поддержать абстракцию данных,
  • обеспечить объектно-ориентированное программирование.
Плана разработки C++ на бумаге никогда не было. Проект, документация и реализация двигались одновременно. Разумеется, внешний интерфейс C++ был написан на C++.

В настоящее время идет работа по созданию единого международного стандарта по C++. Для этой цели был сформирован объединенный комитет (X3J16) по стандартизации ANSI (Американский национальный институт стандартов - American National Standards Institute) и ISO (Международная организация по стандартам - International Standards Organization) для языка C++.
Изданное в 1990 году описание языка с комментариями принято комитетом ANSI в качестве исходного материала для стандартизации C++.
С тех пор рабочей группой WG21 комитета ANSI выпущено несколько предварительных версий стандарта. Первый рабочий проект указанного стандарта был представлен 25 января 1994 года.

Вскоре после завершения работы над первым проектом Александр Степанов создал библиотеку стандартных шаблонов (Standard Template Library, STL), устанавливающую набор основополагающих процедур, используемых для обработки данных.

Весьма полная реализация соглашений по языку C++ выполнена в широко распространенных компиляторах Turbo C++ и Borland C++ фирмы Borland.
Количество компиляторов, соответствующих требованиям, предъявляемым стандартом языка C++, постоянно растет. К наиболее известным компиляторам относятся компиляторы фирм: Jensen & Partners, Microsoft, AT&T, Glockenspiel, Guidelines, Zortech.
