Пункты для конспекта (лекция 2)

  1. Язык С был разработан в 1972 г. Дэнисом Ритчи и Кеном Томпсоном в рамках разработки ОС UNIX.
  2. Язык программирования C++ был разработан Бьярном Страуструпом в 1979 году. Он поддерживает объектно-ориентированное программирование
  3. Сначала язык носил условное название "Си с классами". Название C++ придумал Рик Масситти
  4. Александр Степанов создал библиотеку стандартных шаблонов (Standard Template Library, STL)
  5. Функцией называется ...
  6. Все программы на Си++ должны начинаться с функции, называемой main().
  7. Последовательность инструкций, составляющих функцию, часто называют телом функции.
  8. Открывающая и закрывающая фигурные скобки { и } называются ограничителями и служат для выделения части кода в единый блок.
  9. Точка с запятой ; в языке C является разделителем и ...
  10. Библиотека - это ...
  11. Файлы заголовков (header file) или библиотеки обычно имеют расширение *.h
  12. Подключение библиотек осуществляется в самом начале программы.
  13. Строки исходного кода, начинающиеся со знака #, являются, как правило, директивами препроцессора, т.е. управляют обработкой текста программы еще до его передачи собственно компилятору. 
  14. 		#include <stdio.h> - искать файл библиотеки в каталоге INCLUDE
		#include "stdio.h"   - искать файл библиотеки в текущем каталоге
  15. Текст программы, записанный на С хранится в файле с расширением *.cpp
  16. Трансляция - ...
  17. Языки программирования интерпретирующего типа при исполнении программы за один проход переводят в машинные коды одну строку программы.
  18. Языки программирования компилирующего типа, напротив, сначала переводят весь текст программы в машинные коды, а затем полученный файл может быть запущен на выполнение.

  20. Объектный модуль - *.obj - ...
  21. Правила хорошего стиля программирования: ...
