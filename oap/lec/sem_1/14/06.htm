Закрытие файла, конец файла

Закрытие файла

После окончания записи в файл или чтения из файла необходимо его закрыть, то есть прервать связь между файлом и программой.

fclose - закрывает поток.
Синтаксис int fclose (FILE *str);
Прототип в stdio.h

fclose закрывает поток str, все буферы, связанные с потоком str, перед закрытием сбрасываются.
Буферы, размещенные системой, освобождаются во время процесса закрытия.
Буферы, назначенные функциями setbuf или setvbuf, не освобождаются автоматически. Однако если в качестве указателя передать функции sevtbuf 0, то он будет освобожден при закрытии потока.

fclose возвращает при успешном завершении значение 0.
Если были обнаружены какие-либо ошибки, функция fclose возвращает значение EOF.

Конец файла

feof - обнаруживает конец файла (EOF) в потоке.
Синтаксис int feof(FILE *str);
Прототип в stdio.h

feof является макрокомандой, которая производит проверку данного потока str на признак конца файла (EOF).
Если признак получил значение хотя бы один раз, операции чтения файла сохраняют это значение до тех пор, пока не будет вызвана функция rewind или файл не будет закрыт.

feof возвращает ненулевое значение, если при последней операции ввода потока str был обнаружен конец файла, и 0 в противном случае.
