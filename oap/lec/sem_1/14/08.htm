Построчное чтение данных из файла и запись в файл

fgets - получает строку символов из потока str и помещает ее в string. Ввод завершается при получении символа перехода на следующую строку.
Синтаксис char *fgets(char *string, int n, FILE *str);
Прототип в stdio.h

fputs - выводит строку символов в поток, копирует строку, ограниченную нулевым байтом в поток str, не добавляет в конец строки символ перехода на новую строку и не выводит нулевой символ.
Синтаксис int fputs(char *string, FILE *str);
Прототип в stdio.h

При успешном завершении возвращает последний выведенный символ, в противном случае возвращает EOF.

fread считывает n элементов данных, каждый длиной size байтов, из потока str в блок с адресной ссылкой ptr. Общее число вводимых байт равно n ∙ size.
Синтаксис size_t fread(void *ptr, size_t size, size_t n, FILE *str);
Прототип в stdio.h При успешном завершении функция возвращает количество элементов данных (не байтов), реально прочитанных. В случае достижения конца файла или возникновения ошибки возвращает short count (возможно 0).

fwrite записывает данные в поток, добавляет n элементов данных, каждое величиной size байт в данный выходной поток. Данные записываются из ptr. Общее число выведенных байт равно n ∙ size. ptr должен быть объявлен как указатель на некоторый объект.
Синтаксис int fwrite(void *ptr, int size, size_t n, FILE *str);
Прототип в stdio.h

При успешном завершении fwrite возвращает число выведенных элементов (не байт), при ошибке возвращает меньшее число. 

	#include<stdio.h>
	#include<string.h>
	#include<conio.h>
	void main()
	{
		FILE *str; char ch, msg[] = "Тестовый пример";
		str = fopen("dum.fil", "a+");	// создать файл для изменения 
		fwrite(msg, strlen(msg), 1, str);	// записать в файл данные 
		fseek(str, 0, SEEK_SET);	// перейти в начало файла 
		do
		{
			ch = fgetc(str);	// ввести символ из файла 
			putch(ch);		// вывести символ на экран 
		} while( !feof(str) );		// пока не конец файла
		printf("Конец файла\n");
		fclose(str);		// закрыть файл
		getch();
	}
