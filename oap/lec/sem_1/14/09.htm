Форматированный ввод и вывод

fscanf - выполняет форматированный ввод символов, строк, чисел из потока.
Синтаксис int fscanf(FILE *str, char *format [ , adress, ... ] );
Прототип в stdio.h

fscanf сканирует посимвольно набор вводимых полей, считывая их из потока.
Затем каждое поле из потока форматируется в соответствии со спецификацией формата, передаваемой в виде указателя на строку format, и запоминается в аргументах, передаваемых после параметра format.

Количество аргументов должно совпадать с количеством спецификаций формата. Описание спецификаций формата приведено в описании функции scanf.

fscanf завершает сканирование отдельного поля при появлении пробела, являющегося разделителем, или при вводе поля, для которого указана ширина.
Возвращает количество успешно прочитанных, преобразованных и запомненных полей.
Возвращаемое значение не включает в себя прочитанные поля, которые не были сохранены.
Если fscanf делает попытку чтения в конце файла, то возвращается EOF.
Если не было записано ни одного поля, возвращается значение 0.

fprintf посылает форматированный вывод в поток, получает набор аргументов, по одному для каждой спецификации формата, и выводит данные в поток.

Количество аргументов должно совпадать с числом спецификаций. Описание спецификаций формата приведено в функции printf.

Синтаксис int fprintf(FILE *str, char *format [,argument,...]);
Прототип в stdio.h

Функция возвращает число выведенных байт.
При появлении ошибки возвращает EOF.

Приведенные ниже программы записывает в файл данные разных типов. 

	#include<stdio.h>
	#include<conio.h>
	void main()
	{
		FILE *str;
		int i = 100; float f = 1.239; char c = 'g'; clrscr();
		// открыть файл для изменения 
		str = fopen("dum.fil", "w+");	
		fprintf(str, "%10d %5c %7.2f", i, c, f);	// вывести в файл данные 
		fclose(str);			// закрыть файл 
		getch();
	}
