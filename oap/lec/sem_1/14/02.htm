Указатель на файлы

Ввод или вывод информации в файлы обеспечивается с помощью так называемого указателя на файлы, который является указателем на файловую структуру в памяти. При записи информации в файл или при чтении из файла программа получает необходимую информацию из структуры. 
Определение указателя на файл:	 	FILE *fp;
Имя структуры FILE говорит программе о том, что определяемая переменная fp является указателем именно на файловую структуру.

Если планируется использовать одновременно несколько файлов, нужны указатели для каждого из них.

Например:       FILE *fp, *kt, *dr;

Два указателя требуются, например, в том случае, если необходимо прочитать информацию с диска и распечатать ее на принтере.
