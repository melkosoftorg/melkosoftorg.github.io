Маршрут поиска файла

searchpath - просматривает маршрут в поисках файла.
Синтаксис char *searchpath(char *filename);
Прототип в dir.h

searchpath делает попытку найти файл с именем filename, используя маршрут MS-DOS, определяемый в переменной окружения PATH.

Указатель на полное имя маршрута файла возвращается в качестве значения функции.
Сначала просматривается текущая директория рабочего диска.
Если файл там не найден, вызывается переменная операционной среды PATH и каждая директория в маршруте PATH просматривается до тех пор, пока файл не будет найден или маршрут не будет исчерпан.

Как только файл будет найден, возвращается строка, содержащая полное имя маршрута файла. Эта строка затем может быть использована в вызовах функций open или exec... для получения доступа к файлу.
Возвращаемая строка с именем файла помещается в static буфер и разрешается при каждом последующем вызове функции searchpath.

Если файл успешно найден, возвращается указатель на строку filename, в противном случае, функция searchpath возвращает значение NULL. 

		char *ch= "BC.EXE", *p;
		p = searchpath(ch); printf("Маршрут для %s: %s\n", ch, p);
