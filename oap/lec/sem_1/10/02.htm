Основания чисел различных систем счисления

Манипуляторы потоков dec, oct, hex и setbase задают основание чисел, требуют включения заголовочного файла iomanip.h.

Целые числа обычно интерпретируются как десятичные (с основанием 10).

Для изменения основания интерпретации целых чисел в потоке используется

  • манипулятор hex, чтобы установить шестнадцатеричный формат представления элементов данных (с основанием 16),
  • манипулятор oct, чтобы установить восьмеричный формат представления данных (с основанием 8),
  • манипулятор dec для возврата к основанию потока 10.

Основание потока может быть изменено с помощью манипулятора потока setbase, который принимает один целый параметр со значениями 10, 8 или 16, задающими соответствующие основания системы счисления.
Поскольку манипулятор setbase принимает параметр, он называется параметризованным манипулятором потока. 

	#include<iostream.h>
	#include<iomanip.h>
	void main()
	{
		int n;
		cout << "Десятичное целое число "; cin >> n;
		cout << "\n16 система счисления " << hex << n
			<< "\n 8 система счисления " << oct << n
			<< "\nдесятичное целое число " << dec << n
			<< endl << setbase(8) << n << endl;
	}
	Результаты:	Десятичное целое число 3456
			16 система счисления       d80
			8 система счисления       6600
			десятичное целое число 3456
			6600
