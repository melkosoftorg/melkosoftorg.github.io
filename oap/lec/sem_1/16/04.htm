Линии

line вычерчивает линию между двумя заданными точками(x0,y0) и (x1,y1), причем текущая позиция не изменяется.
lineto вычерчивает прямую линию от текущей позиции до (x, y), затем устанавливает текущую позицию в (x, y).
linerel вычерчивает прямую линию от текущей позиции до точки, отстоящей от нее на расстояние (dx, dy). Текущая позиция наращивается на (dx, dy). 
Синтаксис	void far line(int x0, int y0, int x1, int y1);
		void far lineto(int x, int y);
		void far linerel(int dx, int dy);

Каждая из функций вычерчивает линию текущего цвета, используя при этом текущий стиль и толщину линий. 

	int x1, y1, x2, y2, x3, y3, x, y, dx, dy;
	// линия из точки (20, 150) в точку (120, 50)
	x1 = 20; y1 = 150; x2 = 120; y2 = 50;
	line(x1, y1, x2, y2);

	// линия из точки (400, 300) в точку (600, 420)
	x = 400; y = 300; x3 = 600; y3 = 420;
	moveto(x, y); lineto(x3, y3);

	// линия из точки (400, 300) в точку (400+200, 420+(-100))
	dx = 200; dy = -100;
	moveto(x, y);
	linerel(dx, dy);

	// линия из точки (400+200, 420+(-100)) в точку (20, 150)
	lineto(x1, y1);
