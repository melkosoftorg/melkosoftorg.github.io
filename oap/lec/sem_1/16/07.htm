Многоугольники и прямоугольники

rectangle вычерчивает прямоугольник, используя текущие стиль линии, толщину и цвет вычерчивания. 

Синтаксис     void far rectangle(int left, int top, int right, int bottom);

(left, top) - левый верхний угол прямоугольника, (right, bottom) - правый нижний угол прямоугольника.

bar вычерчивает не окантованную заполненную с использованием текущего шаблона и цвета заполнения прямоугольную полосу. 

Синтаксис        void far bar(int left, int top, int right, int bottom);

Программный код для построения прямоугольников. 

        setbkcolor(15); setcolor(1); setfillstyle(9, 4);
        rectangle(50, 50, 80, 60);
        bar(50, 70, 80, 80);

Например, чтобы получить изображение такого робота, можно обойтись одними закрашенными прямоугольниками. 

#include<graphics.h>
#include<conio.h>
#include<stdio.h>
#include<stdlib.h>
#include<dos.h>
#include"..\\h\\init.h"
main()
{
	init();
	setbkcolor(0);
	// голова
	setfillstyle(1, 11); bar(200, 40, 300, 180);
	bar(180, 80, 200, 140); bar(300, 80, 320, 140);
	// глаза
	setfillstyle(1, 4);	
	bar(220, 60, 240, 80); bar(260, 60, 280, 80);
	// нос
	setfillstyle(1, 8); bar(240, 100, 260, 120);
	// рот
	setfillstyle(6, 7); bar(220, 140, 280, 160);
	// туловище
	setfillstyle(8, 11);
	bar(220, 180, 280, 200); bar(200, 200, 300, 340);
	// руки и ноги
	setfillstyle(1, 11);
	bar(140, 200, 220, 240); bar(280, 200, 360, 240);
	bar(190, 330, 230, 380); bar(270, 330, 310, 380);
	getch(); closegraph();
}

bar3d вычерчивает трехмерную прямоугольную полосу с дальнейшим заполнением ее текущим шаблоном и цветом заполнения. 

Синтаксис        void far bar3d(int left, int top, int right, int bottom, int depth, int topflag);
Верхний левый и нижний правый углы прямоугольника задаются параметрами (left, top) и (right, bottom) соответственно.

Глубина полосы в точках растра задается параметром depth. Для получения окантованной двумерной полосы используйте bar3d с depth = 0.

Трехмерная окантовка полосы рисуется текущими стилем и цветом линии.
Параметр topflag регулирует необходимость вычерчивания трехмерной вершины полосы.
Если topflag не ноль, то вершина вычерчивается; в противном случае нет, делая возможным установку нескольких полос одна над другой. 

   Рисунок иллюстрирует использование функции bar3d

        setbkcolor(WHITE); setcolor(GREEN);
        bar3d(230, 50, 250, 150, 15,1); 
        bar3d(220, 150, 260, 180, 15, 1);
        bar3d(300, 150, 340, 180, 15, 0); 
        bar3d(300, 50, 340, 150, 15, 1);

Для вычисления типичной глубины трехмерной полосы берется 25% ширины полосы, например: bar3d(left, top, right, bottom, (rightbottom) / 4, 1);

drawpoly вычерчивает многоугольник с numpoints вершинами, используя для этого текущие цвет и стиль линии.
fillpoly вычерчивает многоугольник, а затем заполняет его, используя для этого текущие цвет и стиль заполнения. 

Синтаксис        void far drawpoly(int numpoints, int far *polypoints);
                void far fillpoly(int numpoints, int far *polypoints);

polypoints указывает на последовательность (длиной numpoints*2) целых чисел. Каждая пара из этих чисел задает координаты x, y вершины многоугольника.

Построение многоугольников произвольной формы и их заполнение 

   int poly[10];
   // координаты вершин четырехугольника
   poly[0] = 20; poly[1] = 20;          // первая
   poly[2] = 90; poly[3] = 30;          // вторая
   poly[4] = 150; poly[5] = 90; // третья
   poly[6] = 60; poly[7] = 70;          // четвертая
   poly[8] = poly[0]; poly[9] = poly[1];
   drawpoly(4, poly);   // построение многоугольника
   // построение многоугольника и его заполнение
   for (int i = EMPTY_FILL; i < USER_FILL; i++)
   {    // i - стиль заполнения, 9 - цвет
        setfillstyle(i, 9);
        fillpoly(4, poly);
        getch();
   }

Замечание: Для того, чтобы начертить замкнутую фигуру с n вертикалями, надо передать в drawpoly n + 1 координату, где каждая n-ая координата равна 0.
В случае возникновения ошибки при заполнении многоугольника graphresult вернет величину -6.
