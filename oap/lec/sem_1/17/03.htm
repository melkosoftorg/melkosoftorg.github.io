Размер шрифта

setusercharsize задает пользовательский коэффициент увеличения символов для штриховых шрифтов

Синтаксис void far setusercharsize(int multx, int divx, int multy, int divy);

setusercharsize дает возможность управлять размером знаков из штриховых шрифтов.
Значения, установленные setusercharsize, действительны только в случае, когда предыдущий вызов settextstyle установил charsize = 0. С помощью setusercharsize можно задать коэффициент масштабирования ширины и высоты символов.
По умолчанию ширина вычисляется как multx : divx, высота как multy : divy.

Например, чтобы сделать текст вдвое шире и на 50% выше, чем он принят по умолчанию, нужно задать: 

		multx = 2; divx = 1; multy = 3; divy = 2;
textheight возвращает высоту строки в точках растра, используя для этого размер текущего шрифта и коэффициент увеличения.
textwidht возвращает ширину строки в точках растра, используя для этого размер текущего шрифта, длину строки и коэффициент увеличения. 
Синтаксис	int far textheight(char far *textstring);
		int far textwidht(char far *textstring);
Функции полезны для установки расстояния между строками, вычисления ширины и высоты графического типа, определения размеров заголовка для размещения его в графе или рамке и тому подобное.

Например, для шрифта 8 · 8 точек растра и коэффициента усиления 1 (установленного функцией settextstyle) строка Borland C имеет 8 точек растра в высоту и 48 точек в ширину.

Очень важно для вычисления высоты и ширины строк использовать функции textwidht и textheight, а не производить их самостоятельно. Использование функций позволяет исключить модификацию программы при работе с различными шрифтами
