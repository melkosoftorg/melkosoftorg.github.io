Window - меню окон
Вызов: Alt + W

Меню Window (Окно) содержит команды управления окнами. Большая часть окон, которую открываете из этого меню, содержит все стандартные элементы окна, в число которых входят полосы прокрутки, маркер закрытия окна и маркеры распахивания окна на весь экран.
У нижнего края меню Window (Окно) появляется команда Window\List (Окно\Список). Для того чтобы получить список всех открытых окон, а также недавно закрытых окон, следует выбрать эту команду. Информация о недавно закрытом окне появляется со словом closed (закрыто) перед именем этого окна. Чтобы повторно открыть это окно, его необходимо выбрать.

Меню Window (Окно) позволяет:
  • изменить размер или местоположение активного окна,
  • установить порядок расположения окон,
  • закрывать окна,
  • открывать окно сообщений,
  • открывать окно выводимых данных,
  • открывать окно выражений просмотра,
  • открывать окно выводимых программой данных,
  • открывать окно регистров,
  • открывать окно проектов и примечаний к ним,
  • выводить список открытых файлов.

 

Команда Size/Move (Изменить размер/Переместить)
Вызов: Ctrl + F5

Команда Window\Size/Move (Окно\Изменить размер/Переместить) используется для того, чтобы изменить размер или местоположение активного окна.

При выборе этой команды в ответ на нажатие клавиш управления курсором происходит перемещение активного окна. После того, как окно заняло требуемое положение, следует нажать клавишу Enter.
Перемещение окна может также осуществляться с помощью буксировки его строки заголовка.

Если в момент нажатия клавиши управления курсором будет удерживаться в нажатом состоянии клавиша Shift, размер окна будет изменяться.
Когда окно приняло необходимый размер, следует нажать клавишу Enter.
Если у данного окна имеется угол изменения размера, то для изменения размера этого окна можно отбуксировать этот или любой другой угол.

 

Команда Zoom (Распахнуть окно на весь экран)
Вызов: F5

Команда Window\Zoom (Окно\Распахнуть окно на весь экран) используется для изменения размера окна, в результате чего оно получает максимально допустимый размер.
Если это окно уже распахнуто до максимально возможного состояния, то выбор данной команды приводит к восстановлению его предыдущего размера.
Для того чтобы распахнуть окно на весь экран или вернуть его к предыдущему состоянию, можно также подвести мышь к произвольному месту верхней строки (за исключением мест размещения пиктограмм) и дважды подряд нажать кнопку мыши.

 

Команда Cascade (Каскадное расположение)

Команда Window\Cascade (Окно\Каскадное расположение) располагает все открытые окна каскадом (уступами), чтобы край последующего окна выступал бы из-под предыдущего.

 

Команда Tile (Мозаичное расположение)

Команда Window\Tile (Окно\Мозаичное расположение) располагает все открытые окна таким образом, чтобы они были одновременно видны и не перекрывали одно другое.

 

Команда Next (Следующее)
Вызов: F6

Команда Window\Next (Окно\Следующее) делает активным следующее окно, в результате выбранное окно расположится на самой вершине стека окон.

 

Команда Close (Закрыть)
Вызов: Alt + F3

Команда Window\Close (Окно\Закрыть) используется для того, чтобы закрыть активное окно.
Для закрытия окна можно также подвести указатель мыши к маркеру закрытия окна в верхнем левом углу окна и нажать кнопку мыши.

 

Команда Close all (Закрыть все)

Команда Window\Close all (Окно\Закрыть все) используется для того, чтобы закрыть все открытые окна.

 

Команда Message (Сообщение)

Команда Window\Message (Окно\Сообщение) используется для того, чтобы открыть окно Message (Сообщение) и сделать его активным.
В окне Message (Сообщение) отображаются сообщения об ошибках и предупреждающие сообщения, которые можно использовать для справки.
Кроме того, можно выбрать одно из этих сообщений, в результате чего в окне редактирования соответствующее место будет выделено световым маркером.
Если сообщение относится к файлу, который в настоящий момент не загружен в память, то для загрузки этого файла можно нажать клавишу пробела. В этом окне можно также отобразить данные, которые выводятся программой переноса.

Если в окне Message (Сообщение) выбирается какое-либо сообщение об ошибке, то для отображения местоположения ошибки в окне редактирования следует нажать клавишу Enter, и окно редактирования станет текущим, и в нем будет отображена точка возникновения ошибки.

Для того чтобы закрыть это окно, следует подвести указатель мыши к маркеру закрытия окна и нажать кнопку мыши, либо выбрать команду Window\Close (Окно\Закрыть).

 

Команда Output (Выводимые данные)

Команда Window\Output (Окно\Выводимые данные) используется для открытия окно Output (Выводимые данные) и его активизирования. Оно отображает текст любой введенной в операционной системе командной строки, а также текстовую выдачу, которая генерируется программой.
Графические данные не отображаются.

Команда Output (Выводимые данные) удобна при отладке, поскольку позволяет одновременно просматривать исходный код, переменные и выводимые данные. Эта возможность особенно полезна в том случае, если в блоке диалога Options\Environment (Параметры\Среда работы) устанавливается режим отображения 43/50 строк, и осуществляется отладка стандартной программы, которая работает с 25-строчным режимом отображения. В результате можно видеть практически все выводимые программой данные, и в то же самое время иметь достаточное число строк, чтобы просматривать исходный код и значения переменных.

Если хотите видеть выводимые программой данные в полноэкранном режиме - или если программа работает с графикой - выберите вместо этой команды команду Window\User Screen (Окно\Экранный кадр пользователя).

Для того чтобы закрыть это окно, следует подвести указатель мыши к его маркеру закрытия и нажать кнопку мыши или выбрать команду Window\Close (Окно\Закрыть).

 

Команда Watch (Выражение просмотра)

Команда Window\Watch (Окно\Выражение просмотра) используется для того, чтобы открыть окно Watch (Выражение просмотра) и сделать его активным. Оно отображает выражения и их изменяющиеся значения, поэтому можно следить за тем, как программа осуществляет вычисление значений ключевых переменных.
Эта команда используется во всплывающих меню Debug\Watches (Отладка\Выражения просмотра) для добавления или удаления из окна выражений просмотра.

 

Команда User Screen (Экранный кадр пользователя)
Вызов: Alt + F5

Команда Window\User Screen (Окно\Экранный кадр пользователя) используется для просмотра выводимых программой данных в полноэкранном режиме.
Если предпочитаете видеть выводимые программой данные в окне системы Borland C++, то вместо данной команды следует выбрать команду Window\Output (Окно\Выводимые данные). При выборе данной команды нажатие любой клавиши или кнопки мыши приводит к возврату в интегрированную среду.

 

Команда Register (Регистры)

Команда Window\Register (Окно\Регистры) используется для того, чтобы открыть и сделать активным окно Register (Регистры). Оно отображает содержимое регистров центрального процессора и используется при отладке встроенных ассемблерных модулей и модулей Турбо Ассемблера проекта.

Для того чтобы закрыть это окно, следует подвести указатель мыши к маркеру закрытия окна и нажать кнопку мыши, либо выбрать команду Window\Close (Окно\Закрыть).

 

Команда Project (Проект)

Команда Window\Project (Окно\Проект) используется для открытия окна Project (Проект), которое позволяет просмотреть файлы, используемые для создания программы.

 

Команда Notes (Примечания)

Команда Window\Notes (Окно\Примечания) используется для записи каких-либо подробностей, создания перечней того, что следует сделать, или иных деталей, которые касаются файлов проекта.

 

Команда List all... (Перечень)
Вызов: Alt + 0

Команда Window\List all... (Окно\Перечень) используется для получения перечня всех окон, которые были открыты. Этот перечень содержит имена всех файлов, которые открыты в настоящий момент, а также последних восьми файлов, которые были открыты в окне редактирования, но в последствие были закрыты.
Закрытые файлы появляются в перечне с префиксом closed (закрыт) перед именем файла.

Если выбираете из этого перечня имя файла, который к настоящему моменту открыт, система Borland C++ переносит окно с этим файлом на вершину стека и делает это окно активным.
Если выбираете из этого перечня имя закрытого файла, система Borland C++ повторно открывает этот файл в окне редактирования. Размер и позиция окна этого файла будут точно такими же, как и на момент закрытия окна. Курсор будет позиционирован туда, где он располагался при закрытии окна.

Для выдачи на экран полного перечня всех открытых окон и всех закрытых окон редактирования можно нажать комбинацию клавиш Alt + 0.
