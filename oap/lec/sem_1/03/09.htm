Project - меню проекта
Вызов: Alt + P

Меню Project (Проект) содержит все команды управления проектом, которые необходимы для того, чтобы:

  • загрузить имеющийся или создать новый проект,
  • удалить проект,
  • добавить или удалить из проекта файлы,
  • включить параметры переопределения командной строки, выбрать транслятор,
  • просмотреть включаемые файлы.

Можно посмотреть информацию о том, как управлять мультифайловыми проектами.

 

Команда Open Project (Открыть проект)

Команда Open Project (Открыть проект) отображает блок диалога Load Project File (Загрузить файл проекта), который позволяет выбрать и загрузить какой-либо проект или создать новый проект, введя его имя.
Этот блок диалога позволяет выбрать имя файла аналогично тому, как это делается в блоке диалога File\Open (Файл\Открыть).
Выбранный файл будет использоваться в качестве файла проекта.

Файлом проекта является файл, который содержит всю информацию, необходимую для полной компиляции и компоновки выполняемого файла проекта. Система Borland C++ использует имя файла проекта при создании файлов с расширениями имен *.EXE и *.MAP.
Обычно файл проекта получает расширение имени *.PRJ.

 

Команда Close Project (Закрыть проект)

Команду Project\Close Project (Проект\Закрыть проект) следует использовать, когда хотите удалить проект и вернуться к используемому по умолчанию проекту (TCDEF.DPR).

 

Команда Add Item (Добавить элемент)

Команду Project\Add Item (Проект\Добавить элемент) следует использовать, когда хотите добавить к списку проекта какой-либо файл.
В результате выбора этой команды на экране появляется блок диалога Add Item to Project List (Добавить элемент к списку проекта). Данный блок диалога во многих отношениях аналогичен блоку диалога Load File (Загрузить файл) (команда File\Open (Файл\Открыть)).

Выбор кнопки Add (Добавить) приводит к занесению имени выделенного в настоящий момент файла в список Files (Файлы) окна Project (Проект).
Имя выбранного файла добавляется в список File (Файл) окна Project (Проект) в строку, которая расположена непосредственно за световым маркером окна Project (Проект).
Световой маркер продвигается каждый раз после добавления какого-либо файла. (Если окно Project (Проект) находится в активном состоянии, то для добавления файла можно нажать клавишу Ins.)

 

Команда Delete Item (Удалить элемент)

Команду Project\Delete Item (Проект\Удалить элемент) следует использовать, когда хотите удалить имя какого-либо файла в окне Project (Проект).
Если окно Project (Проект) находится в активном состоянии, то для удаления файла можно нажать клавишу Del.

 

Команда Local Options (Локальные параметры)

Команда Local Options (Локальные параметры) открывает блок диалога Override Options (Переопределение параметров).

Важное замечание: Не поддерживаются следующие параметры командной строки: Q, y, E, M, c, e, l, L и I.

Блок диалога Override Options (Переопределение параметров) позволяет включить параметры переопределения командной строки для конкретного модуля, входящего в состав файла проекта. Он позволяет также задать объектному файлу конкретное маршрутное имя (идентификатор диска + имя директории) а также само имя этого файла. Кроме того, можно выбрать для данного модуля определенный транслятор.

Все программы, которые вы инсталлировали в блоке диалога Transfer (Перенос) с "промаркированными" триггерными кнопками Translator (Транслятор), появляются в списке Project File Translators (Трансляторы объектных файлов в проекте).

[ ] Overlay this module (Сделать данный модуль оверлейным)
"Промаркируйте" параметр Overlay this module (Сделать данный модуль оверлейным), если хотите, чтобы выбранный модуль или библиотека (или элемент проекта) были бы оверлейными. Данный параметр является локальным; он распространяется только на один файл. Он будет "запрещен", если не промаркирована триггерная кнопка Overlay support (Поддержка оверлейных модулей) (расположенная в меню Options\Compile\Code Generation (Параметры\ Компиляция\ Генерация кода)).

[ ] Exclude debug information (Исключить отладочную информацию)
"Промаркируйте" параметр Exclude debug information (Исключить отладочную информацию), чтобы предотвратить перенесение отладочной информации, внесенной в выбранный модуль, в результирующий выполняемый (EXE) файл. Эту триггерную кнопку следует использовать для уже отлаженных модулей большой программы. Можно изменить те модули, в которых будет содержаться отладочная информация, если просто "промаркировать" эту кнопку и перекомпоновать программу (никакой компиляции не требуется).

[ ] Exclude from link (Исключить из компоновки)
"Промаркируйте" параметр Exclude from link (Исключить из компоновки), если не хотите, чтобы этот модуль прикомпоновывался бы к результирующей программе.

 

Команда Include Files (Включаемые файлы)

Команду Project\Include Files (Проект\Включаемые файлы) следует использовать для отображения блока диалога Include Files (Включаемые файлы); она нужна в случае, когда хотите увидеть, какие файлы включены в тот файл, который выбрали из окна Project (Проект).
Когда находитесь в окне Project (Проект), для отображения блока диалога Include Files (Включаемые файлы) можете нажать клавишу пробела. Если еще не создали проект, эта команда будет запрещена.

После того, как файл был скомпилирован, об этом файле собирается информация (заметьте, что в окне Project (Проект) содержится информация о размере кода). В этом состоянии менеджеру проектов известно также, к каким включаемым файлам обращается данный модуль. Можно увидеть имена включаемых файлов для модуля, расположенного в активном окне редактирования, с помощью блока диалога Include Files (Включаемые файлы).
Если находитесь в окне менеджера проектов, то для обращения к этому блоку диалога следует нажать клавишу пробела.
Если находитесь в окне редактирования, то для обращения к этому блоку диалога следует перейти к меню Project (Проект) и выбрать команду Include Files (Включаемые файлы).
В списке отображенных файлов можно осуществлять прокрутку. Стандартным (выполняемым по умолчанию) действием является просмотр выбранного файла, поэтому нажатие клавиши Enter приводит к открытию указанного включаемого файла в окне редактирования.
