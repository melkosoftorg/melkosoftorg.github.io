Edit - меню редактирования
Вызов: Alt + E

Прежде чем использовать большинство команд в меню Edit (Редактирование), необходимо получить представление о выборе текста, поскольку большая часть выполняемых в редакторе действий относится к выбранному фрагменту текста.
Выбрать текст означает выделить его цветовым (или световым) атрибутом.
Можно выбрать текст либо с помощью клавиатурных команд, либо с помощью мыши; принципы одинаковы, хотя действия различны.

При работе с клавиатурой можно воспользоваться одним из следующих методов:

  • нажмите клавишу Shift с одновременным нажатием клавиш управления курсором (стрелки вверх, вниз, влево, вправо, Home, End, Page Up, Page Down);
  • нажмите комбинацию клавиш Ctrl + K B, чтобы пометить начало выделяемого блока. Затем переместите курсор в конец фрагмента текста и нажмите комбинацию клавиш Ctrl + K K.
  • для выбора слова необходимо переместить курсор к данному слову и нажать комбинацию клавиш Ctrl + K T
  • для выбора строки в целом, необходимо нажать комбинацию клавиш Ctrl + K L.

При работе с мышью:

  • для выбора фрагмента текста отбуксируйте указатель мыши по нужному вам тексту. Если необходимо продолжить выделение за границами окна, необходимо лишь отбуксировать указатель мыши за границу окна, и в окне автоматически начнется прокрутка;
  • для выбора одной строки необходимо подвести указатель мыши в произвольное место этой строки и дважды подряд нажать кнопку мыши;
  • для построчного выбора текста необходимо нажать и отпустить кнопку мыши, а затем отбуксировать указатель мыши (то есть один раз кратковременно нажать и отпустить кнопку мыши, а затем практически без паузы повторно нажать кнопку мыши и, не отпуская ее, начать буксировку указателя мыши).
  • для того, чтобы уменьшить или увеличить выделенный фрагмент текста, необходимо нажать и удерживать нажатой клавишу Shift и в то же самое время кратковременно нажать кнопку мыши в каком-либо месте документа.

После того, как выделен необходимый фрагмент текста, становятся доступными расположенные в меню Edit (Редактирование) команды, и можно использовать текстовый буфер (Clipboard).
Текстовый буфер предназначен для отсечения и вклеивания фрагментов текста. Он представляет собой специализированное окно системы Borland C++, хранящее тот фрагмент текста, который отсекли или скопировали, и который можете теперь вклеить куда-либо еще.
Текстовый буфер действует в тесном взаимодействии с командами меню Edit (Редактирование).

Меню Edit (Редактирование) позволяет выполнять
  • отмену и восстановление изменения,
  • отсечение выделенного фрагмента текста,
  • копирование выделенного фрагмента текста,
  • вклейку выделенного фрагмента текста,
  • удаление выделенного фрагмента текста,
  • копирование примеров, приведенных в справочной системе,
  • просмотр и редактирование содержимого текстового буфера.

 

Команда Undo (Назад на шаг - отменить изменение)
Вызов: Alt + Bsp

Последовательное применение данной команды Edit\Undo позволяет вернуться на предыдущее действие и восстановить исправленные строки.

Команда Redo (Вперед на шаг - восстановить изменение)
Вызов: Shift + Alt + Bsp

Последовательное применение данной команды Edit\Redo позволяет перейти к следующему действию.

 

Команда Cut (Отсечь)
Вызов: Shift + Del

Команда Edit\Cut (Редактировать\Отсечь) удаляет выделенный фрагмент текста из документа и заносит его в текстовый буфер.
Затем можно вклеить текст в любой другой документ (или в какое-либо другое место того же самого документа) путем выбора команды Paste (Вклеить).
Текст в текстовом буфере остается выделенным, поэтому можно вклеивать тот же самый текст многократно.

 

Команда Copy (Копировать)
Вызов: Ctrl + Ins

Команда Edit\Copy (Редактировать\Копировать) оставляет выделенный текст нетронутым, но заносит в текстовый буфер точную копию этого текста.
Затем можно вклеить текст в любой другой документ (или в какое-либо другое место того же самого документа) путем выбора команды Paste (Вклеить).
Можно также скопировать текст из окна Help (Подсказка).

 

Команда Paste (Вклеить)
Вызов: Shift + Ins

Команда Edit\Paste (Редактировать\Вклеить) вставляет текст, расположенный в текстовом буфере, в текущее окно в ту позицию, где располагается курсор.
Тот текст, который вклеиваете, представляет собой помеченный в настоящий момент фрагмент текста в окне Clipboard (текстовый буфер).

 

Команда Clear (Стереть)
Вызов: Ctrl + Del

Команда Edit\Clear (Редактировать\Стереть) удаляет выбранный фрагмент текста, но не заносит его в текстовый буфер. Это означает, что нельзя в дальнейшем вклеить этот текст, как это было бы возможно в результате выбора команд Cut (Отсечь) или Copy (Копировать).
Восстановить удаленный текст нельзя.

 

Команда Copy Example (Скопировать пример)

Команда Edit\Copy Example (Редактировать\Скопировать пример) копирует предварительно выбранный в окне Help (Подсказка) текст примера в окно Clipboard (текстовый буфер).
Примеры уже определены как предназначенные для вклейки блоки, поэтому нет нужды беспокоиться относительно маркирования того примера, который требуется.

 

Команда Show Clipboard (Показать содержимое текстового буфера)

Команда Edit\Show Clipboard (Редактировать\Показать содержимое текстового буфера) открывает окно Clipboard (текстовый буфер), в котором хранятся фрагменты текста, отсеченного и скопированного из других окон.
Тот текст, который в настоящий момент выделен (промаркирован световым или цветовым атрибутом), представляет собой тот текст, который система Borland C++ будет использовать при выборе команды Paste (Вклеить).
Окно Clipboard (текстовый буфер) можно считать списком "предыстории" операций отсечения и копирования.
Окно Clipboard (текстовый буфер) аналогично любому другому окну редактирования; его можно перемещать, его размеры можно менять, его содержимое можно прокручивать и редактировать.
Единственное отличие заключается в том, как оно хранит результаты выполнения команд Cut и Copy.
Когда выбираете текст в окне Clipboard (текстовый буфер), а затем выполняете команду Cut (Отсечь) или Copy (Копировать), выбранный текст немедленно появляется у нижнего края окна.
