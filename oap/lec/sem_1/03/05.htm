Search - меню поиска
Вызов: Alt + S

Меню Search (Поиск) позволяет вам осуществлять
  • поиск текста,
  • замену текста,
  • повторный поиск,
  • переход к строке с заданным номером,
  • поиск местоположения ошибок в файлах,
  • поиск объявлений функций.

 

Команда Find (Найти)
Вызов: Alt + S F

Команда Search\Find (Поиск\Найти) отображает блок диалога Find (Найти), который позволяет вам ввести образец для поиска (в окно Text to Find) и задать параметры, оказывающие влияние на процесс поиска. (Эта команда может также быть вызвана с помощью комбинации клавиш Ctrl + Q F).

Блок диалога Find (Найти) содержит несколько кнопок и блоков:

  • Options (Опции)
    [X] Case sensitive
    (чувствительность к регистру)     		Следует занести маркер (X - нажатием клавиши пробел или нажатием кнопки мыши), если вам необходимо, чтобы система Borland C++ различала бы символы верхнего и нижнего регистров.
    [X] Whole words only
    (только целые слова)     		Следует занести маркер, если вам необходимо, чтобы система Borland C++ осуществляла бы поиск только целых слов (то есть, строк, с обеих концов которых располагаются символы пунктуации или символы пробела).
    [X] Regular expression
    (регулярные выражения)     		Следует занести маркер, если вам необходимо, чтобы система Borland C ++ распознавала используемые утилитой GREP метасимволы в строке поиска.

    Метасимволы в строке поиска

    ^ символ в начале строки соответствует началу строки текста
    $ символ в конце выражения соответствует концу строки текста
    . соответствует любому символу
    * символ, предшествующий звездочке, соответствует любому числу экземпляров (в том числе нулевому) этого символа. Например, последовательности bo* будет соответствовать строка bot, b, boo, а также be.
    + символ, предшествующий плюсу, соответствует любому числу экземпляров (но не нулевому) этого символа. Например, последовательности bo+ будет соответствовать строка bot или boo, но не be или b.
    [ ] символы, заключенные в скобки, соответствуют любому одному символу, который присутствует в скобках, но не другим символам. Например, [bot] соответствует символам b,o или t
    [^] символ "крышечки" в начале строки, заключенной в скобки, соответствует префиксу "не". Следовательно, последовательность [^bot] соответствует любым символам за исключением b,o или t.
    [-] дефис, заключенный в угловые скобки, означает диапазон символов. Например, [b-o] соответствует любому символу в диапазоне от b до o
    \ обратная косая перед метасимволом указывает системе Borland C++ интерпретировать данный символ буквально, а не как метасимвол. Например, \^ соответствует символу ^, а не представляет начало строки

    Для того, чтобы начать поиск, необходимо ввести строку в блок ввода и выбрать кнопку OK (Выполнить).
    Для того, чтобы отменить поиск строки, необходимо выбрать кнопку Cancel (Отмена).
    Если вы хотите ввести строку, поиск которой вы уже осуществляли, необходимо нажать клавишу "стрелка вниз", чтобы отобразить список "предыстории", из которого следует осуществить выбор.
    Вы можете также выбрать то слово, на котором в настоящий момент позиционирован курсор в окне редактирования, и использовать его в блоке Find (Найти), вызвав лишь команду Find (Найти) из меню Search (Поиск).
    Вы можете также выбрать из текста окна редактирования дополнительные символы, нажав клавишу "стрелка вправо".

  • Direction (Направление)
    (•) Forward (Вперед)
    ( ) Backward (Назад)     		Осуществите выбор (стрелка вверх, вниз или мышь) из группы селективных кнопок, чтобы указать системе Borland C++, в каком направлении она должна осуществлять поиск - начиная от начала (которое устанавливается с помощью селективной кнопки "Начало поиска")

  • Scope (Область поиска)
    (•) Global (Глобальная)
    ( ) Selected text (Выделенный текст)     		Осуществите выбор из группы селективных кнопок, чтобы указать системе Borland C++, в какой части файла необходимо осуществлять поиск. Поиск может производиться во всем файле (Global) или только в выбранном тексте.

  • Origin (Начало поиска)
    (•) From cursor (От курсора)
    (•) Entire scope (Весь диапазон)     		Осуществите выбор из группы селективных кнопок, чтобы определить, где начинается поиск.
    Если выбрана кнопка Entire scope (Весь диапазон), то селективные кнопки Direction (Направление) определяют, с конца или с начала выбранного диапазона должен начинаться поиск.
    Требуемый диапазон поиска задается с помощью селективных кнопок Scope (Область поиска).

 

Команда Replace (Заменить)
Вызов: Ctrl + Q A

Команда Search\Replace (Поиск\Заменить) отображает блок диалога, который позволяет вам вводить искомый текст (в окно Text to Find) и текст, на который его следует заменить (в окно New Text).
Блок диалога Replace (Заменить) содержит несколько селективных кнопок и триггерных кнопок - многие из них идентичны тем, которые имеются в блоке диалога Find (Найти), описанном выше.
Дополнительная триггерная кнопка Prompt to Replace (Подтверждение на замену) определяет, будет ли выдаваться запрос на каждое изменение.

Для того, чтобы начать процесс поиска необходимо ввести искомую строку и строку замены в блоки ввода и выбрать кнопку OK (Выполнить) или кнопку Change All (Заменить все).

Нажатие клавиши Cancel (Отменить) приводит к отмене поиска и удалению этого блока диалога.
Если вы хотите ввести строку, которую вы использовали ранее, нажмите клавишу "стрелка вниз", чтобы отобразить список "предыстории", из которого следует сделать выбор.
Если система Borland C++ обнаруживает искомый текст, она запрашивает, хотите ли вы осуществить замену. Если Вы выбрали кнопку OK (Выполнить), система будет осуществлять поиск и замену только первого экземпляра искомой строки.
Если вы выбираете кнопку Change All (Заменить все), то система будет осуществлять поиск и замену всех обнаруженных экземпляров строки в направлении, которое задается параметрами Direction (Направление), Scope (Область поиска) и Origin (Начало поиска).

Аналогично тому, как это делается в блоке диалога Find (Найти), вы можете выбрать слово, расположенное у курсора в окне редактирования, и воспользоваться им в блоке ввода Text to Find (Образец текста для поиска). Для этого необходимо лишь вызвать команду Find или Replace (Найти или Заменить) из меню Search (Поиск). Кроме того, вы можете также выбрать из текста окна редактирования дополнительные символы, нажав клавишу "стрелка вправо".

 

Команда Search Again (Повторить поиск)
Вызов: Ctrl + L

Команда Search\Search Again (Поиск\Повторить поиск) повторяет действие последней команды Find (Найти) или последней команды Replace (Заменить).
Все параметры, которые были заданы при последнем обращении к использованному блоку диалога (Find (Найти) или Replace (Заменить)), остаются при выборе команды Search Again (Повторить поиск) действительными.

 

Команда Go to Line Number (Перейти к строке с номером)

Запрашивает у вас номер той строки, к которой требуется осуществить переход.
Система Borland C++ отображает текущий номер строки и номер столбца в левом нижнем углу каждого окна редактирования.

 

Команда Previous Error (Предыдущая ошибка)
Вызов: Alt + F7

Команда Search\Previous Error (Поиск\Предыдущая ошибка) перемещает курсор на позицию возникновения предыдущего сообщения об ошибке или предупреждающего сообщения.
Эта команда оказывается доступной только в том случае, если в окне Message (Сообщение) имеются сообщения, с которыми связаны какие-либо номера строк.
Эти сообщения генерируются командами компиляции и переноса (transfer), которые используют фильтр сообщений Capture (Захват).

Команда Next Error (Следующая ошибка)
Вызов: Alt + F8

Команда Search\Next Error (Поиск\Следующая ошибка) перемещает курсор на позицию возникновения следующего сообщения об ошибке или предупреждающего сообщения.
Эта команда оказывается доступной только в том случае, если в окне Message (Сообщение) имеются сообщения, с которыми связаны какие-либо номера строк.
Эти сообщения генерируются командами компиляции и переноса (transfer), которые используют фильтр сообщений Capture (Захват).

 

Команда Locate Function (Определить местоположение функции)

Команда Search\Locate Function (Поиск\Определить местоположение функции) отображает блок диалога, в который вы должны ввести имя искомой функции.
Данная команда оказывается доступной только в период проведения отладочного сеанса.
Введите имя функции или нажмите клавишу "стрелка вниз", чтобы выбрать какое-либо имя из списка "предыстории".
В противоположность команде Find (Найти), данная команда ищет объявление функции, а не случаи ее вызова.
