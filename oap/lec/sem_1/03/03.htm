File - Файловое меню
Вызов: Alt + F

Меню File (Файл) позволяет
  • создавать и открывать файлы программ в окнах редактирования,
  • сохранять внесенные изменения,
  • выполнять другие действия над файлами,
  • выходить в оболочку DOS,
  • покидать систему Borland C++.

 

Команда New (Новый)

Команда File\New (Файл\Новый) позволяет вам открывать новое окно редактирования со стандартным именем NONAMExx.CPP (где вместо букв xx задается число в диапазоне от 00 до 99).
Эти файлы с родовым именем NONAME (безымянный) используются в качестве временного буфера для редактирования; когда вы сохраняете на диске файл с подобным именем, система Borland C++ запрашивает у вас действительное имя для этого файла.

 

Команда Open (Открыть)
Вызов: F3

Команда File\Open (Файл\Открыть) отображает блок диалога, предназначенный для выбора файлов, чтобы вы могли выбрать файл программы, который будет открыт в окне редактирования.
Блок диалога содержит в себе:

  • блок ввода,
  • список файлов,
  • кнопки с пометками Open (Открыть), Replace (Заменить), Cancel (Отменить) и Help (Подсказка),
  • информационную панель, которая описывает выбранный в настоящий момент файл.

Блок ввода Name (имя) позволяет вводить имя файла явным образом или позволяет ввести имя файла со стандартными метасимволами, которые используются в операционной системе (* и ?), чтобы отфильтровать имена, появляющиеся в списке файлов.

Можно выполнить одно из указанных далее действий:

  • Ввести полное имя файла и выбрать кнопку Replace (Заменить) или Open (Открыть).
    В результате выбора кнопки Open (Открыть) файл будет загружаться в новое окно редактирования.
    Если вы выбираете кнопку Replace (Заменить), то к данному моменту должно иметься активное окно редактирования; в результате выполнения операции Replace (Заменить) содержимое окна будет заменено содержимым выбранного файла.

  • Ввести имя файла с метасимволами.
    Это позволяет отфильтровать список файлов таким образом, чтобы он соответствовал введенной вами спецификации. Нажать клавишу "стрелка вниз", чтобы выбрать спецификацию файла из списка "предыстории", который содержит введенные вами ранее спецификации файлов.

  • Просмотреть содержимое других директорий, выбрав имя директории из списка файлов.

Если вы введете имя полностью и нажмете клавишу Enter, система Borland C++ откроет указанный файл. (Если вы вводите имя файла, который система Borland C++ не может обнаружить, она автоматически создаст и откроет новый файл с таким именем.).

Если вы нажимаете клавишу "стрелка вниз", когда в блоке ввода мигает курсор, то под этим блоком появляется список "предыстории". Этот список отображает последние восемь имен файлов, которые были введены. Имя из этого списка можно выбрать, подведя к нему указатель мыши и дважды подряд кратковременно нажав кнопку мыши. Выбор можно также произвести с помощью клавиш управления курсором и нажатия клавиши Enter.

Важное замечание: Если вы выбираете вместо кнопки Open (Открыть) кнопку Replace (Заменить), выбранный файл заменит файл, расположенный в активном окне редактирования вместо того, чтобы расположиться в новом окне.

После того, как вы ввели имя файла или выбрали его, выберите кнопку Open (Открыть).
Если вы изменили свои намерения, то выберите кнопку Cancel (Отменить).
После того, как файл выбран, вы можете также нажать клавишу Enter или подвести указатель мыши к имени файла и дважды подряд кратковременно нажать кнопку мыши.

Важное замечание: Для поиска имени файла можно ввести букву по нижнему регистру, а для поиска имени директории можно ввести букву по верхнему регистру.

Блок списка Files (файлы) отображает все имена файлов в текущей директории, которые соответствуют спецификациям в блоке ввода, отображает имя родительской директории а также имена всех поддиректорий.
Подведите к блоку списка указатель мыши и нажмите кнопку мыши, либо нажимайте клавишу Tab до тех пор, пока имя блока списка не будет выделено световым маркером. Теперь можно нажать клавишу "стрелка вверх" или "стрелка вниз", чтобы выбрать имя файла, а затем нажать клавишу Enter, чтобы открыть его.
Чтобы открыть файл, можно также подвести указатель мыши к его имени в блоке и дважды подряд кратковременно нажать кнопку мыши.
Возможно, вам потребуется осуществить прокрутку содержимого блока, чтобы просмотреть имена всех файлов. Если у вас имеется более одной панели имен, то можно также использовать клавиши "стрелка влево" и "стрелка вправо".

Панель информации о файле, расположенная у нижнего края блока диалога Open a File (Загрузить файл), отображает маршрутное имя, имя файла, дату и время создания, а также размер выбранного вами в блоке списка файла. (Ни один из элементов в этой панели нельзя выбрать).
По мере прокрутки по блоку списка содержимое этой панели обновляется, отображая информацию по каждому из файлов.

 

Команда Save (Сохранить)
Вызов: F2

Команда File\Save (Файл\Сохранить) осуществляет запись на диск того файла, который находится в активном окне редактирования.
Если активное окно редактирования в настоящий момент не существует, то данным элементом меню нельзя будет воспользоваться.
Если файл имеет используемое по умолчанию имя NONAMExx.CPP, система Borland C++ откроет блок диалога Save File As (Сохранить под именем), который позволяет переименовать данный файл и сохранить его в другой директории или на другом дисководе.

Команда Save As ... (Сохранить под именем)

Команда File\Save As ... (Файл\Сохранить под именем) позволяет сохранить файл в активном окне редактирования под другим именем, в другой директории или на другом дисководе.
Когда вы выбираете данную команду, вы видите блок диалога Save File As (Сохранить файл под именем). Введите новое имя (возможен также ввод идентификатора дисковода и имени директории), а затем подведите указатель мыши к кнопке OK (Выполнить) и нажмите кнопку мыши, либо выберите кнопку OK (Выполнить). Будут обновлены заголовки всех окон, в которых содержится данный файл.

Команда Save All (Сохранить все)

Команда File\Save All (Файл\Сохранить все) действует аналогично команде Save (Сохранить) с тем исключением, что она осуществляет запись на диск содержимого всех модифицированных файлов, а не только того файла, который находится в активном окне редактирования. Если ни одно из окон редактирования не открыто, то эта команда переводится в состояние "запрещена".

 

Команда Change Dir ... (Сменить директорию)

Команда File\Change Dir ... (Файл\Сменить директорию) позволяет задать идентификатор дисковода и имя директории, которые следует сделать текущими.
Текущей директорией является та директория, которая используется системой Borland C++ для сохранения файлов и поиска файлов.
При использовании относительных маршрутных имен с помощью команды Options\Directories (Параметры\Директории) они задаются только относительно текущей директории.

Сменить директорию можно двумя методами:

  • введите маршрутное имя новой директории в блок ввода и нажмите клавишу Enter;
  • выберите необходимую вам директорию в дереве директорий (если вы используете клавиатуру, то нажмите клавишу Enter, чтобы сделать выбранную директорию текущей), а затем выберите кнопку OK (Выполнить) или нажмите клавишу Esc, чтобы покинуть блок диалога.

Если вы выбираете кнопку OK (Выполнить), то внесенные вами изменения будут реализованы, а блок диалога будет удален с экрана.

Если вы выбираете кнопку ChDir (Сменить директорию), в блоке списка дерева директорий произойдут изменения - выбранная директория станет текущей, и будут отображены поддиректории выделенной световым маркером директории.
Нажатие клавиши Enter или подведение к необходимому элементу указателя мыши и двойное нажатие подряд кнопки мыши приводит к тому же самому результату.

Если вы изменяете свои решения относительно выбранной вами директории и хотите вернуться к предшествующей директории (и вы еще не покинули блок диалога), то выберите кнопку Revert (Инвертировать).

Команда Print (Печатать)

Команда File\Print (Файл\Печатать) позволяет вам вывести на печать содержимое активного окна редактирования.
Система Borland C++ "раскрывает" символы табуляции (переводит символы табуляции в соответствующее число пробелов), а затем посылает файл на устройство печати, заданное в операционной системе. Данная команда будет "запрещена", если содержимое активного окна не может быть выведено на печать.
Для вывода на печать только выделенного текста следует воспользоваться комбинацией клавиш Ctrl + K P.

 

Команда DOS Shell (Выход в оболочку DOS)

Команда File\DOS Shell (Файл\Выход в оболочку DOS) позволяет временно выйти из системы Borland C++, чтобы выполнить команду DOS или запустить какую-либо программу.
Для того, чтобы возвратиться в систему Borland C++, необходимо ввести с клавиатуры EXIT и нажать клавишу Enter.
Вы можете обнаружить, что во время отладки не хватает памяти для того, чтобы выполнить эту команду. В этом случае необходимо завершить сеанс отладки путем выбора команды Run\Program Reset (Выполнение\Переустановка программы) (или нажать комбинацию клавиш Ctrl + F2).

Важное замечание: Во время выхода в оболочку операционной системы не следует инсталлировать какие-либо резидентные программы (например, SideKick), поскольку в результате может произойти неверное распределение памяти.

Примечание: При работе в режиме двух мониторов командная строка DOS появляется на экране системы Borland C++, а не на экране, куда выдаются данные программы пользователя. Это позволяет вам обратиться к DOS, не разрушая выведенных вашей программой данных. Поскольку выводимые вашей программой данные будут все время доступны на одном из мониторов системы, команда Window\User Screen (Окно\Экранный кадр пользователя) будет "запрещена", а нажатие комбинации клавиш Alt + F5 не приведет ни к каким результатам.

Вы можете также воспользоваться элементами подменю переходов, которые присутствуют в системном меню для того, чтобы оперативно переключиться к другой программе, не выходя из системы Borland C++.

 

Команда Quit (Выйти)
Вызов: Alt + X

Команда File\Quit (Файл\Выйти) осуществляет выход из системы Borland C++, удаляет ее из памяти, и возвращает вас к запросу со стороны DOS.
Если вы внесли какие-либо модификации, которые еще не были сохранены, то система Borland C++ запросит вас, хотите ли вы, чтобы они были сохранены перед выходом.
