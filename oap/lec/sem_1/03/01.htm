Назначение интегрированной среды

Система программирования Borland C++ (ver. 3.1) была разработана фирмой Borland в 1992 году для создания программ под управлением MS-DOS. Поэтому ресурсы, предоставляемые разрабатываемому приложению, подчиняются тем ограничениям, которые действовали в среде MS-DOS. Главные из них - объем оперативной памяти не более 640 Кбайт (за минусом того, что занимают компоненты операционной системы), объем каждого массива не более 64 Кбайт (на самом деле, еще немного меньше), диапазон данных типа int от -32768 до 32767. Эти ограничения наиболее характерны для так называемых 16-битных приложений. Во время запуска таким приложениям предоставляется полный экран дисплея, работающего в текстовом режиме.

После старта BC 3.1 в среде Windows, на экране появляется окно интегрированной среды, представленное на рисунке

Интегрированная отладочная среда Borland C++ обеспечивает средства для подготовки, компиляции, отладки и выполнения программы, а также предоставляет расширенный многооконный редактор, возможность работы с мышью, для чего необходимо загрузить соответствующий драйвер, и обслуживает полный цикл создания и поддержки выполняемых программ.

Задание:

  • Откройте Far, нажмите F2 для вызова главного меню.
  • Войдите в каталог вашей группы, выбрав название своей группы.
  • Войдите в подкаталог sem_1.
  • Для вызова Borland C++ запустите файл cpp.bat.
  • Перед Вами интегрированная отладочная среда Borland C++.

Интегрированная отладочная среда Borland C++ является многооконной средой.

Главное меню

После загрузки Borland C++ на экране появляется главное меню с компонентами (смотри рисунок ниже):

  1. Строка главного меню - основное средство доступа ко всем командам меню.

  2. Область для отображения окон: окно редактора, окно сообщений, ... .
    Существует несколько типов окон , но большая их часть имеет следующие общие элементы:
    • строку заголовка (на рисунке: NONAME00.CPP и Watch);
    • маркер закрытия окна (на рисунке: зеленый прямоугольник в верхнем левом углу окна - закройте открытое Вами диалоговое окно);
    • полосы прокрутки (на рисунке: полосы вертикальной прокрутки (справа) и горизонтальной прокрутки (внизу));
    • угол изменения размера окна (на рисунке: любой из углов прямоугольной активной области, выделенной двойной линией рамки);
    • маркер распахивания окна на весь экран (на рисунке: зеленая стрелка в правом верхнем углу окна);
    • номер окна (на рисунке: левее маркера распахивания окна на весь экран).

  3. Строка состояния (линия статуса).
    Строка состояния, расположенная у нижнего края экрана, выполняет следующие функции:
    • Напоминает об основных клавишах и клавишах активизации, которые в настоящий момент могут быть применены к активному окну.
    • Позволяет установить указатель мыши на обозначения клавиш и кратковременно нажать кнопку мыши для выполнения указанного действия, вместо того, чтобы выбирать команды из меню или нажимать соответствующие клавиши.
    • Сообщает, какое действие выполняется программой.
    • Предлагает состоящие из одной строки советы и рекомендации по любой выбранной команде меню и элементам блока диалога.

В любой момент работы активным является

  • или строка меню - обозначается инверсным изображением одного из полей,
  • или одно из окон - показывается или увеличением яркости заголовка окна или заменой одинарной линии рамки окна на двойную линию.

Задание: Проверьте, как меняется состав линии статуса при переходе между открытыми Вами окнами.

В верхней строке располагаются названия пунктов главного меню, краткое назначение которых таково:

  • File - общение с файловой подсистемой;
  • Edit - ввод и редактирование исходной программы;
  • Search - поиск в тексте исходной программы;
  • Run - запуск программы в автоматическом или пошаговом режиме;
  • Compile - компиляция исходной программы;
  • Debug - отладка программы;
  • Project - управление проектом сборки программы из нескольких модулей;
  • Options - настройка параметров интегрированной среды;
  • Window - управление дополнительными окнами системы программирования;
  • Help - обращение к файлам помощи.

Нажатие клавиши F10 - активизирует строку главного меню. После этого можно осуществлять выбор полей меню любым из способов:

  • перемещением клавишами "стрелка влево", "стрелка вправо" с последующим нажатием клавиши Enter,
  • вводом с клавиатуры первого символа имени поля.

В каждом из этих случаев произойдет высвечивание на экране подменю, содержащего опции. Выбор опций подменю осуществляется аналогично выбору поля меню:

  • перемещением клавишами "стрелка вниз", "стрелка вверх" с последующим нажатием клавиши Enter,
  • вводом с клавиатуры первого символа имени опции.

Подтверждение выбора подменю осуществляется нажатием клавиши Enter.

Для завершения работы с подменю необходимо нажать клавишу Esc.

Выбор полей меню и опций подменю можно ускорить воспользовавшись нажатием Alt + выделенная буква нужного меню или подменю.

Пояснение: Alt + E означает удерживать Alt и нажать E, Ctrl + F9 - удерживать Ctrl и нажать F9, и так далее.

Задание: Потренируйтесь в выборе полей меню.

Некоторые команды подменю могут быть помечены символами:

  1. ► - подменю вызывает еще одно новое подменю (выполните: Alt/Options/Environment - Меню параметров/Среда работы).

  2. ... - будет отображен блок диалога (из уже открытого подменю выполните: Preferences... - Приоритеты).

    В диалоговом окне могут быть использованы: ключи и переключатели, рамка ввода и списка. Переход внутри диалогового окна обеспечивается клавишей Tab (последующий набор) и Shift+Tab (предыдущий набор).

    • Ключ с триггерными кнопками (в открытом Вами диалоговом окне: Auto-Save - Автоматическое сохранение).
      [X] - соответствующий режим включен,
      [   ] - соответствующий режим выключен.
      Переключение осуществляется клавишей пробел.

    • Переключатель с селективными кнопками, которые позволяют выбрать только один вариант из заданной группы.
      (в открытом Вами диалоговом окне: Screen Size - Размер экрана, Source Tracking - Трассировка исходного кода, ...)
      (•) - включение режима.

    • Текстовый переключатель со стандартными кнопками OK - выполнить, Cancel - отменить, Help - помощь. Перемещать прямоугольник обратной засветки в выбранную позицию клавишами Tab или Shift + Tab, затем для выбора Enter или Alt + выделенная буква нужного пункта.

      Задание: Закройте открытое Вами окно диалога и выполните Alt/File/Open... или F3

    • Рамка ввода служит для ввода данных, например, имени загружаемой в окно редактора программы.

    • Рамка списка содержит перечень позиций, например, имен файлов, из которых надо сделать выбор.

    • Рамка истории содержит ответы на ранее заданный вопрос. Их можно увидеть, если в режиме рамки ввода нажать клавишу стрелка вниз.
