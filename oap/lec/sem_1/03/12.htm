Help - меню помощи
Вызов: Alt + H

Меню Help (Подсказка) позволяет обратиться к оперативной подсказке, которая выдается в специальное окно.
Система подсказки содержит информацию практически по всем аспектам интегрированной среды и системы Borland C++.
Кроме того, при выборе команды в строке состояния появляется однострочное меню и рекомендации по использованию блока диалога.

Для открытия окна Help (Подсказка) следует осуществить одно из следующих действий:

  • в произвольный момент времени нажать клавишу F1 (в том числе, когда находитесь в блоке диалога или когда выбрана какая-либо команда меню).
  • нажать комбинацию клавиш Ctrl+F1 для получения подсказки по языку, когда активным является окно редактирования и курсор позиционирован на каком-либо слове.
  • подвести указатель мыши к слову Help (Подсказка), когда оно появляется в строке состояния или в блоке диалога, и нажать кнопку мыши.

Для того, чтобы закрыть окно Help (Подсказка), можно

  • нажать клавишу Esc,
  • подвести указатель мыши к маркеру закрытия окна и нажать кнопку мыши,
  • выбрать команду Window\Close (Окно\Закрыть).

Окно Help (Подсказка) может находиться на экране во время работы в другом окне до тех пор, пока не откроете окно Help (Подсказка) из блока диалога или не нажмете клавишу F1 в тот момент, когда выбрана какая-либо команда меню.
Если в тот момент, когда находитесь в окне Help (Подсказка), нажмете клавишу F6 или подведете указатель мыши к другому окну и нажмете кнопку мыши, окно Help (Подсказка) по-прежнему останется на экране.

Важное замечание: Если за подсказкой обращаетесь из блока диалога или из меню, нельзя изменять размер окна подсказки или копировать его содержимое в окно текстового буфера. В этом случае нажатие клавиши Tab производит перемещение к средствам управления блоком диалога, а не к очередному ключевому слову.

Экранный кадр подсказки часто содержит ключевые слова (выделенный световым атрибутом текст), который можно выбрать, чтобы получить дополнительную информацию. Нажатие клавиши Tab приводит к перемещению к любому из ключевых слов; после этого для получения дополнительной информации необходимо нажать клавишу Enter.
Альтернативный способ заключается в том, чтобы переместить курсор к выделенному световым атрибутом слову и нажать клавишу Enter.
При работе с мышью для перехода к тексту подсказки, на который ссылается данный элемент, можно подвести указатель мыши к этому слову и дважды подряд нажать кнопку мыши.
Кроме того, можно перемещать курсор по окну подсказки к любому слову и нажимать комбинацию клавиш Ctrl+F1 в целях получения информации по данному слову. Если это слово в системе подсказки не обнаружено, то в индексе (тематическом указателе) выполняется инкрементальный поиск, и отображается наиболее близкое слово.

Когда окно Help (Подсказка) находится в активном состоянии, можно копировать текст из этого окна и вклеивать его в окно редактирования. Эта процедура осуществляется точно так же, как и при работе в окне редактирования.
Можно также скопировать стандартно выбранные примеры программ из окна подсказки, если выбрать команду Edit\Copy Example (Редактировать\Копировать пример).

Меню Help (Подсказка) предлагает:
  • содержание справки,
  • тематический указатель справки,
  • тематический поиск,
  • возврат к предшествующей теме справки,
  • информацию по системе подсказки,
  • выбор справочных файлов помощи,
  • информацию о системе.

 

Команда Contents (Содержание)
Вызов: F1

Команда Help\Contents (Подсказка\Содержание) открывает окно Help (Подсказка), в котором отображается перечень тем (содержание). Из этого окна можно перейти к любой другой части системы подсказки.

Можно получить информацию по работе с системой подсказки, когда находитесь в окне Help (Подсказка), путем нажатия клавиши F1.
К этому экранному кадру можно также добраться, если подвести указатель мыши к строке состояния и нажать кнопку мыши.

 

Команда Index (Индекс (Тематический указатель))

Команда Help\Index (Подсказка\Индекс (Тематический указатель)) открывает блок диалога, в котором отображается полный перечень ключевых слов, по которым может быть получена подсказка. (Выделенный световым атрибутом в экранных кадрах подсказки текст, который позволяет быстро перейти к связанной с данным ключевым словом теме).

В этом списке можно производить прокрутку и осуществлять инкрементальный поиск, нажимая соответствующие литеры на клавиатуре.
Например, чтобы просмотреть то, что в системе подсказки имеется по теме "printing" (вывод на печать), можно последовательно нажать клавиши p r i. Когда нажимаете клавишу p, курсор перемещается к первому ключевому слову, которое начинается с литеры p. Когда затем нажимаете клавишу r, курсор перемещается к первому ключевому слову, которое начинается с комбинации литер pr. Когда затем нажимаете клавишу i, курсор перемещается к первому ключевому слову, которое начинается с комбинации литер pri и так далее.

Важное замечание: Для того, чтобы выбрать ключевое слово, можно также нажать клавишу табуляции.

Когда нашли интересующее ключевое слово, выберите его, переместив к нему курсор и нажав клавишу Enter. (Можно также переместить к нему маркер управления мышью и дважды подряд нажать кнопку мыши).

 

Команда Topic Search (Тематический поиск)
Вызов: Ctrl + F1

Команда Help\Topic Search (Подсказка\Тематический поиск) отображает справку по выбранному элементу синтаксиса языка.
Чтобы получить справку по языку, необходимо переместить курсор на какой-либо элемент в меню редактирования и выбрать команду Topic Search (Тематический поиск).
Таким путем можно получить подсказку по таким аспектам языка, как имена функций (например, printf), файлам заголовков, зарезервированным словам и так далее. Если в системе подсказки такой элемент отсутствует, будет отображен наиболее близкий по тематическому указателю элемент.

 

Команда Previous Topic (Предшествующая тема)
Вызов: Alt + F1

Команда Help\Previous Topic (Подсказка\Предшествующая тема) открывает окно Help (Подсказка), в котором отображается тот текст, который был просмотрен в последний раз при обращении к системе подсказки.

Система Borland C++ позволяет возвращаться назад на 20 просмотренных экранных кадров подсказки.
Для того, чтобы просмотреть отображенный в последний раз экранный кадр подсказки, можно подвести указатель мыши к строке состояния и нажать кнопку мыши.

 

Команда Help on Help (Информация по системе подсказки)

Команда Help\Help on Help (Подсказка\Информация по системе подсказки) открывает экранный кадр, в котором объясняется, как пользоваться системой подсказки в Borland C++. Если уже находитесь в системе подсказки, этот кадр может быть вызван нажатием клавиши F1.

 

Команда Active file... (Активизация файлов помощи)

Команда Help\Active file... (Подсказка\Активизация файлов помощи) открывает окно диалога, позволяющее выбрать файл справки по работе в конкретной среде программирования: IDE and C++ Language, Windows API, ObjectWindows API, Turbo Vision API.

 

Команда About (О системе)

Когда выбираете данную команду, появляется окно диалога, в котором предлагается информация относительно авторских прав и номера версии системы Borland C++.
Чтобы закрыть данное окно, необходимо нажать клавишу Esc либо подвести указатель мыши к кнопке OK (Выполнить) и нажать кнопку мыши. Для закрытия этого окна можно также нажать клавишу Enter.
