Другие команды редактирования

Перемещение
Команды
Функция
Автоотступ Options/Environment/Editor маркировать Autoindent Mode Открывает диалоговое окно Editor Options, в котором можно переключить независимую кнопку Autoindent Mode. Обеспечивает автоматический отступ последовательных строк. Когда Autoindent активна, то отступ текущей строки повторяется на каждой следующей строке; то есть при нажатии Enter курсор возвращается не к первому столбцу, а к начальному столбцу предыдущей непустой строки. Если хотите изменить отступ, используйте ПРОБЕЛ и "стрелки" для выбора нового столбца. По умолчанию автоотступ включен.
Задать место Ctrl+K n Помечает 10 мест в тексте путем нажатия Ctrl+K, сопровождаемой цифрой маркера (от 0 до 9). После отметки расположения можно работать в любом месте файла и затем легко вернуться к помеченному месту путем использования команды Ctrl+Q n. В каждом окне можно помечать до 10 мест
Найти маркер места Ctrl+Q n Поиск десяти маркеров места (n - любое число от 0 до 9) в тексте. Перемещает курсор к заранее установленному маркеру путем нажатия Ctrl+Q и номера маркера
Новый файл File/New Открывает новое окно
Открыть файл File/Open (F3) Позволяет загрузить существующий файл в окно редактора
Выйти из редактора File/Quit (Alt+X) Осуществляет выход из Borland C. Будет выдан запрос, хотите ли Вы сохранить файл на диске
Восстановить строку Edit/Restore Line или Ctrl+Q L Позволяет отменить изменения, сделанные над последней строкой. Строка восстанавливается в своем изначальном виде независимо от выполненных изменений. Эта команда работает только с последней модифицированной или удаленной строкой
Сохранить файл File/Save (F2) Сохраняет файл и возвращается в редактор
Печатать файл File/Print Печатает файл
Табуляция Tab По умолчанию табулирует на восемь столбцов в редакторе Borland C
Режим табуляции Options/Environment/Editor маркировать Use Tab Character Открывает диалоговое окно Editor Options, в котором можно установить независимую кнопку Use Tab Character. Когда опция включена, можно вставлять символ табуляции (8-й символ ASCII); а когда отключена, табуляция автоматически вставляется как соответствующее число пробелов
