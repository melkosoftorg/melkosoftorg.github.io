Изменить яркость символов

highvideo - вызывает изображение символов с повышенной яркостью, устанавливая для этого бит интенсивности в текущем цвете переднего плана.

normvideo - вызывает нормальное отображение символов, устанавливая для этого текстовый атрибут (фон и передний план) в значение, которое он имел при запуске программы.

lowvideo - вызывает пониженную яркость отображения символов, сбрасывая для этого в текущем цвете переднего плана бит интенсивности.

Данные функции не влияют на все символы, отображаемые в данный момент времени на экране. Их влияние распространяется только на те символы, которые отображаются функциями непосредственного вывода на консоль (такими, как cprintf), причем после обращения к данным функциям. 

Синтаксис	void highvideo(void);
			void normvideo(void);
			void lowvideo(void);
Прототип в	conio.h
