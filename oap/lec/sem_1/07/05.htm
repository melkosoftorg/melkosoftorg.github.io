delline - стирает строку, в которой находится курсор, и поднимает все строки, находящиеся ниже курсора, на одну строку вверх. Функционирует в текущем активном текстовом окне. 
Синтаксис void delline(void);
Прототип в conio.h
insline - вставляет пустую строку в позицию курсора текстового окна, используя при этом текущий цвет фона. Все строки под пустой строкой сдвигаются на одну строку вниз, последняя строка экрана пропадает. 
Синтаксис void insline(void);
Прототип в conio.h
movetext - переносит содержимое прямоугольника экрана, заданного с помощью left, top, right, bottom, в новый прямоугольник такого же размера.
Верхний левый угол этого нового прямоугольника задается парой (newleft, newtop).
Все координаты соответствуют абсолютным координатам экрана.
В случае успеха movetext возвращает 1; при ошибке (например, если вы задали координаты вне диапазона текущего режима экрана) movetext возвращает 0. 
Синтаксис int movetext(int left, int top, int right, int bottom, int newleft, int newtop);
Прототип в conio.h

/* Копирование содержимого прямоугольника с верхним левым и нижним правым углами, 
   заданными соответствующими парами точек (5, 15) и (20, 25), в новый прямоугольник с 
   левым верхним углом, заданным парой (10,10).
*/
movetext (5, 15, 20, 25, 10, 10);
