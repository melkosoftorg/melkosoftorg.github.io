Изменение режима доступа к файлу

_chmod - изменяет режим доступа файла;
  chmod - изменяет режим доступа файла. 
Синтаксис	#include<dos.h>
		int _chmod(char * filename, int func[,int attrib]);
		int chmod(char * filename, int func[,int attrib]);
Прототип в io.h

Функция chmod назначает способ доступа к файлу *filename соответственно маске, содержащейся в аргументе permiss.
Аргумент filename представляет собой адресный указатель символьной строки, именующей файл.

Аргумент permiss может содержать одну или обе символические константы S_IWRITE и S_IREAD (константы определяются в файле sys\stat.h).

S_IWRITE - разрешение на запись
S_IREAD - разрешение на чтение
S_IREAD | S_IWRITE - разрешение на чтение и запись

Функция _chmod может либо вызывать, либо устанавливать атрибуты файла системы MS- DOS.
Если аргумент func = 0, функция возвращает текущие атрибуты системы MS-DOS заданного файла.
Если аргумент func = 1, атрибут устанавливается по значению переменной attrib.

Аргумент attrib может быть одной из следующих символических констант (определенных в файле dos.h):

FA_RDONLY - атрибут "только чтение"
FA_HIDDEN - скрытый файл
FA_SYSTEM - системный файл

При успешном завершении смены режима доступа файла функция chmod возвращает значение 0, в любом другом случае функция возвращает значение -1.

При успешном завершении функция _chmod возвращает значение, соответствующее слову - атрибуту доступа, иначе функция возвращает значение равное -1.

В случае ошибки, переменная errno получает одно из следующих значений:

  • ENOENT - траектория доступа (PATH) или имя файла не найдены,
  • EACCES - отказ доступа. 
	#include<sys\stat.h>
	#include<stdio.h>
	#include<io.h>
	void make_read_only(char *filename);
	int main(void)
	{
		make_read_only("TEST.$$$");
		make_read_only("MYFILE.FIL");
		return 0;
	}
	void make_read_only(char *filename)
	{
		int stat;
		stat = chmod(filename, S_IREAD);
		if (stat)
			printf("Не могу сделать %s только для чтения\n", 
				filename);
		else
			printf("%s имеет атрибут только для чтения\n", 
				filename);
	}
