Получение данных о файле

filelength - получает размер файла в байтах.
Синтаксис long filelength(int handle);
Прототип в io.h

Функция filelength возвращает длину в байтах файла, соответствующего номеру handle.
При успешном завершении функция filelength возвращает значение типа long длину файла в байтах.
При ошибочном завершении функция возвращает значение -1L, и глобальной переменной errno присваивается: EBADF - неверный номер файла. 

   #include<string.h>
   #include<stdio.h>
   #include<fcntl.h>
   #include<io.h>
   #include<sys\stat.h>
   int main(void)
   {
      int handle;
      char buf[20] = "0123456789";
      // создать файл из 10 байт
      handle = open("DUMMY.FIL", O_RDWR|O_CREAT|O_TRUNC|S_IREAD|S_IWRITE);
      write(handle, buf, strlen(buf));
      // Напечатать размер файла
      printf("Размер файла в байтах: %ld\n", filelength(handle));
      // Закрыть файл
      close(handle); return 0;
   }
      После исполнения программы:	Размер файла в байтах: 10
fileno - получает логический номер файла.
Синтаксис int fileno(FILE * stream);
Прототип в stdio.h

fileno представляет собой макрокоманду, которая возвращает логический номер файла для заданного потока stream.
Если поток stream имеет более одного номера, функция fileno возвращает номер, назначенный данному потоку при первом открытии.
fileno возвращает целое число, обозначающее логический номер файла, соответствующий потоку stream.
