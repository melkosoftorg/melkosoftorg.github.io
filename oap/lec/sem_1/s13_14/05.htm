Закрытие файла

_close - закрывает файл, обозначенный параметром handle,
  close - закрывает файл, обозначенный параметром handle. 
Синтаксис	int _close (int handle);
		int close (int handle);
Прототип в io.h

close и _close закрывают файл, обозначенный аргументом handle.
Аргумент handle обозначает логический номер файла, присваиваемый ему при вызове функций: _creat, creat, creatnew, creattemp, dup, dup2, _open, open.

Функция close, вызванная для закрытия текстового файла, открытого для записи, записывает в конце файла символ CTRL-Z. При вызове функции _close данный символ не записывается.

При успешном завершении, функции close и _close возвращают значение 0, иначе возвращается значение -1.
Неудачное завершение работы обеих функций, если аргумент handle не верен, оставляет файл открытым и присваивает глобальной переменной errno значение: EBADF Неверный номер файла.
