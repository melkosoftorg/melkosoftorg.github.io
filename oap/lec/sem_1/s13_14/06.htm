Указатель чтения - записи файла

lseek устанавливает указатель чтения/записи файла.
Синтаксис long lseek(int handle, long offset, int fromwhere);
Прототип в io.h

lseek устанавливает указатель файла, соответствующий параметру handle, в новую позицию, которая находится на offset байтов впереди месторасположения заданного параметром fromwhere.

Параметр fromwhere должен иметь одно из трех значений 0, 1 или 2, которые представлены тремя символическими константами (определенными в файле stdio.h) следующим образом:

SEEK_SET 0 - начало файла
SEEK_CUR 1 - текущая позиция указателя файла
SEEK_END 2 - конец файла

Функция lseek возвращает значение смещения новой позиции указателя, измеряемой в байтах от начала файла.

В случае ошибки функция lseek возвращает значение -1L и глобальная переменная errno получает одно из следующих значений:

  • EBADF - неверный номер файла
  • EINVAL - неверный аргумент
На устройствах, непригодных для поиска, таких, как терминалы и принтеры, возвращаемое значение не определено.

tell получает текущую позицию указателя файла, соответствующего параметру handle и выражает ее количеством байтов от начала файла.
Синтаксис long tell(int handle);
Прототип в io.h

tell возвращает текущее значение указателя файла.
Возврат значения -1 (long) означает ошибку, и глобальной переменной errno присваивается значение: EBADF - неверный номер файла. 

   #include<string.h>
   #include<stdio.h>
   #include<fcntl.h>
   #include<io.h>
   int main(void)
   {
      int handle;
      char msg[] = "Hello world";
      if((handle = open("TEST.$$$", O_CREAT | O_TEXT | O_APPEND)) == -1)
      {
         perror("Ошибка:");
         return 1;
      }
      write(handle, msg, strlen(msg));
      printf("Указатель файла находится на %ld байте\n", tell(handle));
      close(handle); return 0;
   }

rewind устанавливает указатель в начало потока.
Эквивалентно fseek(stream, 0L, SEEK_SET), за исключением того, что rewind обнуляет признаки конца файла и ошибки, а fseek только признак конца файла.

Синтаксис int rewind(FILE *stream);
Прототип в stdio.h
