Разделение доступа к файлу

    lock - устанавливает "замки" для разделения доступа к файлам.
unlock - освобождает "замки" разделенного доступа к файлам. 
Синтаксис	int lock(int handle, long offset, long length);
		int unlock(int handle, long offset, long length);
Прототип в io.h

Функции lock и unlock устанавливают механизм разделенного доступа к файлам.
"Замок" может быть помещен в произвольную, не перекрывающуюся область любого файла.

Программа, делающая попытку прочесть или записать в "запертую" область, должна будет возобновлять операцию до трех раз.
Если все три попытки неудачны, вызов завершается по ошибке.

Функция unlock удаляет "замок"; чтобы избежать ошибки, "замок" должен быть удален перед закрытием файла.

Перед завершением программа должна удалить все "замки".

Обе функции возвращают 0 при успешном завершении, -1 при ошибке.
