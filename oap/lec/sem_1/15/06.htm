Обработка ошибок в графическом режиме

grapherrormsg возвращает строку сообщения об ошибке, соответствующую заданному коду ошибки
graphresult возвращает код ошибки последней графической операции, приведшей к ней.

Если ошибка произошла при вызове функции графической библиотеки (шрифт, требуемый функцией settextstule, не найден), то устанавливается внутренний код ошибки.
С помощью graphresult можно получить код ошибки последней графической операции.

Обычная начальная последовательность действий при работе с графикой может быть следующей:

  1. Создать функцию собственной библиотеки init() для инициализации графики и сохранить ее в файле init.h в каталоге H - библиотеки. 
    	void init()
	{
		// требование автоопределения
		int graphdriver = DETECT, graphmode;
		// инициализация графики;
		// параметр "..\\..\\chr" - относительный маршрут к подкаталогу CHR с файлами драйверов и шрифтов
		initgraph(&graphdriver, &graphmode, "..\\..\\chr");
		int error = graphresult();
		if(error != grOk)
		{	puts("Ошибка графики");
			puts(grapherrormsg(error));
			exit(1);
		}
	}
  2. Подключить созданный библиотечный файл к необходимому рабочему файлу. 
    	#include<graphics.h>
	#include<stdlib.h>
	#include<stdio.h>
	#include<conio.h>
	#include"..\\h\\init.h"
  3. Инициализировать графику вызовом функции init() и по окончании работы закрыть графический режим. 
    	void main()
	{	
		init();

		// ... - тело программы

		getch();
		closegraph();
	}
