Пункты для конспекта (лекция 4)

  1. Выражение представляет собой объединение операций и операндов.
  2. Операнд - это то, над чем выполняется операция.
  3. Результат выражения определяется типами обоих операндов.
  4. При выполнении операций производится автоматическое преобразование типов
  5. Операция sizeof() ... Значением выражения sizeof является ...
  6. Если действие производится с двумя операндами, операция называется бинарной
  7. Если действие производится с одним операндом, операция называется унарной.
  8. Оператор преобразования типа имеет следующий синтаксис: 
    		(новый_тип)переменная 	или	 (новый тип)выражение.
  9. Математические операции только для целочисленных вычислений ... (перечислить)
  10. Операция деления по модулю % ... Ее результатом является ...
  11. 13 % 5 (читается: 13 по модулю 5) имеет значение 3. 
  12. 		
       		a = a + b;    сокращается до    a += b;
		a = a - b;     сокращается до    a -= b;
		a = a * b;     сокращается до    a *= b;
		a = a / b;      сокращается до    a /= b;
		a = a % b;    сокращается до    a %= b;
  13. Операции увеличения - инкремента (++) и уменьшения - декремента (--)...
  14. В языке допускается префиксная или постфиксная формы операций ++ и --. (Привести пример)
  15. Если операция инкремента или декремента для переменной используется отдельно от других операций, то порядок действий несущественен
  16. Счетчик — это ... 
    	a = a + 1	// (a++) 	или 	a = a - 1	// (a--)
  17. Аккумулятор - ... 
    	s = s + num; 	// или s += num;
  18. Операция последовательного вычисления обозначается запятой (,) и используется для вычисления двух и более выражений.
