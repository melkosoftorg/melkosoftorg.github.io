Операция присваивания =

Операция присваивания - одна из самых простых и наиболее часто используемых операций в любом языке программирования. Она предназначена для вычисления нового значения некоторой переменной, а также для определения значения, возвращаемого функцией.

Оператор присваивания на языке «С» имеет следующий синтаксис: 

	Переменная = значение;
	Переменная = выражение;

Переменная – любая переменная, объявленная в программе.
Значение – любое числовое или символьное значение.
Выражение – математическое или логическое выражение, которое может содержать вызовы функций.

Операция выполняется следующим образом:

  • Вычисляется значение выражения в правой части присваивания.
  • После этого переменная, указанная в левой части, получает вычисленное значение.
    При этом тип выражения должен быть преобразуем с помощью операции преобразования к типу переменной.

Тип выражения определяется типом операндов, входящих в него. 

		x = (y + z) / (2 + z * 10) - 1 / 3;
		LogPer = (A > B) && (C <= D);

Можно применять последовательные присваивания.

Например:   sum = a = b = 25.7  
Присваивание производится справа налево: 25.7 будет присвоено b, затем значение b будет присвоено a, которая в свою очередь будет присвоена sum, так что все три переменных получат одно и то же значение (а именно, 25.7).

Любой оператор присваивания, заключенный в круглые скобки, является выражением с определенным значением, которое получается в результате этого присваивания. 

Например:	выражение 6 + (sum = 5 + 3) имеет значение 14,
		выражение (sum = 5 + 3) имеет значение 8,
		поэтому выражение ((sum = 5 + 3) <= 10) будет всегда
		иметь значение "истина", так как 8 <= 10.

Математические операции только для целочисленных вычислений

К таким операциям относят:

  • взятие остатка от деления (%);
  • операции над битами (&,|,^,!);
  • операции сдвигов (<<, >>);
  • операции инкремента (++) и декремента(--).
