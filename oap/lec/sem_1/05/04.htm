Операция деления по модулю %

В C++ есть операция нахождения остатка от деления.
Операция целочисленного деления как отдельная отсутствует. Вместо этого применяют обычное деление, но с приведением к целым типам. 

Например:
			15 / 3 = 5
			18 / 5 = 3
			123 / 10 = 12
			7 / 10 = 0
Но!
			18 / 5.0 = 3.6
			7.0 / 10 = 0.7

Операция деления по модулю % используется в целочисленной арифметике. Ее результатом является остаток от деления целых чисел.
13 % 5 (читается: 13 по модулю 5) имеет значение 3.

С помощью операции % можно найти остаток от деления одного целого числа на другое. 

Например
			15 % 3 = 0
			18 % 5 = 3
			123 % 10 = 3
			7 % 10 = 7

Над числами с плавающей точкой данная операция не производится.

Фрагмент программы иллюстрирует использование операций деления для перевода секунд в минуты и секунды. 

		int sec, mi, left;
		#define   SM   60		// число секунд в минуте
		printf("Введите число секунд для перевода в минуты "); 	
		scanf("%d", &sec); 
		mi = sec / SM;		// количество минут
		left = sec % SM;		// оставшееся число секунд
		printf("%d c это %d мин и %d c\n", sec, mi, left);

Пусть было введено 95 секунд.
Значение mi = 95 / 60 равно 1 (целое число типа int)
Значение left = 95 % 60 равно 35 (остаток от деления).
Будет выведен результат: 95 с это 1 мин 35 с
