Комбинированные операции

C позволяет использовать некоторые сокращения при написании выражений, содержащих многочисленные операции.
Любое выражение вида 
		<переменная> = <переменная> <операция> <выражение>;
может быть заменено на 
		<переменная> <операция> = <выражение>;

       		a = a + b;    сокращается до    a += b;
		a = a - b;     сокращается до    a -= b;
		a = a * b;     сокращается до    a *= b;
		a = a / b;      сокращается до    a /= b;
		a = a % b;    сокращается до    a %= b;

Пусть		int a = 7, b = 2;    
Вычислим	a *= b;
Эта запись означает, что параметру a присваивается новое значение, равное предыдущему его значению, умноженному на b.
Получим новое значение a, равное 14 (a = 7 * 2).

Примеры: 

	a += 5;   	РАВНОСИЛЬНО 	a = a + 5;
	b *= 3+c; РАВНОСИЛЬНО 	b = b * (3 + c);
	c /= d;   	РАВНОСИЛЬНО 	c = c / d;
