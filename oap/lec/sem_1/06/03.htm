Директивы препроцессора

Препроцессор C - это текстовый процессор, используемый для обработки текстового исходного файла на первой фазе компиляции.

Он просматривает программу до компилятора и заменяет символические аббревиатуры в программе на соответствующие директивы, отыскивает другие необходимые файлы и так далее.

Директивы препроцессора - это инструкции, предназначенные для препроцессора.

Назначение директив:

  • сделать исходные программы проще для модификации,
  • осуществлять компиляцию для различных реализаций компилятора C.

Директивы могут появляться где угодно в исходном файле, но они применимы только к остатку исходного файла от места, где они появились.

Препроцессор C распознает директивы: 

	# define			# else			# endif
	# elif			# error			# if
	# ifdef			# ifndef			# include
	# line			# pragma		# undef
	# - null-директива
