Другие стандартные функции математической библиотеки

Функция
Действие
hypot(x,y) вычисляет гипотенузу c прямоугольного треугольника с катетами x и y. Так как    c2 = x2 + y2, то c = sqrt(x * x + y * y).
exp(x) экспоненциальная функция, ex    exp(1) = 2.718282.
log(x) Натуральный логарифм, ln(x), x > 0.
Логарифм по любому основанию можно выразить через натуральные логарифмы
log10(x) десятичный логарифм, lg(x), x > 0
pow(x, y) вычисляет xy. Ошибка области, если x = 0 и y ≤ 0 или x < 0, y - не целое.    2.341.5 запишется pow(2.34, 1.5) = 3.579512.
Эта функция необходима для вычисления выражения вида - pow(x, n / m)
pow10(p) вычисляет 10p. Результат вычисляется в виде double.
Все аргументы считаются допустимыми. p типа int - целое число.
sqrt(x) - корень квадратный из x, x ≥ 0
ceil(x) находит наименьшее целое в виде double, не меньшее x
ceil(6.25) = 7.00, ceil(-6.25) = -6.00
floor(x) находит наибольшее целое в виде double не превышающее значение x.
floor(6.25) = 6.00, floor(-6.25) = -7.00
fabs(x) абсолютное значение (модуль) числа с плавающей точкой
fabs(-6.25) = 6.25
ldexp(x,n) вычисляет значение двойной точности x∙2n, n - целое число.
ldexp(6.25,3.00) = 6.25 · 23.00 = 50.00,
ldexp(-6.25,-4.00) = -6.25 · 2-4.00 = -0.39
frexp(x, int *exp) разбивает число двойной точности x на мантиссу m и порядок n.
Мантисса принимает значение большее или равное 0.5 и меньшее 1.
Целое n такое, что x = m ∙ 2n. f = frexp(x, &z);
frexp(6.25)=0.78 · 23, f = 0.78, z = 3
modf(x, double *ip) разделяет число с плавающей точкой на целую и дробную часть.
Целая часть запоминается в область памяти *ip.
Дробная часть выдается как результат. f=modf(x, &y);
f = modf(6.25, 6.00), y = 6.00 f = 0.25,
fmod(x,y) вычисляет x по модулю y - остаток от деления x на y в виде числа с плавающей точкой. Знак результата совпадает со знаком x.
fmod(6.25, 3.00) = 0.25, fmod(-6.25, 4.00) = -2.25
