Пункты для конспекта (лекция 8)

  1. Функция scanf - ...
  2. Функция scanf сканирует (просматривает) клавиатуру, определяет, какие клавиши нажаты, затем интерпретирует ввод, основываясь на указателях формата (преобразователи символов)
  3. Функция scanf имеет одно очень важное отличие: ...
  4. scanf("%d %d", &a, &b); (что означает команда?)
  5. Операция адреса &
  6. Если необходимо ввести строку, то оператор получения адреса со строковой переменной использовать НЕ надо
  7. Функция gets - ...
  8. Функция getch - ...
  9. Функция getchar - ...
  10. Функция getche - ...

  11. Функция puts - ...
  12. Параметром функции puts может быть один из типов данных: константное выражение, строковая константа. строковая переменная
  13. Функция putchar...
  14. Параметром функции putchar может являться: Символ, Символьная константа, Символьная переменная
  15. Функция putch ...
  16. Функции семейства ...printf позволяют выводить на дисплей данные всех типов и работать со списком из нескольких аргументов. Кроме того, можно определить способ форматирования данных.
  17. рrintf ("строка формата", аргумент_1, аргумент_2, ...);
  18. Каждая функция семейства ...printf возвращает количество выводимых байтов.
  19. Cтрока формата - символьная строка, состоящая из двух типов объектов:
    • просто символы (переносятся в вывод без изменений),
    • спецификации преобразования (применяются для форматирования).
  20. Спецификации формата имеют форму: 
    	% [флажки] [ширина] [. точность] [размер] тип
  21. Каждая спецификация начинается с символа процента %
  22. Список типов преобразований (из таблицы)
  23. Символы-флажки (из таблицы)
  24. Спецификация ширины - устанавливает минимальную ширину поля для выводимого значения (из таблицы)
  25. Спецификация точности всегда начинается с точки ( . ), отделяющей ее от предшествующей спецификации ширины(из таблицы)
