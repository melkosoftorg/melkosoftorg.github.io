Ввод и вывод строки


Ввод строки

Функции ввода строк считывают из потока целые строки, то есть последовательности символов, до первого встретившегося символа конца строки.

gets - вводит строку в переменную, параметром функции является имя переменной. То есть функция считывает последовательность символов из потока stdin до тех пор, пока не встретится конец строки, и помещает ее в заданную строку.
Функция gets() позволяет вводить строки, содержащие символы пробела и табуляции.

Внимание: прежде чем вызывать gets(), необходимо позаботиться о выделении достаточного количества памяти для хранения строки.
Если память не была распределена корректно, введенная строка может затереть важные данные и вызвать ошибки в программе.

Синтаксис char *gets(char *s);

Прототип в stdio.h 

		char name [60]; 
		puts("Как вас зовут?"); gets(name);
		puts("Привет"); puts(name);

Функция gets читает все, что набирается до нажатия Enter. Она не помещает Ввод в строку, однако, в конец строки добавляет нулевой символ \0.

 


Вывод строки

puts - выводит строку на экран и завершает вывод символом новой строки.

Синтаксис int puts(const char *s);

Прототип в stdio.h

Параметром может быть один из типов данных:

  • Константное выражение: 
    	puts("Первая программа на С++ !");

  • Строковая константа: 
    	#define STR "Всем привет!"
	main()
	{
		puts(STR);
	}
  • Строковая переменная: 
    	char greet[20]="Всем привет!"
	puts(greet);
