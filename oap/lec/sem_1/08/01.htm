Ввод и вывод данных в языке C

Практически во всех программах выполняется ввод и вывод данных. Очень часто о качестве и практической ценности программы можно судить именно по тому, насколько хорошо она справляется с задачами ввода-вывода.

Источники данных для ввода и места назначения вывода имеют собирательное название устройства.
Устройствами являются:

  • клавиатура,
  • принтер,
  • модем,
  • монитор,
  • жесткий диск,
  • и так далее.

Весь ввод и вывод в программах на C выполняется через потоки независимо от того, откуда вводятся данные и куда они выводятся.

Поток - это последовательность символов, последовательность байт данных.
Поток ввода - это поток байт, принимаемый программой.
Поток вывода - это поток, посылаемый программой на устройства.

Стандартные потоки ввода - вывода C открываются автоматически. В стандартной библиотеке функций C имеется большое количество функций для ввода-вывода с использованием потоков.

В очень многих программах на C используется ввод с клавиатуры, то есть из стандартного потока stdin.

Вводом называется процесс предоставления компьютеру информации, необходимой для работы программы.

Функции, используемые для ввода данных, зависят от типа данных и способа их представления.

Функции ввода подразделяются на три иерархических уровня:

  • функции ввода строк,
  • функции ввода отдельных символов,
  • функции форматированного ввода.

Выводом называется процедура переноса данных из памяти компьютера в другое место (на экран, отпечатать на принтере, сохранить на диске в виде файла).

Функции, используемые для вывода данных, зависят от типа данных и способа их представления. Функции вывода на экран делятся на три категории:

  • функции вывода строк,
  • функции вывода символов,
  • функции форматированного вывода
