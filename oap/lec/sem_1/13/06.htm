Внешние статические переменные

Можно описать статические переменные вне любой функции, что создаст внешнюю статическую переменную.

Обычная внешняя переменная может использоваться функциями в любом файле.

Внешняя статическая переменная может использоваться только функциями того же самого файла, причем после определения переменной. 

		static randx = 1;
		rand()
		{ ... }

Все глобальные переменные по умолчанию статические и не могут быть иными, спецификатор static для глобальных переменных используется для ограничения доступа, а не для изменения класса памяти.
