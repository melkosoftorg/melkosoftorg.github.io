Глобальные переменные

Глобальные переменные - переменные известные нескольким функциям, нелокальные переменные.

Каждая переменная имеет тип и принадлежит к некоторому классу памяти. Определение класса памяти переменной зависит от того, где переменная описана и какое ключевое слово (если оно есть) используется.

Класс памяти позволяет определить:

  • какие функции имеют доступ к переменной (область действия),
  • как долго переменная находится в памяти.

Для того чтобы создать внешнюю переменную, ее следует определить перед функцией main() 

//Программа, содержащая внешнюю переменную. 
int temp;
main()
{
	printf("Введите значение температуры: ");
	scanf("%d", &temp);
	convert();
	freeze();
	boil();
}
convert()
{
	float celsius;
	celsius = (5.0 / 9.0) * (temp - 32);
	printf("%d градусов по шкале Фаренгейта соответствует %6.2f градусам по шкале Цельсия\n", temp, celsius);
	return(0);
}
freeze()
{
	printf("Это составляет %d градусов от точки замерзания воды\n", temp-32);
	return(0);
}
boil()
{
	printf("Это составляет %d градусов от точки кипения воды\n", 212-temp);
	return(0);
}
Результаты:
			Введите значение температуры: 65
			65 градусов по шкале Фаренгейта =  18.33 градусам по шкале Цельсия
			33 градусов до точки замерзания воды
			147 градусов до точки кипения воды

Если в программе одновременно присутствуют внешняя и локальная переменные с одним и тем же именем, функция, в которой определена локальная переменная, будет иметь доступ только к ней и не сможет использовать внешнюю переменную. Если глобальная и локальная переменные имеют одинаковое имя, то, чтобы получить их значения, надо по-разному к ним обращаться, например, переменная i 

	#include<iostream.h>
	int i = 0;			// глобальная переменная
	void main()
	{
		int i = 7;		// локальная переменная

		cout << "Глобальная ::i = " << ::i << '\t' << "Локальная  i = " << i << endl;
	}
	Результаты:
				Глобальная ::i = 0      Локальная  i = 7
