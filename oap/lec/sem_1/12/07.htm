Аргументы по умолчанию

Обычно при вызове функции в нее передается конкретное значение каждого аргумента, но в языке C++ имеется возможность назначать значения аргументов функции, заданные по умолчанию.

Аргумент по умолчанию - это значение, которое присваивается параметру функции при ее вызове, если при этом соответствующий аргумент функции не указан.

Если аргумент по умолчанию не указан в вызове функции, то в вызов автоматически передается значение этого аргумента по умолчанию.

Если вызывается функция с двумя или более аргументами по умолчанию и если пропущенный аргумент не является самым правым в списке аргументов, то все аргументы справа от пропущенного тоже пропускаются.

Аргументы по умолчанию должны быть указаны при первом упоминании имени функции - в прототипе.

Значения по умолчанию могут быть:

  • константами,
  • глобальными переменными,
  • вызовами функций.

В рассматриваемой программе встраиваемая (inline) функция вычисляет объем параллелепипеда. Всем трем аргументам (длине, ширине, высоте) было дано значение по умолчанию равное 1 (int l = 1, int w = 1, int h = 1).
В первом вызове функции не указаны аргументы, поэтому используются все три значения по умолчанию.
Во втором вызове передается один параметр - длина, поэтому для ширины и высоты используются параметры по умолчанию.
В третьем вызове переданы параметры для длины и ширины, параметром по умолчанию является высота.
В четвертом вызове значения по умолчанию не используются, поскольку переданы функции все три параметра.
Аргументы по умолчанию - самые правые аргументы в списке параметров. 

	#include<conio.h>
	#include<iostream.h>
	inline int boxVol(int l=1, int w=1, int h=1)
	{
		return l*w*h;
	}
	main()
	{	clrscr();   cout << "Объем параллелепипеда\n\n";
		cout << "Размером по умолчанию " << boxVol();
		cout << "\n10 x 1 x 1 равен " << boxVol(10);
		cout << "\n10 x 5 x 1 равен " << boxVol(10, 5);
		cout << "\n10 x 5 x 3 равен " << boxVol(10, 5, 3);
		getch(); return 0;
	}
		Результаты:	Объем параллелепипеда
				Размером по умолчанию 1
				10 x 1 x 1 равен 10
				10 x 5 x 1 равен 50
				10 x 5 x 3 равен 150

Если задается для какого-то параметра значение по умолчанию, то нужно предусмотреть значения по умолчанию для всех последующих параметров функции. Таким образом, параметры со значениями по умолчанию должны стоять после всех остальных параметров в списке.

Небольшой пример: 

	#include<conio.h>
	#include<istdio.h>
	int Sum(int a, int b, int c = 0, int d = 0);   // Правильный прототип
	//
	// Неправильный прототип:
	// int Sum(int a, int b = 0, int c, int d = 0);
	//
	int Sum(int a, int b, int c, int d)   // Определение функции
	{
		return a + b + c + d;
	}

	main()
	{
		printf("1 + 2 + 3 + 4 = %d\n", Sum(1, 2, 3, 4));
		printf("    1 + 2 + 3 = %d\n", Sum(1, 2, 3));
		printf("        1 + 2 = %d\n", Sum(1, 2));
		//
		// Недопустимый вызов:
		// printf("1 + 2 + 4 = %d\n", Sum(1, 2,, 4));
		//
		getch();
		return 0;
	}
