Пункты для конспекта (лекция 12)

  1. C++ позволяет определить несколько функций с одним и тем же именем, но с разными наборами параметров. Эта особенность называется перегрузкой функций.
  2. Перегрузку функций удобно применять там, где ряд функций выполняет однотипную работу над аргументами различных типов.
  3. Рекурсивная функция - функция, которая вызывает саму себя или непосредственно, или косвенно через другую функцию.
  4. Для сокращения времени, расходуемого на вызовы функций, особенно небольших, предусмотрены встраиваемые (inline) функции.
  5. Спецификация inline перед указанием типа результата в объявлении функции "советует" компилятору сгенерировать копию кода функции в соответствующем месте программы, чтобы избежать вызова этой функции. Компилятор может игнорировать спецификацию inline и обычно так и делает для всех функций, кроме самых малых.
  6. Имеются два способа обращения к функциям:
    • вызов по значению (cоздается копия аргумента и она передается вызываемой функции)
    • вызов по ссылке (возможность прямого доступа к передаваемым данным).
  7. Чтобы показать, что параметр функции передан по ссылке, после типа параметра в прототипе функции ставится символ амперсанда &.
    Например: int funс(int &b)
  8. Аргумент по умолчанию - это значение, которое присваивается параметру функции при ее вызове, если при этом соответствующий аргумент функции не указан.
  9. Параметры со значениями по умолчанию должны стоять после всех остальных параметров в списке.
    Например: int Sum(int a, int b, int c = 0, int d = 0);
