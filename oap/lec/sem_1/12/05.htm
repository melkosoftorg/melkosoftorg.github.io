Встраиваемые функции

В языке C++ для сокращения времени, расходуемого на вызовы функций, особенно небольших, предусмотрены встраиваемые (inline) функции.

Спецификация inline перед указанием типа результата в объявлении функции "советует" компилятору сгенерировать копию кода функции в соответствующем месте программы, чтобы избежать вызова этой функции.
В результате получается множество копий кода функции, встраиваемых в программу, вместо единственной копии, которой передается управление при каждом вызове функции.

Компилятор может игнорировать спецификацию inline и обычно так и делает для всех функций, кроме самых малых. 

	#include<conio.h>
	#include<iostream.h>
	inline void fun(char *str)
	{
		cout << str; 
	}
	void main()
	{	clrscr();
		fun("Пример встраиваемой функции\n");
		getch();
	}

И еще один простейший пример встраиваемой функции: 

	#include<stdio.h>
	#include<conio.h>
	inline int Max(int a, int b)
	{
		return a > b? a : b;
	}
	void main()
	{
		int x = 11, y = 22;
		printf("Max(%d, %d) = %d \n", x, y, Max(x, y));
		getch();
	}
