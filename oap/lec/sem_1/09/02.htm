Заголовочные файлы библиотеки iostream

Большая часть программ на языке C++ включает заголовочный файл <iostream.h>, который содержит основные сведения, необходимые для всех операций с потоками ввода- вывода, включает объекты cin, cout, cerr и clog, которые соответствуют стандартному потоку ввода, стандартному потоку вывода, не буферизованному и буферизованному стандартным потокам вывода сообщений об ошибках.

Предусмотрены возможности, как для форматированного ввода-вывода, так и для неформатированного.

Заголовочный файл <iomanip.h> содержит информацию, полезную для обработки форматированного ввода-вывода при помощи так называемых параметризованных манипуляторов потока.

Заголовочный файл <fstream.h> содержит важную информацию для проведения операций с файлами, обработка которых осуществляется под контролем пользователя.

Заголовочный файл <strstrea.h> содержит информацию, важную для выполнения форматированного ввода-вывода в память. Это похоже на обработку файлов, но операции ввода-вывода проводятся с символьными массивами, а не с файлами.

Заголовочный файл <stdiostr.h> включает нужные сведения для программ, использующих для выполнения операций ввода-вывода сочетание стилей языков C и C++.
