Работа со строкой символов

Программа иллюстрирует работу со строкой символов. В ней используется функция strlen(name) библиотеки string.h, позволяющая вычислить количество символов строки name. 
	#include<stdio.h>
	#include<conio.h>
	#include<string.h>
	#define PR "Какое великолепное имя!"
	main()
	{
		char name[50];
		int dl, dl1, pam, pam1; clrscr();
		printf("Как вас зовут? \n"); scanf("%s", name); 
		printf("Привет, %s. %s \n", name, PR);
		dl = strlen(name); pam = sizeof name;
		dl1 = strlen(PR); pam1 =sizeof(PR);
		printf("Имя состоит из %d букв, \n занимает %d ячеек памяти.\n", dl, pam);
		printf("Хвалебная фраза %s\nсостоит из %d букв, занимает %d ячеек памяти.\n", PR, dl1, pam1);
		getch(); return 0;
	}
		Результаты

		Как вас зовут? Игорь <Enter>
		Привет, Игорь. Какое великолепное имя!

		Имя состоит из 5 букв, 
		занимает 50 ячеек памяти.
		Хвалебная фраза "Какое великолепное имя!" 
		состоит из 23 букв, занимает 24 ячеек памяти.
При вводе name нуль-символ не надо помещать в конец массива, эта задача будет выполнена функцией scanf() при чтении вводимой строки.

PR - символическая строковая константа; кавычки, в которые заключена фраза, идентифицируют ее как строку, поэтому в ее конец будет помещен нуль-символ.

%s - напечатать строку.

Строка "к" не то же самое, что символ 'к':

  • 'к' - объект одного из основных типов (char), занимает 1 байт памяти,
  • "к" - объект произвольного типа (массива элементов типа char), состоит из двух символов 'к' и нуль символа, занимает 2 байта памяти.
