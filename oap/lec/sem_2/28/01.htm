Функции: инициализировать массив, вывести значения

При работе с массивом довольно часто приходится выполнять однотипные операции, такие как инициализация элементов массива случайными числами, вывод массива на экран. Поэтому есть смысл пополнить собственную библиотеку этими функциями.

Ниже приведены определения простейших из таких функций. Глобальная переменная k задает максимальное значение элемента. В функциях используются целочисленные переменные. В качестве параметров передается в функцию имя массива и количество элементов.

  • Для инициализации массива использована функция rand(). 

    // целочисленный массив
void setrand(int list[], int size) 
{
	int i;
	for (i = 0; i < size; i++) 
		list[i] = rand() % k;
}

    // массив вещественных значений
void setrand(float list[], int size) 
{
	int i;
	for (i = 0; i < size; i++) 
		list[i] = (rand() % k) * .11;
}

  • Для инициализации массива использована функция random(k) 

    // целочисленный массив
void setrandom(int list[], int size) 
{
	int i;
	for (i = 0; i < size; i++) 
		list[i] = random(k);
}

    // массив вещественных значений
void setrandom(float list[], int size) 
{
	int i;
	for (i = 0; i < size; i++) 
		list[i] = random(k) * .11;
}

  • Вывод на экран массива 

    // целочисленный массив
void get_mas(int list[], int size) 
{
	int i;
	for(i = 0; i < size; i++)
		printf("%7d", list[i]);
	printf("\n\n");
}

    // массив вещественных значений
void get_mas(float list[], int size) 
{
	int i;
	for(i = 0; i < size; i++)
		printf("%7.2f", list[i]);
	printf("\n\n");
}
