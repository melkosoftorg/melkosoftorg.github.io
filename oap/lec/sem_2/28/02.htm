Функция: поменять местами два числа

В алгоритмах сортировки (упорядочить по возрастанию, упорядочить по убыванию) очень часто приходится выполнять операцию: поменять местами два значения a и b.

Для реализации такого алгоритма введена дополнительная переменная temp, в которой временно хранится одна из переменных. 

// для целочисленных переменных
void swap(int *a, int *b) 
{
	int temp;
	temp = *a; 
	*a = *b; 
	*b = temp;
}

// для вещественных переменных
void swap(float *a, float *b) 
{
	float temp;
	temp = *a; 
	*a = *b; 
	*b = temp;
}
