Календарно-тематический план  


Повторительно-обобщающее занятие по теме:

"Классы и абстрагирование данных"

В результате изучения темы студент должен:

  • знать:
    • что такое классы, конструкторы и деструкторы;
    • объектно-ориентированные инструменты для создания новых типов данных;
    • как использовать наследование для создания новых классов при объектно-ориентированном программировании;
  • уметь:
    • использовать объектно-ориентированное программирование для создания новых классов из уже существующих;
    • создавать массивы объектов классов;
    • создавать, удалять динамические массивы объектов класса.

Ответить на вопросы

  1. Что понимают под объектно-ориентированным программированием ?

  2. Охарактеризуйте основные понятия объектно-ориентированного программирования:
    • инкапсуляция, пакетирование (Encapsulation),
    • наследование (Inheritance),
    • полиморфизм (Polymorphism) - многоформенность.

  3. Охарактеризуйте спецификатоды доступа: private, public, protected.

  4. Что такое класс и как его можно определить ?

  5. Что является компонентами класса ?

  6. Конструктор, его назначение, синтаксис определения в теле класса и вне тела класса.

  7. Какой вид имеет явный вызов конструктора при непустом списке фактических параметров?

  8. Какой вид имеет явный вызов конструктора, приводящий к созданию объекта, не имеющего имени?

  9. Деструктор, его назначение, синтаксис определения в теле класса и вне тела класса.

  10. Определение компонентной функции
    • в теле класса,
    • вне тела класса.

  11. Как можно создать объекты класса ?

  12. Укажите способы обращения к компонентам класса
    • с помощью уточненного имени,
    • через указатель на объект класса,
    • через ссылку.

  13. Для чего используется операция :: ?

  14. Определите класс Auto с компонентными данными: Марка автомобиля, Цена, Год выпуска.
    Предусмотрите методы для ввода значений и вывода информации на экран.

  15. Создайте возможные экземпляры объектов класса Auto.

  16. Охарактеризуйте наследование (inheritance) классов:
    • Преимущества использования наследования.
    • Какой класс называется базовым?
    • Какой класс называется производным?
    • Что такое множественное наследование?
    • Какой вид имеет шаблон производного класса?

  17. Какие классы называются
    • полиморфными?
    • абстрактными?
    • локальными?

  18. Перечислите три этапа объектно-ориентированного программирования, являющиеся объектно-ориентированным анализом предметной области.

  19. Укажите этапы программирования в объектно-ориентированном стиле, приводящие к созданию объектно-ориентированной среды (библиотеки классов), позволяющей решать задачи моделирования в определенной предметной области.
