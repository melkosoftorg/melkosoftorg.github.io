Примеры программного кода

  1. Составное условие ИЛИ: истина - True, если хотя бы одно из условий истинно.
    То есть, если хотя бы на одно из условий мы получает ответ да, будет выполнена ветвь да. 
    	int x = 2, y = 3, i;
	if(x >= 2 || y < 3)
		i = x * y;
	else
		i = x + y;
	printf("i = %d", i);

    Проверим условия: 2 >= 2 (да), 3 < 3 (нет).
    Выполняется команда i = x * y; (i = 2 * 3 = 6).
    Получим ответ: i = 6.

  2. В примере проверяется условие: если остаток от деления двух целых чисел равен нулю, то знак вещественного числа меняется на противоположный и оно выводится с тремя десятичными знаками, в противном случае число не изменяется и выводится с двумя десятичными знаками. 
    	int x = 7, y = 5; float m = -2.34567;
	if(x % y == 0)
	{
		m = -m; printf("%2.3f", m);
	}
	else
		printf("%2.2f", m);

    Проверим условие: 7 % 5 == 0 (нет, 2 != 0)
    Получим ответ: -2.35

  1. Составное условие И: истина - True, если каждое из условий истинно.
    То есть, на каждое из условий должен быть получен ответ да, чтобы была выполнена ветвь да. 
    	int x = 2, y = 3, i;
	if(x == 2 && y == 3)
		i = x * y / 2;
	i += x / 2 + y;
	printf("i = %d", i);

    Проверим условия: 2 == 2 (да), 3 == 3 (да).
    Выполняется команда i = x * y / 2; (i = 2 * 3 / 2 = 3).
    После этого управление передается следующей команде.
    То есть, имея значение i = 3, выполняем i += x / 2 + y; (i = 3 + 2 / 2 + 3 = 7).
    Получим ответ: i = 7.

  2. Составное условие И. В данном примере при проверке условий получаем: 2 > 2 (нет), 3 == 3 (да).
    Выполняется команда: i = x / 2 + y; (i = 2 / 2 + 3 = 4). 
    	int x = 2, y = 3, i;
	if(x > 2 && y == 3)
		i = x * y + 1;
	i = x / 2 + y;
	printf("i = %d", i);

    Получим ответ: i = 4.
