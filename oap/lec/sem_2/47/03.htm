Препроцессорные операции в строке замещения

В последовательности лексем, образующих строку замещения, предусматривается использование двух операций:

  • Операция # - помещается перед параметром, требует, чтобы текст, замещающий данный параметр в формируемой строке, заключался в кавычки.

    Рассмотрим макроопределение:   #define print(A) printf(#A" = %f", A)
    К макросу   print(A)   можно обращаться, подставляя вместо параметра   A   произвольные выражения, формирующие результаты вещественного типа.
    Например,   print(sin(M_PI / 2));
    В результате получим:   sin(M_PI / 2) = 1.000000

  • Операция ## - допускается только между лексемами строки замещения, позволяет выполнять конкатенацию (склеивание) лексем, включаемых в строку замещения.

    Рассмотрим макроподстановки в макроопределениях с одинаковым списком параметров и одинаковыми аргументами. 

    		float x = 1.0708, y = .5;	// фактические параметры
Макроопределения:
		#define zero(a, b, c, d) a(bcd)
		#define one(a, b, c, d) a(b c d)
		#define two(a, b, c, d) a(b##c##d)

    В результате получим следующее:

    Макровызов
    Результат макроподстановки
    Полученный результат
    cout << zero(sin, x, +, y);
    sin(bcd)
    		Ошибка: Undefined symbol 'bcd'
    cout << endl << one(sin, x, +, y);
    sin(x + y)
    		1 - пробелы сохранены в результате
    cout << endl << two(sin, x, +, y);
    sin(x+y)
    		1 - конкатенация аргументов без пробелов
