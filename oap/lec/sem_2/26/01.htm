Бит, байт, адрес

RAM - Random Access Memory - память произвольного доступа.

1 бит - информация типа 0 или 1 (ложь или истина).

Восемь бит группируются в один байт.

Большим группам бит, как правило, присваивается имя:

  • обычно два байта составляют слово,
  • четыре байта составляют длинное слово,
  • для IBM PC шестнадцать байт составляют параграф.

Каждый байт в памяти компьютера имеет собственный уникальный адрес

  • если данный байт имеет адрес N,
  • то предыдущий байт имеет адрес N - 1,
  • следующий байт - N + 1.

Когда компьютер работает, часть памяти используется операционной системой. При выполнении программы ее код (последовательность инструкций на машинном языке) и данные (информация, используемая программой) также занимает часть памяти.

При объявлении переменной в программе компилятор выделяет для хранения этой переменной участок памяти с уникальным адресом. Компилятор ассоциирует этот адрес с именем переменной. Всякий раз, когда в программе используется имя переменной, автоматически происходит обращение по адресу к соответствующему месту в памяти.

Адрес переменной - это просто число, с которым можно обращаться как с обычным числовым значением. Зная адрес переменной, можно завести вторую переменную и поместить в нее адрес первой.

Пусть объявлена и инициализирована переменная:   int ram = 122;
Компилятор выделил для нее ячейку памяти с адресом начального байта   1004   и ассоциировал имя  ram  с этим адресом.

Объявим переменную   p_ram   для хранения адреса переменной   ram.   Вначале она не инициализирована. Для нее выделяется память, но никакое значение туда не помещается.
Поместим адрес переменной   ram   в   p_ram.   Поскольку переменная   p_ram   теперь содержит адрес переменной   ram,   она указывает на   ram,   то есть является указателем на   ram.
