Примеры программного кода

  1. Цикл do ... while с вложенной полной развилкой (оператор if ... else).
    Тело цикла выполняется при значениях управляющей переменной цикла i равных 1, 3, 5, 7. Как только i принимает значение равное 9 - выходим из цикла. 
	int i = 1;
	do
	{	if(i < 3)
			printf("%d ", i);
		else 
			printf("%d ", i % 3);
	} while((i += 2) < 9);


	Получим ответ: 1 0 2 1
i Условие Вывод
1 1 < 3 (да) 1
3 3 < 9 (да)  
  3 < 3 (нет) 3 % 3 = 0
5 5 < 9 (да)  
  5 < 3 (нет) 5 % 3 = 2
7 7 < 9 (да)  
  7 < 3 (нет) 7 % 3 = 1
9 9 < 9 (нет)  

  1. Цикл for с вложенными последовательными полной развилкой (условный оператор ?:, составное условие И) и неполной развилкой (оператор if, пустая ветвь нет).
    Тело цикла выполняется при значениях управляющей переменной цикла i равных -2, -1, 0, 1. Как только i принимает значение равное 2 - выходим из цикла. 
	float x = 6.25; int i, m;
	for(i = -2; i < 2; i++)
	{
		m = (i < 0 && i % 2 == 0) ? ceil(x) : floor (x);
		if(i != 0)
			printf("%d %d ", m, m % i);
	}


	Получим ответ: 7 1 6 0 6 0
i Условие m Вывод
-2 -2 < 0 (да) И -2 % 2 == 0 (да) 7  
  -2 != 0 (да)   7 7 % (-2) = 1
-1 -1 < 2 (да)    
  -1 < 0 (да) И -1 % 2 == 0 (нет) 6  
  -1 != 0 (да)   6 6 % (-1) = 0
0 0 < 2 (да)    
  0 < 0 (нет) И 0 % 2 == 0 (да) 6  
  0 != 0 (нет)    
1 1 < 2 (да)    
  1 < 0 (нет) И 1 % 2 == 0 (нет) 6  
  1 != 0 (да)   6 6 % 1 = 0
2 2 < 2 (нет)    

  1. Цикл while с вложенной полной развилкой (условный оператор ?:, составное условие ИЛИ).
    Тело цикла выполняется при значениях управляющей переменной цикла i равных -1, 0, 1. Как только i принимает значение равное 2 - выходим из цикла. 
	float x = 6.25; int i = -1, m;
	while(i < 2)
	{
		m = (i < 0 || i % 2 == 0) ? pow(i, 2) : sqrt(x);
		printf("%d %d ", m, i++);
	}

	Получим ответ: 1 -1 0 0 2 1
i Условие m Вывод
-1 -1 < 2 (да)    
  -1 < 0 (да) ИЛИ -1 % 2 == 0 (нет) -1² = 1 1 -1
0 0 < 2 (да)    
  0 < 0 (нет) ИЛИ 0 % 2 == 0 (да) 0² = 0 0 0
1 1 < 2 (да)    
  1 < 0 (нет) ИЛИ 1 % 2 == 0 (нет) sqrt(6.25) = 2 2 1
2 2 < 2 (нет)
