Вложенные ветвления в цикле do...while

Организовать ввод положительных чисел с клавиатуры в окно ввода, снабженное тенью, размером (5, 5, 73, 22) в две колонки и 5 строк.
Окончанием ввода считать ввод с клавиатуры отрицательного числа или заполнение окна ввода.
Ремарки, поясняющие работу программы, расположены в программе. 

	#include<conio.h>
	#include<iostream.h>
	void main()
	{
		int x, i = 6, j = 20;
		// вычистить экран 11 цветом
		window(1, 1, 80, 25); textbackground(11); clrscr();

		// Построить окно с тенью шириной 68
		window(7, 6, 75, 23); textbackground(0); clrscr();
		window(5, 5, 73, 22); textcolor(7); textbackground(1); clrscr();

		// до тех пор, пока позиция вывода в пределах окна
		// И введено положительное число продолжать цикл
		do  
		{
			gotoxy(j, i++); cout << "x = "; cin >> x;
			if(i % 11 == 0)		// начать новую колонку
			{
				j += 25; i = 6;
			}
		} while(x >= 0 && j < 70);
		getch();
	}

Вычислить шесть значений функции при заданных значениях аргумента x.
Чтобы повторить вычисления шесть раз, воспользуемся циклом do...while. Введем переменную i - счетчик числа повторений, которая будет управлять циклом.
До цикла зададим начальное значение управляющей переменной цикла: i = 1.
В теле цикла изменим значение управляющей переменной цикла: i = i + 1.
Проверим условие окончания или повторения цикла: i < 7. Как только получим i = 7, вычисления прекратить.

Среди значений аргументов могут оказаться недопустимые для данной функции значения x = 0 или x = 1, при которых знаменатель обращается в 0, что приведет к ошибке - деление на ноль.
Чтобы избежать этого, неодходимо каждое введенное значение проверить, не является ли оно 0 или 1 и только после этого либо вычислять, либо выводить сообщение о недопустимом значении аргумента. В программе командой continue; организован пропуск опасной итерации.
Итак, мы имеем цикл с постусловием do...while с вложенной в него полной развилкой. 

#include<conio.h>
#include<iostream.h>
main()
{
	clrscr();
	float x, y;
	int i = 1;
	do
	{
		cout << "x = "; cin >> x;
		if(x == 0 || x == 1)
		{
			cout << '\t' << i << ". x = " << x << endl;
			continue;
		}
		y = 1 / x + 1 / (x - 1);
		cout << '\t' << i << ". x = " << x << "\ty = " << y << endl;
	} while(++i < 7);
	getch();
}

Тело цикла - это команды: ввести значение аргумента x, проверить значение аргумента на допустимость и выполнить соответствующую ветвь команды ветвления, нарастить значение управляющей переменной цикла.
