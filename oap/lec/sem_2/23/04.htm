Вложенные ветвления в цикле while

Рассмотрим алгоритм Евклида - нахождение наибольшего общего делителя двух чисел a и b.

Суть алгоритма Евклида можно представить следующими этапами:

  1. Обозревая два числа A и B, переходи к следующему пункту.
  2. Сравни обозреваемые числа (A = B, A < B, A > B) и переходи к следующему пункту.
  3. Если A и B равны, то прекрати вычисления: каждое из чисел дает искомый результат.
    Если числа не равны, то переходи к следующему пункту.
  4. Если первое число меньше второго, то переставь их местами; переходи к следующему пункту.
  5. Вычитай второе число из первого, обозревай два числа: вычитаемое и остаток; переходи к пункту 2.

По указаниям этого алгоритма, можно найти наибольший общий делитель для любой пары натуральных чисел.
Делитель данного числа - число, на которое данное число делится без остатка.
Общий делитель нескольких чисел - число, на которое все данные числа делятся без остатка.
Наибольший общий делитель - наибольший из общих делителей нескольких чисел.
Например, числа 42 и 105 имеют общие делители: 1, 3, 7, 21, наибольший из них - 21. 

#include<stdio.h>
#include<conio.h>
void main(void)
{
	int a, b, temp; clrscr();
	// ввод значений
	printf("Наибольший общий делитель чисел\n\n");
	printf("\ta = "); scanf("%d", &a);
	printf("\tb = "); scanf("%d", &b);
	while(a != b)
	{
		if(a < b)
		{	// поменять местами a и b
			temp = a;
			a = b;
			b = temp;
		}
		a -= b;
	}
	printf("\nравен %d", a);
	getch();
}

Если задать для a и b одинаковые значения, то тело цикла не будет выполнено ни разу.
Имеем цикл while с вложенной неполной развилкой.

Игра с компьютером "Угадай число".

Алгоритм: цикл while с вложенной в цикл полной развилкой с вложенной полной развилкой на ветви нет. 

#include<conio.h>
#include<stdlib.h>
#include<iostream.h>
#define NO  0
#define YES 1
void main()
{
	clrscr();
	int x1, x2 = 0, don = NO, num;
	// Генерируется случайное число от 0 до 32767
	randomize(); num = rand();
	// Чтобы понять программу - число выводится на экран
	cout << num << " - " << endl;
	cout << " ? " << endl;
	cout << "0 - " << RAND_MAX << endl;
	while(don == NO)
	{
		cout << " -> "; cin >> x1;
		x2++;			// количество попыток
		if(num == x1)
			don = YES;
		else if(num < x1)
			cout << " уменьши " << endl;
		     else
			cout << " увеличь " << endl;
	}
	cout << num << " за " << x2 << " раз " << endl;
	getch();
}

Результаты:

23295 -
 ?
0 - 32767
 -> 34
 увеличь
 -> 24562
 уменьши
 -> 23295
23295 за 3 раз

Сгенерировано число num = 23295.
Если ввести значение переменной x1 меньше сгенерированного числа num, будет выведен ответ увеличь.
Если ввести значение переменной x1 больше сгенерированного числа num, будет выведен ответ уменьши.
Если угадал, будет выведено 23295 за 3 раз.
