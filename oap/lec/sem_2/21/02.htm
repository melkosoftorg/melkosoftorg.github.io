Использование ключевого слова enum

В языке C списки определяются директивой #define

			#define sun 0
			#define mon 1
			#define tues 2
			#define wed 3
			#define thur 4
			#define fri 5
			#define sat 6

Современный метод позволяет представлять информацию более абстрактно и содержательно, чем длинный список директив #define. Кроме того, можно описать переменные, принадлежащие к абстрактному типу данных days.
В Borland C введен так называемый перечислимый тип данных, который можно объявить с помощью ключевого слова enum:

typedef enum { sun, mon, tues, wed, thur, fri, sat } days;

Присвоение осуществляется в порядке возрастания значений по умолчанию от sun = 0 до sat = 6. Можно выбирать значения, которые вы хотите присвоить константам, но они должны быть целого типа (включая тип char): 

		enum lev {low = 30, men = 70, im = 200};
Если вы присвоите какое-либо значение только одной константе, то константам, следующим за ней, будут присвоены последовательные значения, идущие за явно присвоенным значением.
Например, enum feline {cat = 20, tiger, lion, puma}; присваивает переменной tiger значение 21, переменной lion значение 22, переменной puma значение 23.

Рассмотрим пример программы, в которой рабочим дням недели присваиваются номера от 1 до 5, субботе - 6, воскресенью - 0 и в зависимости от введенного значения переменной today типа days выводится результат: 

	today равно 1 - 5        Иди работать!
	today равно 6             Убери во дворе и Расслабься!
	today равно 0             Расслабься!
	today не равно 0 - 6  Ошибка


	#include<conio.h>
	#include<stdio.h>
	typedef enum{sun, mon, tues, wed, thur, fri, sat} days;
	main()
	{
		days today;
		clrscr();
		printf("0 - 6\n"); scanf("%d", &today);
		switch (today)
		{
			case mon:	//понедельник
			case tues:	// вторник
			case wed:	// среда
			case thur:	// четверг
			case fri:		// пятница
				puts("Иди работать!");break;
			case sat:	// суббота
				printf("Убери во дворе и ");
			case sun:	// воскресенье
				puts("Расслабься!"); break;
			default:	// нет такого дня недели
				printf("Ошибка\n"); break;
		} getch(); return 0;
	}

Для значений от mon до fri выполняется одна и та же функции puts, после которой оператор break указывает на выход из switch. Если today равно sat, выполняется соответствующая функция printf, а затем выполняется puts ("Расслабься!"); если же today равно sun, выполняется только последняя функция puts.
