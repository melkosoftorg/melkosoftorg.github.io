Указатели на структуры как компоненты структур

Компонентами структур могут быть данные любых типов, за исключением структур того же типа, что и определяемый структурный тип.

Например, определим структуру 

		struct mix
		{	int n;
			double *d;
		};

и введем ее в качестве элемента в следующую структуру типа struct hole 

		struct hole
		{	struct mix exit;
			float b;
		};
Однако элементом структуры может быть указатель на структуру того же типа, что и определяемый структурный тип.
Рассмотрим на примере структурного типа химические элементы, где next - указатель на структуру того же типа, что позволит при необходимости пройтись по всем элементам таблицы. 
		struct element
		{	int number;
			double mass;
			char name[16];
			struct element *next;
		};

Указатели на структуры удобно использовать, когда надо организовать взаимные перекрестные связи между структурами разных типов. Например: 

		struct part
		{	double modul;
			struct cell *element_cell;
			struct part *element_part;
		};

		struct cell
		{	long summa;
			struct cell *one;
			struct part *two;
		};
