Цикл с постусловием - do ... while

Используется в тех случаях, когда вы не знаете точного количества повторов, но в то же время цикл необходимо выполнить хотя бы один раз. 

выражение 1;
do
{
	// тело цикла
	операторы;
	выражение 3;
} while (выражение 2);

Операторы внутри do ... while всегда выполняются хотя бы один раз (так как проверка условия выполнения цикла осуществляется после выполнения последовательности операторов составляющих тело цикла). 

x = a;
do
{
	// серия команд тела цикла
	операторы;
	x += h;
} while (x > b);

Правильность условия проверяется только в конце цикла.

  • выражение 1 - начальное значение управляющей переменной цикла;
  • выражение 2 - условие продолжения цикла;
  • выражение 3 - модификация (приращение) управляющей переменной за каждое выполнение цикла.

Цикл do ... while работает следующим образом:

  1. До цикла задается начальное значение управляющей переменной цикла x = a.
  2. Выполняются команды тела цикла.
  3. Изменяется значение управляющей переменной цикла x += h.
  4. Проверяется условие окончания или продолжения цикла x > b.
    • если условие ложно, то выполняются команды тела цикла (с пункта 2),
    • если условие истинно, выполнение тела цикла прекращается и выполняется команда, следующая за циклом.

Рассмотрим алгоритм построения вложенных квадратов с использованием цикла do ... while.
Длина стороны квадрата равна a. Координаты нижней левой вершины квадрата (x, y), а верхней правой - (x + a, y - a).
Построение начинается с внутреннего квадрата (a = 12) и повторяется до тех пор, пока a < 200.
Цвет рисования изменяется генератором случайных чисел в диапазоне от 0 до 14 (c = random(15)), стиль линии - точечная толщиной 3. 

#include<graphics.h>
#include<conio.h>
#include<stdio.h>
#include<stdlib.h>
#include"..\\h\\init.h"
main()
{
	init();
	randomize();
	setbkcolor(0); setlinestyle(1, 1, 3);
	int x = 310, y = 230, c, h = 6, a = 2 * h;
	do
	{
		c = random(15);
		setcolor(c);
		rectangle(x, y, x + a, y - a);
		x -= h; y += h; 
		a += 2 * h;
	} while(a < 200);
	getch(); closegraph();
}

Первоклассник Петя научился только складывать и вычитать. Поэтому, разжившись деньгами, решил делать по одной покупке, чтобы точно уложиться в имеющуюся сумму. Составим алгоритм планирования закупки товара в магазине на сумму, не превышающую заданную величину.
Известна цена x одной единицы товара и имеющаяся у покупателя сумма p.
Необходимо определить:

  • k - сколько единиц товара можно приобрести,
  • s - сколько он будет стоить,
  • o - сколько денег останется после покупки. 

#include<conio.h>
#include<iostream.h>
main()
{
	clrscr();
	float x, p, s, o;
	int k;
	cout << "x = "; cin >> x;
	cout << "p = "; cin >> p;
	// начальные значения сумм обнулить
	s = 0;
	k = 0;
	do
	{
		k++;
		s += x;
	} while(p - s >= x);
	o = p - s;
	cout << "k = " << k << "\ts = " << s << "\to = " << o;
	getch();
}

Тело цикла будет повторяться до тех пор, пока денег будет достаточно для следующей покупки. Как только остаток денег окажется меньше стоимости следующей покупки, выходим из цикла.
