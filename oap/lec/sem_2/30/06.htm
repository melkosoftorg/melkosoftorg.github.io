Функция: инициализировать двумерный массив

Мы рассматривали функции инициализации одномерного массива случайными числами.

Для инициализации массива vector можно написать в main setrand(vector, VS) Массиву vector будут присвоены случайные числа, являющиеся результатом работы датчика случайных чисел.

Чтобы модифицировать setrand для работы с двумерным массивом, надо изменить программу: 

	void setrand(int matrix[][CS], int rsize)
	{
		int i,j;
		for (i = 0; i < rsize; i++)
		{
			for (j = 0; j < CS; j++)
				matrix[i][j] = rand();
		}
	}

CS - глобальная константа, определяющая второе измерение массива, то есть, любой массив, передаваемый setrand, получит второе измерение массива, равное CS.

Можно сделать иначе. Пусть имеется массив matrix[15][7], который надо передать в setrand.

Используем следующее описание: setrand(int list[], int size)

Обращение к функции будет иметь вид: setrand(matrix, 15 * 7);

Массив matrix будет рассматриваться функцией setrand как одномерный массив, содержащий 105 элементов (15 строк * 7 столбцов), с которым будут произведены необходимые действия.
