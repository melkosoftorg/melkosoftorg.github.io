функция imin: возвратить индекс минимального числа массива

Рассмотрим функцию, возвращающую индекс минимального числа массива int: 
	int imin(int list[], int size)
	{
		int i, minindx, min;
		minindx = 0;
		min = list[minindx];
		for (i = 1; i < size; i++)
			if (list[i] < min)
			{
				min = list[i];
				minindx = i;
			}
		return(minindx);
	}
Одна из важных особенностей C:
необязательно знать при трансляции величину list[], потому что компилятор считает list[] начальным адресом массива, и не заботится о том, где его конец.
