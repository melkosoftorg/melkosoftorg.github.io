Повторительно-обобщающее занятие по теме:

"Операторы передачи управления"

В результате изучения темы студент должен:

  • знать:
    • типы базовых алгоритмических структур и способы их соединения в сложных алгоритмах.
    • синтаксис операторов передачи управления;
  • уметь:
    • использовать операторы ?:, if и if-else для принятия решения в соответствии со значением выражения;
    • использовать операторы switch ... case ... default ... break для упрощения вложенных конструкций if-else;
    • использовать операторы for, while, do-while для создания циклов.

    1. Деление обыкновенных дробей по правилам:
      1. Числитель первой дроби умножить на знаменатель второй дроби.
      2. Знаменатель первой дроби умножить на числитель второй дроби.
      3. Записать дробь, числитель которой есть результат выполнения пункта 1, а знаменатель - результат выполнения пункта 2.
    2. Вычисление значения функции по формуле y = (1 + x)2 / (2 x).
    3. Вычисление произведения всех целых чисел от M = -4 до тех пор пока произведение меньше 150 (на 0 не умножать).
    4. Ввод последовательности символов до тех пор, пока не встретился символ z
    5. Вычисление длины окружности и площади круга радиуса R.
    6. Вычисление значения функции по формуле y = (1 + x)2.
    7. Вычисление наибольшего общего делителя двух натуральных чисел M и N по правилам:
      1. Если числа равны, то взять их общее значение в качестве ответа; в противном случае продолжить выполнение алгоритма
      2. Определить большее из чисел
      3. Заменить большее число разностью большего и меньшего значений
      4. Вернуться к выполнению пункта 1
    8. Нахождение корней квадратного уравнения a x2 + b x + c = 0:
      1. Найти дискриминант D = b2 - 4∙a∙c.
      2. Если D > 0 - два действительных различных между собой корня: x1, 2 = (-b ± D½) / (2∙a).
      3. Если D = 0 - два действительных равных между собой корня: x1, 2 = -b / (2∙a).
      4. Если D < 0 - оба корня уравнения мнимы.
    9. Нахождение значений функции y = ex на отрезке [2, 7] с шагом 0.45.
    10. Нахождение площади кольца, внутренний радиус которого равен r, а внешний R.
