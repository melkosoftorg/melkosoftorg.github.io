Обращение к битовым полям

Для обращения к битовым полям используются те же конструкции, что и для обращения к обычным элементам структур: 
			имя_структуры.имя_поля

			(*указатель_на_структуру).имя_поля

			указатель_на_структуру -> имя_поля

			ссылка_на структуру.имя_поля
Для введения структуры XX и указателя *px допустимы операторы: 
			XX.a = 1; 
			px = &xx; 
			px -> b = 2;

В любом случае битовые поля следует воспринимать как небольшие целые числа без знака с заданным количеством двоичных разрядов.

Битовое поле длиной n бит может содержать число из диапазона от 0 до 2n - 1 включительно.

Если попытаться поместить в такое поле число, не входящее в допустимый диапазон, то сообщения об ошибке не будет, но результаты окажутся непредсказуемыми.

Следующая программа показывает, как влияет на объем памяти, выделяемый под экземпляр структурного типа, наличие битовых полей. 

	#include<conio.h>
	#include<iostream.h>
	void main()
	{
		// Объявление шаблона структуры point
		struct p1
		{
			int x;
			int y;
		} xx;
		struct p2
		{
			int x : 6 ;
			int y;
		} yy;
		cout << "Количество байт памяти для структуры" <<  endl;
		cout << "\tБез битовых полей: " << sizeof(xx) <<  endl;
		cout << "\tС битовыми полями: " << sizeof(yy) <<  endl;
		getch();
	}

	Количество байт памяти для структуры
		Без битовых полей: 4
		С битовыми полями: 3
