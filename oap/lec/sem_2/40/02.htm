Определение структуры с битовыми полями

имеет формат: 
struct
{
	тип_поля имя_поля : ширина_поля;
	тип_поля имя_поля : ширина_поля;
	... 
} имя_структуры;
   
	struct
	{
		int a : 1;
		int b : 2;
		float c;
		
	} XX, *px;

  • тип_поля - один из базовых целых типов int, char, short, long и их знаковые и беззнаковые варианты.
  • имя_поля - идентификатор, выбираемый пользователем.
  • ширина_поля - целое неотрицательное десятичное число, значение которого не должно превышать длины слова конкретной ЭВМ (0 - 16).

Элементами структуры могут быть как битовые поля, так и поля обычных типов. Битовые поля принято ставить в начале определения структуры, перед всеми остальными полями.

Инициализировать, например, прямым присваиванием элементы структуры XX можно следующим образом: 

			XX.a = 0;
			XX.b = 1;
			XX.c = 5.67;
