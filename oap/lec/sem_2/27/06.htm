Получение адреса элементов массива

Определим адреса четырех элементов типа int и float. Для хранения значений типа int будет выделено по 2 байта, типа float - по 4 байта. 
	#include<stdio.h>
	#include<conio.h>
	int main()
	{
		int dates[4], *pti, index;
		float bills[4], *ptf;
		// присваивает адрес указателю массива
		pti = dates;
		ptf = bills;
		for(index = 0; index < 4; index++)
			   printf("указатели + %d: int %6p float %6p\n", index, pti+index, ptf+index);
		getch(); return 0;
	}

	Результаты:

			указатели + 0:     int 0FF8     float 0FE0
			указатели + 1:     int 0FFA     float 0FE4
			указатели + 2:     int 0FFC     float 0FE8
			указатели + 3:     int 0FFE     float 0FEC

Обратите внимание, что массивы dates и bills не инициализированы, то есть, значения им не присвоены, но тем не менее необходимый объем памяти выделен.
