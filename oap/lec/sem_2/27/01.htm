Понятие массива и его определение

Переменные, с которыми мы имели дело до сих пор, хранили только одно значение (текущее значение) в каждый момент времени.

Массив - индексированный набор данных определенного типа.
  1. Все элементы массива имеют один и тот же тип и расположены в памяти подряд.
  2. Каждый элемент или значение в массиве является отдельной переменной и с ней можно обращаться как с обычной переменной, но так как они собраны в массив, их можно трактовать как общность.

Определение одномерного массива: 

			тип имя_массива [константное выражение];
  • тип - тип данных элементов массива,
  • константное выражение определяет размер массива - количество элементов массива (N).
При определении массива может выполняться его инициализация, то есть элементам массива задаются конкретные значения.
Например: int tin[6] = {10, 28, 35, 2, 78, 9};
Первый элемент массива - tin[0],       последний элемент массива - tin[6-1].

Общий объем памяти в байтах определяется выражением N*(sizeof(тип)).

Если значений меньше, чем элементов в массиве, оставшиеся элементы автоматически получат нулевые значения. 

Например,	инициализация массива          int a[10] = {5, -3, 9, 2}; 
		равноценна инициализации   int a[10] = {5, -3, 9, 2, 0, 0, 0, 0, 0, 0};

Нулевые значения всем элементам массива можно присвоить с помощью объявления int b[10] = {0};

Несколько описаний массивов 

	int temp[36];			// внешний массив из 36 целых чисел
	main()
	{
		float rain[45];		// автоматический массив из 45 чисел типа float
		static char code[12];	// статический массив из 12 символов
		extern temp[];		// внешний массив, размер указан выше
	}

В языке C указатели и массивы тесно связаны. Имя массива без квадратных скобок является указателем на первый элемент массива.
Это имя к тому же является постоянным указателем (адресной константой): его нельзя изменить и оно остается фиксированным все время выполнения программы.

Пусть объявлен массив   int data[10];
Тогда data - адрес первого элемента массива, который можно получить иначе: &data[0].
