Корни нелинейных уравнений

Приближенное значение корня нелинейного уравнения

Уточнить корень уравнения ln(x) - x + 5 = 0 с точностью t = 10-5 методом итерации на отрезке [5, 100].
Выразим аргумент x как функцию x, то есть x = f(x). Имеем x = ln(x) + 5.
Значением корня уравнения будет точка пересечения графиков функций y1 = x и y2 = ln(x) + 5.
В этой точке значение уравнения будет приблизительно (с точностью t = 10-5) равно нулю.

Обозначим x0 - начальное приближение, n - счетчик числа итераций. 

#include<conio.h>
#include<iostream.h>
#include<math.h>
void main()
{
	int n; float t = 1e-5, x, x0, x1, b, y;
	for(x = 5; x <= 100; x += 95)
	{
		x0 = x; n = 1; cout << "x0 = " << x;
		while( (b = fabs((x1 = log(x0) + 5) - x0)) > t)
		{
			n++;
			x0 = x1;
		}
		y = log(x1) - x1 + 5;
		cout << "\tx" << n << " = " << x1 << "\ty = " << y << endl;
	 }
	getch();
}

При исполнении программы получим:

	x0 = 5	x8 = 6.936847	y = 5.769464e-07
	x0 = 100	x9 = 6.936848	y = -2.392485e-07
График функций
    y1 = x

    y2 = ln(x) + 5
