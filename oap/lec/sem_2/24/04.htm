Примеры программного кода

  1. Вложенные циклы for.
    Управляющая переменная внешнего цикла j, принимает значения 1, 2, 3; выход из цикла при j = 4.
    Управляющая переменная внутреннего цикла i, принимает значения 2, 5, 8; выход из цикла при i = 11. Внутренний цикл - пустой, его телом является ;. 
	int i, j;
	for (j = 1; j < 4; j++)
	{
		for(i = 2; i < 10; i += 3);
		printf("%d ", i * j);
	}


	Получим ответ: 11 22 33

  1. Вложенные циклы do ... while.
    Управляющая переменная внешнего цикла i, принимает значения 7, 10; выход из цикла при i = 13.
    Управляющая переменная внутреннего цикла j, принимает значения 2, 3; выход из цикла при j = 4. 
	int i = 7, j;
	do
	{	j = 2;
		do
		{	j++;

  1. Вложенные циклы for и while.
    Управляющая переменная j внешнего цикла for принимает значения 1, 2; выход из цикла при j = 3.
    Управляющая переменная i внутреннего цикла while, принимает значения 8, 9, 17, 18; выход из цикла при i = 26. 
	int i, j;
	for(j = 1; j < 3; j++)
