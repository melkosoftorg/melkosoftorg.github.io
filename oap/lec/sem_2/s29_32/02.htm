Файлы и потоки

В C++ каждый файл рассматривается как последовательный поток байтов. Каждый файл завершается или маркером конца файла (EOF - end-of-file marker) или указанным числом байтов, записанным в служебную структуру данных поддерживающей системой.

Когда файл открывается, то создается объект и с этим объектом связывается поток. Автоматически создаются четыре объекта:

  • cin - объект стандартного потока ввода - дает возможность программе вводить данные с клавиатуры,
  • cout - объект стандартного потока вывода - позволяет программе выводить данные на экран,
  • cerr и clog - объекты стандартного потока ошибок - позволяют программе выводить на экран сообщения об ошибках.

Потоки, связанные с этими объектами, обеспечивают каналы связи между программой и отдельными файлами или устройствами.

Для обработки файлов в C++ должны быть включены заголовочные файлы <iostream.h> и <fstream.h>.

Файл <fstream.h> включает определения классов потоков

  • ifstream - для ввода из файла,
  • ofstream - для вывода в файл,
  • fstream - для ввода-вывода файлов.

Файлы открываются путем создания объектов этих классов потоков. Эти классы потоков являются производными (то есть наследуют функциональные возможности) соответственно от классов istream, ostream и iostream.

Пояснения: функции-элементы, операции и манипуляторы, описанные в теме  Потоковый ввод - вывод,  могут быть также применены и к потокам файлов.
