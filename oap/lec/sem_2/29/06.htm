Строим оси координат

Для построения осей координат необходимо знать положение начала координат, то есть положение на экране точки с координатами ( x0, y0 ).

Рассчитаем координаты начала координат для всех возможных ситуаций: 

		x0 = 2 * ox ;
		if(a <= 0)
			x0 += ceil(fabs(a)) * mx;

		y0 = 2 * oy;
		if(ymax >= 0)
			y0 += (ceil(ymax) * my);

Построим оси координат, обозначим их направление и нанесем единичные отрезки.
Удобно наносить деления окружностями радиусом в один пиксель, начиная от начала координат вверх и вниз для оси OY, влево и вправо для оси OX.
Деления на оси координат необходимо нанести, учитывая значения полученных масштабных отрезков, для чего надо проанализировать эти значения:

  • если значения mx и my окажутся слишком маленькими использовать значение шага равным, например, mx ∙ 10, my ∙ 100.
  • если значения mx и my очень большие - значение шага можно уменьшить, установить, например, mx ∙ .01, my ∙ .1 
		// ось OX
		line(ox, y0, xe - ox, y0);
		line(xe - ox, y0, xe - ox - 10, y0 - 3); line(xe - ox, y0, xe - ox - 10, y0 + 3);
		for(i = x0; i < (xe - 2 * ox); i += mx)
			circle(i, y0, 1);
		for(i = x0; i > 2 * ox; i -= mx)
			circle(i, y0, 1);

		// ось OY
		line(x0, ye - oy, x0, oy);
		line(x0 - 3, oy + 11, x0, oy); line(x0 + 3, oy + 11, x0, oy);
		for(i = y0; i < ye - 2 * oy; i += my)
			circle(x0, i, 1);
		for(i = y0; i > 2 * oy; i -= my)
			circle(x0, i, 1);

Теперь можно вызвать функцию graph() с набором фактических параметров для построения графика функции и на этом завершить функцию osi(). 

		graph(mx, my, x0, y0, kol);
	}
