Строим точечный график

Теперь можно построить график функции по точкам, количество которых kol и их координаты (xi, yi) известно. Функция может быть следующей: 

	void graph(int mx, int my, int x0, int y0, int kol)
	{
		for(int i = 0; i < kol ; i++)
		{
			putpixel(x0 + x[i] * mx, y0 - y[i] * my, 2);
		} 
	}
