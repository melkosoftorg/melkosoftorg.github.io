Создаем массив значений аргументов и результатов

Сформировать массивы N значений xi и yi заданной функции на отрезке [a, b].

Пусть на отрезке [-1, 3] необходимо в N = 200 точках вычислить и запомнить значение функции

Тогда программный код может выглядеть следующим образом: 

	const int N = 200;
	float x[N + 1], y[N + 1];
	float a = -1, b = 3, xn;
	int num = 0;
	cout << "Вычисление значений функции y = x * x * sin(M_PI * x)";
	for(xn = a; xn <= b; xn += (b - a) / N)
	{	x[num] = xn;
		y[num] = x[num] * x[num] * sin(M_PI * x[num]);
		num++;
	}
