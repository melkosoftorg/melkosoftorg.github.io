Строим кусочно - линейный график

При построении кусочно - линейного графика функции необходимо соседние точки соединить прямой.

В программе для наглядности построение каждой очередной точки заменено окружностью.
Кроме того количество точек уменьшено до 15, в результате чего четко видна ломаная линия графика. 

	void graph(int mx, int my, int x0, int y0, int kol)
	{
		circle(x0 + x[0] * mx, y0 - y[0] * my, 2);
		for(int i = 1; i < kol ; i++)
		{
			lineto(x0 + x[i] * mx, y0 - y[i] * my);
			circle(x0 + x[i] * mx, y0 - y[i] * my, 2);
		} 
	}
