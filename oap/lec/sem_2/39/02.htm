Понятие объединения

Со структурами "в близком родстве" находятся объединения, которые вводятся с помощью служебного слова union.

Объединение - это средство, позволяющее запоминать данные различных типов в одном и том же месте памяти.

В отличие от структуры, всякий раз только один из элементов объединения доступен для обращения. Причина этого проста: все элементы объединения занимают одну и ту же область памяти, как бы располагаясь один поверх другого.

Назначение объединений - обеспечить возможность доступа к одному и тому же участку памяти с помощью объектов разных типов.

Как и для структур, для объединений может быть введен объединяющий тип: 

			union имя_объединяющего_типа
			{
				элементы_объединения
			};

Например: 

			union mixt
			{
				double d;
				long f[3];
				int k[4];
			};

Введя тип объединения mixt, можно определять конкретные объединения, их массивы, указатели и ссылки на объединения: 

		mixt ma, mb[4];		// Объединение и массив объединений
		mixt *pt;			// Указатель на объединение
		mixt &rmix = ma;		// Ссылка на объединение

Каждый из созданных экземпляров может содержать или вещественное d, или массив длинных целых f, или массив целых k, но не все значения одновременно.
