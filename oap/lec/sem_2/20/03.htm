Примеры программного кода

Условный оператор ?: с составным условием И.
Если все условия истинны, выполняется команда после вопросительного знака, в противном случае - после двоеточия. 

	int x = 3, y = -3, i;
	i = (x <= 3 && y > 0) ? x * y : x + y;
	printf("i = %d", i);

Проверим условия: 3 <= 3 (да), -3 > 0 (нет).
Выполняется команда i = x + y; (i = 3 + (-3) = 0).
Получим ответ: i = 0.

Условный оператор ?:
Строковой переменной будет присвоено значение больше, если условие истинно, в противном случае будет присвоено значение меньше. 

	char *str; float k = 1.3786; 
	int x = ceil(k), y = floor(k);
	str = (x > y) ? " больше " : " меньше ";
	printf("%d%s%d", x, str, y);

Вычислим значения переменных:
x = ceil(k) (x = ceil(1.3786) = 2)
y = floor(k) (y = floor(1.3786) = 1)
Проверим условие: 2 > 1 (да).
Получим ответ: 2 больше 1.
