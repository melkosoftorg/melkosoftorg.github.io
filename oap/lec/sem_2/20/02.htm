Составной условный оператор   ?  :

Условный оператор может быть составным, например, при нахождении положения точки A(x, y):
  • точка A(x, y) принадлежит окружности радиуса   r,   если   x2 + y2 = r2,
  • точка A(x, y) находится внутри круга  радиуса     r,   если   x2 + y2 < r2,
  • точка A(x, y) находится  вне   круга   радиуса       r,   если   x2 + y2 > r2.

Оператор ветвления используется для организации разветвляющихся алгоритмов. В довольно большом количестве случаев вместо этого оператора можно использовать условную операцию. Например, вычислить значение функции:

Оператор ветвления 

	#include<stdio.h>
	#include<math.h>
	void main()
	{
		 float x, y;
		 printf(“X = ”); scanf(“%f”,&x);
		 if(x < 0) y = x*x;
		   else if(x < 5) y = x + 5;
		     else y = sqrt(x);
		 printf(“Y = %.2f\n”, y);
	}

Условная операция 

	#include<stdio.h>
	#include<math.h>
	void main()
	{
		 float x, y;
		 printf(“X = ”); scanf(“%f”,&x);
		 y=(x < 0) ? (x*x) : ((x < 5) ? x+5 : sqrt(x));
		 printf(“Y = %.2f\n”, y);
	}
