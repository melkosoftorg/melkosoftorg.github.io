Преобразование строки в длинное целое значение

strtol преобразует строку в длинное целое значение, возвращает значение преобразованной строки или 0 в случае ошибки, прекращает чтение строки на первом символе, который нераспознан.

Синтаксис long strtol(char *str, char **endptr, int base);
Прототип в stdlib.h

Если параметр base лежит в пределах между 2 и 36, длинное целое число выражается по основанию base.
Если base равно 0, первые несколько символов строки str определяют основание преобразуемого значения.

  • Если base есть 1, это будет считаться неверным значением.
  • Если base < 0, также будет считаться неверным значением.
  • Если base < 36, это будет неверным значением.
Любое неверное значение для base устанавливает результат в 0 и присваивает указателю текущего символа - указатель на начало строки.
Если подразумевается, что значение str будет интерпретироваться как десятичное, любой символ, кроме цифр от 0 до 9, не будет распознаваться.
Если значение str будет интерпретироваться, как число по любому другому основанию, будут распознаваться только те цифры и буквы, которые имеются в числах по данному основанию. 
Например,	base = 5, будут распознаваться цифры от 0 до 4;
		base = 20, будут распознаваться цифры от 0 до 9 и буквы от А до J.

Аргумент str - это последовательность символов, которые могут интерпретировать, как длинное целое.
Они должны соответствовать формату: 

			[ws] [sn] [0] [x] [ddd]

			[ws]	- необязательный	незначащий символ;
			[sn]	- необязательный	знак (+ или -);
			[0]	- необязательный	нуль (0);
			[x]	- необязательный	x или X.
			[ddd]	- необязательные	цифры.
