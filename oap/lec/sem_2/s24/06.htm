Преобразование целого значения
в строку символов

itoa преобразует целое значение в строку символов, может возвращать до 17 байт, возвращает адресный указатель на строку string, завершения по ошибке не существует.

ltoa преобразует длинное целое значение в строку символов, может возвращать до 33 байт, возвращает адресный указатель на строку string, завершения по ошибке не существует.

ultoa преобразует длинное целое без знака в строку, может возвращать до 33 байт, возвращает адресный указатель на строку string, завершения по ошибке не существует. 

Синтаксис	char *itoa (int value, char *string, int radix);
		char *ltoa (long value, char *string, int radix);
		char *ultoa(unsigned long value, char *string, int radix);
Прототип в	stdlib.h
Данные функции конвертируют значения value в строку с нулевым окончанием и записывают результат в параметр string.

Для функции itoa значение value - целое,
для функции ltoa значение value - длинное целое,
для функции ultoa значение value - длинное целое без знака.

Параметр radix определяет базу, которая будет использована при конвертировании значения value; она должна лежать между 2 и 36 (включительно).
Для функций itoa и ltoa, если значение value отрицательно, и значение radix 10, первый символ строки string - знак минус ( - ).
Для функции ultoa иначе. Кроме того, функция ultoa не производит проверки на переполнение.

Примечание: пространство, выделяемое для параметра string должно быть достаточным, чтобы записать полученную в результате работы функции строку, включая нулевое окончание \0.
