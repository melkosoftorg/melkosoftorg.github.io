Размер файла в байтах

Рассмотрим два способа определения размера файла в байтах.

  • Способ 1 - используя функцию filelength(handle). 
    		i = 0; clrscr();
		printf("\n\nЧтение данных из файла %s\n\n", searchpath(name));
		int handle = open("proba.dat", O_CREAT);
		printf("Длина файла в байтах  %ld\n", filelength(handle));

    Можем получить следующий результат: 

    		Чтение данных из файла   C:\GRUPPA\POV_10\PROBA.DAT
		Длина файла в байтах  12600
  • Способ 2 - посимвольное чтение из файла пока не конец файла (EOF) и подсчет количества символов. 
    		if ((fp = fopen(name, "r")) == NULL)
		{	puts("Невозможно открыть файл");
			exit(1);
		}
		while((ch = fgetc(fp)) != EOF)
		{	i++;
			printf("%c", ch);
		}
		printf("\n\n%d  байт", i); fclose(fp);

Результат в обоих случаях должен получиться один и тот же.
