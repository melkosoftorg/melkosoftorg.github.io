Теперь посчитаем количество в списке женщин и мужчин, средний рост всех.
Счетчикам количества женщин nf, количества мужчин nm, переменной sum, в которой будет накапливаться рост всех людей списка, заданы начальные значения, равные нулю.
Функция strstr(lib[i].aut, "жен.") позволяет осуществлять сортировку по признаку пола. 

		int i, nf = 0, nm = 0; float sum;
		sum = 0;
		for(i = 0; i < col; i++)
		{	sum += lib[i].val;
			if(strstr(lib[i].aut, "жен."))
			{	frau[nf] = lib[i];
				nf++;
			}
			else
			{	man[nm] = lib[i];
				nm++;
			}
		}
		cout << "\nВ списке женщин " << nf << " мужчин " << nm << "\n\n";
		cout << "Ср. рост всех = " << sum / col << "\n\n";
