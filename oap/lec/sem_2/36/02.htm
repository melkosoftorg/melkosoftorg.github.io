Описание массива структур

Процесс описания массива структур аналогичен описанию любого другого типа массива. Например, для MAXB = 100 экземпляров книг: 

			const int MAXB = 100;
			struct book lib [MAXB];

Оператор объявляет lib массивом, состоящим из MAXB элементов.
Каждый элемент массива представляет собой структуру типа book, то есть

  • lib - имя массива, содержащего структуры.
  • lib[0] - book-структура, имя структурной переменной,
  • lib[1] - вторая book-структура и так далее,

Рассмотрим на примере следующего списка книг:

lib[0] - "Искусство программирования для ЭВМ", "Д. Кнут", 1200
lib[1] - "ПЛ/1 для программистов", "Р.Скотт, Н.Сондак", 900
lib[2] - "Язык программирования Си", "Б.Керниган, Д.Ритчи", 300
  . . .
