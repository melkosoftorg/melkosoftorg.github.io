Нестатический компонент класса

Нестатический компонент класса может быть указателем или ссылкой на объект того же класса. Такая возможность позволяет формировать связные списки (цепочки) объектов одного класса. Статическим компонентом класса может быть указатель на объект класса. Это позволит однозначно определить начало связного списка объектов класса.
Например, для моделирования списка можно ввести класс такой структуры: 
	class list
	{	...			// Собственные компоненты
		public:
			static list *begin;	// Начало связного списка
			...
	};
	...
	list *list :: begin = NULL;		// Инициализация статического компонента
	...

Полное определение класса, моделирующего список, не приводится.
