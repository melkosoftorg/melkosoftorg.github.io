Работа с элементами столбца

Если необходимо работать с элементами столбца:
  • Определить номера столбцов, в которых значения всех элементов положительные (отрицательные).
  • Найти максимальный (минимальный) элемент каждого столбца и его индексы.
  • Вычислить сумму (произведение) элементов каждого столбца.
  • Определить количество положительных, отрицательных, нулевых элементов каждого столбца.
внешний цикл должен быть по индексу столбца: вошли в столбец и проанализировали каждый элементы каждой строки этого столбца (номер строки изменяется от 0 до N - 1). Только после этого перешли к следующему столбцу и опять прошли по всем элементам теперь этого столбца, и так далее по всем столбцам матрицы.

Например, для массива mas[3][4] найдем в каждом столбце сумму отрицательных элементов.

Мы должны войти в столбец (внешний цикл), задать начальное значение sum = 0, пройти по всем строкам столбца (внутренний цикл), в теле внутреннего цикла накопить сумму значений, вывести каждое отрицательное число столбца, затем в теле внешнего цикла вывести их сумму. 

        for(j = 0; j < 4; j++)
        {
                sum = 0;
                for(i = 0; i < 3; i++)
                        if (mas[i][j] < 0) 
                        {
                                sum += mas[i][j];
                                gotoxy(1 + 7 * j, 8 + i); printf("%d", mas[i][j]);
                        }
                gotoxy(40, 8 + i); 
                printf("В %d столбце сумма отрицательных элементов - %d ", i, sum);
        }
