Работа с элементами строки

Когда необходимо работать с элементами строки:
  • Определить номера строк, в которых значения всех элементов положительные (отрицательные).
  • Найти максимальный (минимальный) элемент каждой строки и его индексы.
  • Вычислить сумму (произведение) элементов каждой строки.
  • Определить количество положительных, отрицательных, нулевых элементов каждой строки.
внешний цикл должен быть по индексу строки: вошли в строку и проанализировали каждый элемент этой строки (номер столбца изменяется от 0 до N - 1). Только после этого перешли в новую строку и опять прошли по всем элементам теперь этой строки, и так далее по всем строкам матрицы.

Например, для уже сформированного массива mas[3][4] найдем в каждой строке количество элементов больших заданного числа p.

Мы должны войти в строку (внешний цикл), задать начальное значение kol = 0, пройти по всем столбцам строки (внутренний цикл), в теле внутреннего цикла накопить количество значений, затем в теле внешнего цикла вывести результат. 

        for(i = 0; i < 3; i++)
        {
                kol = 0;
                for(j = 0; j < 4; j++)
                        if (mas[i][j] > p) 
                        {
                                kol++;
                                gotoxy(2+7*kol, 6); printf("%d", mas[i][j]);
                        }
                gotoxy(40, 8); 
                printf("В %d строке элементов, больших %d - %d ", i, p, kol);
        }
