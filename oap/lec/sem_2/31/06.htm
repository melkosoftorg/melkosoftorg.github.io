Квадратная матрица и ее диагональные элементы

Если количество строк и количество столбцов двумерного массива (матрицы) одинаково, то такая матрица называется квадратной.

Например, если количество строк и количество столбцов равно N, то целочисленный двумерный массив - квадратная матрица: int mas[N][N];

К списку решаемых задач добавляется работа с диагональными элементами.

Различают диагонали квадратной матрицы:

  • Главную - совокупность элементов матрицы, у которых совпадают номера строки и столбца: mas[0][0], mas[1][1], mas[2][2], ...

      0 1 2 3
    0 -34 167 763 0
    1 483 -679 -31 1
    2 -83 79 231 -21
    3 -183 179 123 121

    Над главной диагональю находятся элементы индексы столбцов которых больше индексов строк.
    Под главной диагональю находятся элементы индексы столбцов которых меньше индексов строк.

  • Побочную - совокупность элементов матрицы, у которых сумма индексов на единицу больше порядка матрицы (порядок матрицы - число строк).
    Поскольку в C++ индексация строк и столбцов, в отличие от математики, начинается не с 1, а с 0, то сумма индексов должна быть на единицу меньше числа строк.
    Например, для матрицы int mas[4][4]; это следующие элементы: mas[0][3], mas[1][2], mas[2][1], mas[3][0].

  0 1 2 3
0 -34 167 763 0
1 483 -679 -31 1
2 -83 79 231 -21
3 -183 179 123 121

Над побочной диагональю находятся элементы сумма индексов строк и столбцов которых меньше N-1.
Под побочной диагональю находятся элементы сумма индексов строк и столбцов которых больше N-1.

В приведенном ниже фрагменте программы вычисляется сумма элементов побочной диагонали, выводятся значения элементов главной диагонали. 

        #define S 4
	int mas[S][S] = {	{-34, 167, 763, 0}, 
					{483, -679, -31, 1},
					{-83, 79, 231, -21},
					{-183, 179, 123, 121}
				};
