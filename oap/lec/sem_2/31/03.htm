Работа со всеми элементами матрицы

При решении задач вида:
  • Вычислить сумму (произведение) всех элементов матрицы.
  • Найти максимальный (минимальный) элемент матрицы и его индексы.
  • Подсчитать количество элементов положительных, отрицательных, нулевых, больших или меньших заданного числа.
нам приходится просматривать все элементы матрицы безразлично в какой последовательности: по строкам или по столбцам.

При этом, если необходимо задать какие-либо начальные значения, сделать это надо до внешнего цикла.

Например, подсчитать количество элементов в массиве mas[3][4], больших заданного числа p = 1600. 

        int mas[3][4] = {{   -34,       0,       11,  105},
                         { 4534, 1567,    -191,     5},  
                         {19374,    -1,  31412,  -15}}; 
        int i, j, p = 1600, kol = 0;
        for(i = 0; i < 3; i++)
                for(j = 0; j < 4; j++)
                        if (mas[i][j] > p) 
                        {
                                kol++;
                                gotoxy(2+7*kol, 6); printf("%d", mas[i][j]);
                        }
        gotoxy(40, 8); printf("В массиве элементов, больших %d - %d ",  p, kol);
В теле внутреннего цикла каждый элемент массива сравнивается с заданным числом, суммируется количество элементов больших этого числа, значение каждого такого элемента выводится.

Вывод накопленного значения kol - за внешним циклом.
