Двумерный числовой массив

Массивы характеризуются

  • именем (идентификатором),
  • размерностью (одномерный, двумерный, трехмерный, ...),
  • размером (количеством элементов).

Компиляторы C++ поддерживают по меньшей мере 12-мерные массивы.

Примером двумерного массива являются таблицы значений, содержащие информацию в строках и столбцах.

Чтобы определить отдельный табличный элемент, нужно указать два индекса:

  • первый (по соглашению) указывает номер строки,
  • второй (по соглашению) указывает номер столбца.
Например, массив       int mas[3][4];
  • Имя массива - mas.
  • Целочисленный двумерный массив - значениями элементов являются целые числа.
  • Количество строк - 3 (индексы 0, 1, 2).
  • Количество столбцов - 4 (индексы 0, 1, 2, 3).
  • Количество элементов массива 12.

В таблице указаны возможные значения элементов массива mas[3][4]. Чтобы обратиться к любому из элементов, достаточно указать значения i и j

Например, значение элемента   mas[1][2] равно -191,   mas[0][3] равно 105,   mas[2][1] равно -1.
