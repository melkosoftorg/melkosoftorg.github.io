Пункты для конспекта (лекция 34)

  1. Символьная строка - это последовательность, состоящая из одного или более символов.
  2. Строка с нулевым конечным символом представляет собой массив элементов типа char.
  3. В C/C++ строки могут иметь любую длину.
  4. Последний элемент строки-массива содержит значение 0 ( \0 - "нуль-символ" ) и таким образом обозначает конец строки.
  5. При использовании строковых констант, нуль в конце строки добавляется компилятором автоматически
  6. Конструкция ... 
    	while (s[n] != 0) n++;
  7. Конструкция ... 
    	while (s[n++] != 0);
  8. Если описать строку как массив типа char, то память для этой строки резервируется автоматически.
  9. Если описать строку как указатель на тип данных char, то память не резервируется: необходимо либо выделить ее, используя функцию malloc или ей подобную, или присвоить ей адрес существующей строки.
  10. Строка "к" не то же самое, что символ 'к'...
  11. Каждая символьная строка является символьным массивом, но не каждый символьный массив является символьной строкой — отличительным признаком является нулевой байт в конце символьной строки.
  12. Вывод строки: 
    	char s[ ] = "Понедельник"; cout << s << endl;
  13. Ввод строки: 
    	char  sd[80];
	cin >> sd;
    Операция cin >> для символьных массивов работает только до пробела.
  14. Размер массива и длина строки символов, которая в этом массиве хранится — это разные вещи. 
    	char s[10]=”Привет!”
	cout << sizeof(s)		// выведет 10 
	cout << strlen(s) << endl;	// выведет 7
