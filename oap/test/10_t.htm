Потоковый ввод - вывод в C++.

Сформулировать определения

  1. Манипулятор потока endl (end line)
  2. Объект стандартного потока ввода cin класса iostream
  3. Объект стандартного потока вывода cout класса ostream
  4. Ввод-вывод на низком уровне
  5. Ввод-вывод на высоком уровне
  6. Заголовочный файл <iostream.h>
  7. Заголовочный файл <iomanip.h>
  8. Заголовочный файл <fstream.h>
  9. Перегруженная операция <<
  10. Перегруженная операция >>

Какие действия производятся следующими командами ?

  1. cout.put ('А');
  2. cout.put ('А').put('\n');
  3. cout.put(65)
  4. cout.precision(3);
  5. cout << setprecision(3) << tt;
  6. int n = 9; cout << n;
  7. int n = 9; cout << hex << n;
  8. int n = 9; cout << oct << n;
  9. int n; cin >> n; cout << setbase(8) << n;
  10. int n; cin >> n; cout << setbase(8) << n << dec << n;
  11. cout << '\n';
