Вопросы тестирования по теме:
Типы данных языка C++.

Сформулировать определения

  1. Определить переменную - ...
  2. Определить константу - ...
  3. Типизированные константы - ...
  4. Описать данные - ...
  5. Целые числа - ...
  6. Числа с плавающей точкой (вещественные) - ...
  7. Строка - ...
  8. При записи вещественных чисел необходимо придерживаться правил:

Обозначения

  1. Последовательность ... является специальным символом, который C вставляет после каждой строки, чтобы отметить конец строки, занимает один элемент памяти.
  2. Восьмеричное число
  3. Шестнадцатеричное число
  4. При присваивании значения переменной типа char необходимо значение (единичный символ) заключать в ...

Что означает команда

  1. Оператор: char elen[12] = "Borland C++";
  2. Оператор: int tdw;
  3. Оператор: int tdw = 10;
  4. Оператор: char isi ='j';
  5. Оператор: float z2_c = 2.4;
  6. Оператор: float z2_c;

В общей форме записи числа с плавающей точкой ±m.nE±k что означает

  1. m.n
  2. E
  3. ±
  4. k
  5. m
  6. n
