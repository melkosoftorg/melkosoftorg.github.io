Вопросы тестирования по теме:
Функции работы с файлами

Сформулировать определения

  1. Файлы ...
  2. Последовательный файл ...
  3. Указатель на файлы ...
  4. Открытие файла ...
  5. Закрытие файла ...
  6. Определение   FILE *fp; ...

В определении   FILE *fopen(char *filename, char *type);

  1. значение символьной строки type равное r означает ...
  2. значение символьной строки type равное w означает ...
  3. значение символьной строки type равное a означает ...
  4. значение символьной строки type равное r+ означает ...
  5. значение символьной строки type равное w+ означает ...
  6. значение символьной строки type равное a+ означает ...

Что делает функция

  1. Функция   int access(char * filename, int amode);
  2. Функция   char *searchpath(char *filename);
  3. Функция   FILE *fopen(char *filename, char *type);
  4. Функция   int fclose (FILE *str);
  5. Функция   void exit (int status);
  6. Макрокоманда   int getc(FILE *str);
  7. Функция   int fgetc(FILE *str);
  8. Макрокоманда   int putc(int ch, FILE *str);
  9. Функция   int fputc(int ch, FILE *str);
  10. Функция   char *fgets(char *string, int n, FILE *str);
  11. Функция   int fputs(char *string, FILE *str);
  12. Функция   size_t fread(void *ptr, size_t size, size_t n, FILE *str);
  13. Функция   int fwrite(void *ptr, int size, size_t n, FILE *str);
  14. Функция   int fscanf(FILE *str, char *format [ , adress, ... ] );
  15. Функция   int fprintf(FILE *str, char *format [,argument,...]);
