Вопросы тестирования по теме:
Организация условных переходов.

Сформулировать определения

  1. Разветвляющийся процесс - ...
  2. Ветвь - ...
  3. Оператор И - ...
  4. Оператор ИЛИ - ...
  5. Оператор goto m - ...
  6. Значение условия, используемого в ветвлении ...
  7. Какое значение получается в результате вычисления данного выражения: 12>5 && !(12<9) || 5<4
  8. Что определяет данное выражение x = (y > x) ? y : x
  9. Укажите правильную запись условного оператора:
    • x = (e1 > e2) ? e1 : e2;
    • x = (e1 > e2) ? e1 > e2;
    • x = (e1 > e2) & e1 : e2;
    • if (e1 > e2) x = e1; else x = e2;
    • if (e1 > e2) ? e1 else e2;
  10. Какие значения e и f выведутся в результате выполнения данного фрагмента программы? 
    	int a = 42; 
    	int b = 2; 
    	int c = 99; 
    	int d = 0; 
    	int e = (b == 0) ? 0 : (a / b); 
    	int f = (d == 0) ? 0 : (c / d); 
    	printf("e = %d ", e); 
    	printf("f = %d", f);
