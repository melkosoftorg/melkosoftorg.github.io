Какое значение примет z после исполнения фрагмента программы:

Пояснения:

  • При вводе с клавиатуры значения правильного ответа необходимо ввести только числовое значение по правилам языка программирования.
    Например, для int z = 17 / 5; правильным ответом будет 3, а для float z = 17 / 5; получим в ответе 3.4

  • Операция (float) 17 / 5 называется преобразованием типов, результат деления преобразуется к вещественному типу и в ответе получим вещественное значение 3.4

  • Команда printf("z = %d", z); выводит на экран значение z как целое число, на что указывает спецификатор формата вывода %d

  • В команде вывода printf("z = %.1f\n", z); спецификатор вывода %.1f указывает, что будет выведено вещественное число с одним десятичным знаком после точки.

 

  1. 	int x = 36, y = 5, z; z = x % y; printf("z = %d", z);
  2. 	int x = 36, y = 5, z; z = x / y; printf("z = %d", z);
  3. 	int a = 13, b = -2, z; z = a / ++b; printf("z = %d", z);
  4. 	int a = 13, b = -3, z; z = a - b--; printf("z = %d", z);
  5. 	int a = 13, b = -4, z; z = a - --b; printf("z = %d", z);
  6. 	int a = 13, b = 5, z; z = a % b++; printf("z = %d", z);
  7. 	int a = 13, b = 7, z; z = a % b--; printf("z = %d", z);
  8. 	int b = 7, z; z = sizeof(b); printf("z = %d", z);
  9. 	float b = 4.76; int z; z = sizeof(b); printf("z = %d", z);
  10. 	int a = 5, b = 2; float z; z = (float) a / b; printf("z = %.1f\n", z);
  11. 	int a = 5, b = 2; float z; z = a / b; printf("z = %.1f\n", z);
  12. 	float a = 5, b = 2, z; z = a / b; printf("z = %.1f\n", z);
  13. 	int x = 1, y = 2, z; z = x % y; printf("z = %d\n", z);
  14. 	int x = 1, y = 2, z; z = x / y; printf("z = %d\n", z);
  15. 	int x = 1, y = 2, z; z = ++x * y++; printf("z = %d\n", z);
  16. 	int a = 20, b = 7, z; z = a % ++b; printf("z = %d", z);
  17. 	int a = 17, b = 6, z; z = a % --b; printf("z = %d", z);
  18. 	int z = 1, a = 15, b = 2; z += a * --b; printf("z = %d", z);
  19. 	int z = 2; a = 7, b = 3; z *= ++a + b; printf("z = %d", z);
  20. 	int z = 12; a = 14, b = 3; z /= a % b; printf("z = %d", z);
  21. Укажите верные фразы:
    • Операции только для вычислений с целыми числами: ++, --, %
    • В языке С нельзя применять последовательные присваивания: sum = a = с = 5;
    • Если действие производится с одним операндом, операция называется бинарной.
    • Результат выражения определяется типами обоих операндов.
    • Над числами с плавающей точкой операция % не производится.
    • Операция последовательного вычисления обозначается точкой с запятой (;)
