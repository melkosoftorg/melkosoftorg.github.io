Лабораторно-практическое занятие № 1: Расчёт цепи со смешанным соединением резисторов.

Цель занятия:

  • Изучить методы расчета электрических цепей.
  • Выполнить расчет цепи со смешанным соединением резисторов.
 

Аппаратно - программное и методическое обеспечение:

  • Персональный компьютер (Intel/AMD)
  • Операционная система (Windows/Linux)
  • Файловый менеджер FAR.
  • Офисный пакет OpenOffice.
  

Выполнить следующие задания:

  1. Повторить тему 1.2. Электрические цепи и сигналы.

  2. Подготовить рабочее место для выполнения лабораторных и практических работ.


  3. Создать папку  D:\Gruppa\p151

  4. Настроить офисный пакет OpenOffice.

  5. Выполнить расчёт цепи со смешанным соединением резисторов согласно своего варианта.

  6. Подготовить “Отчёт по Лабораторно-практической работе №1 – Расчёт цепи со смешанным соединением резисторов” с помощью OpenOffice (Writer).   Имя файла – 0221p01_00p111.odt  
    где:   0221 – шифр предмета,   p01 – номер практической работы,   00p111 – ваше сетевое имя (логин).

  7. Поместить файл 0221p01_00p111.odt   в папку  D:/Gruppa/p151\

  8. Отправить по электронной почте своему преподавателю (E-mail преподавателя ниже)
    краткое сообщение о выполненной работе с прикрепленным файлом 0221p01_00p111.odt 

    В строке "Тема письма" указать шифр предмета (0221), номер практической работы (p01) и своё сетевое имя (00p111), c пробелом ( _ ) между ними,
    например:   0221p01_00p111

    Группа Подгруппа (рабочие места) E-mail преподавателя
    П-122, П-123, П-124, П-131 L306 01-30 отправить. В появившемся окне в поле с подписью "Введите свой UID:" введите свое сетевое имя, в поле с подписью "Введите номер пропуска:" введите номер своего пропуска без запятых, номер находится на обороте карты. Нажмите кнопку "Дальше", затем кнопку "Добавить файл", после чего кнопку "Обзор". В появившемся диалоговом окне найдите файл расчетной таблицы, предварительно сохранив его, и нажмите кнопку "Открыть". Нажмите кнопку "Отправить", если все сделано правильно, то вы получите уведомление об успешной передачe файлов.


Срок выполнения работы до 16 сентября.
