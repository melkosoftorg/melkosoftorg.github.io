2.8.2. Классификация счётчиков

Все счётчики классифицируют:

  • По числу устойчивых состояний триггеров:
    • на двоичных триггерах;
    • на троичных триггерах;
    • на n-ичных триггерах.
  • По модулю счёта:
    • двоично-десятичные;
    • двоичные;
    • с произвольным постоянным модулем счёта;
    • с переменным модулем счёта.
  • По направлению счёта:
    • суммирующие;
    • вычитающие;
    • реверсивные.
  • По способу формирования внутренних связей:
    • с последовательным переносом;
    • с параллельным переносом;
    • с комбинированным переносом;
    • кольцевые.
  • По быстродействию (способу переключения триггера)*:
    • асинхронные (последовательные) счетчики ;
    • синхронные счетчики с асинхронным переносом (синхронно-асинхронные счетчики);
    • синхронные (параллельные) счетчики.
  • Счётчик Джонсона.
 

(*) Принципиальные различия между этими группами проявляются только на втором уровне представления, на уровне модели с временными задержками. Причем больше всего различия эти проявляются при каскадировании счетчиков. Наибольшим быстродействием обладают синхронные счетчики, наименьшим - асинхронные счетчики, наиболее просто управляемые среди других.
